## 1. ルゥレーンTRPGとは

- ルゥレーンTRPGは、剣と魔法のファンタジー世界「ルゥレーン」にて、冒険者となり、冒険に挑むテーブルトークRPGです
- ゲームマスター (GM) とプレイヤー (PL) に分かれてゲームを進行します
    - GMは世界の説明やシナリオの描写、ゲーム進行の管理を担当します
    - PLは自身の分身であるプレイヤーキャラクター (PC) を通じ、ルゥレーンを舞台に冒険を繰り広げます
- ルゥレーンには、太古の時代から遺されたり、宇宙や異界から漂流した様々な"秘宝"が眠っています
- とりわけ強力な"秘宝"は、どういうわけか"ダンジョン"と呼ばれる巨大な迷宮に流れ着くのです
- 秘宝を求めてダンジョンに挑む者を、人々は"冒険者"と呼びます
- そして、あなたたちもまた、そう呼ばれることでしょう

| 役割 | 説明                                                                     |
| :--- | :----------------------------------------------------------------------- |
| GM   | Game Master。世界の描写、NPCの操作、状況判定を担うゲームの進行役         |
| PL   | Player。自分の分身であるPCを操作し、その判断と行動でセッションに参加する |
| PC   | Player Character。PLが操作する、世界に存在する冒険者の人格               |

## 2. 用意するもの

1. ルゥレーンTRPGのルールブック
    - 今読んでいるこのドキュメントです
    - ルールブックは Markdown 形式で Github にて公開しています
2. 大量の6面ダイス
    - ルゥレーンTRPGはたくさんの6面ダイスを使います
    - 1つの卓に30個ほど用意してあると安心です
    - 一人あたり、それぞれ15個程度を個別に持ち寄ると、プレイに便利です

> その他の小道具:
> - ダイストレイがあるとプレイがしやすくなります

## 3. ロールシステム

- ルゥレーンTRPGでは、ダイスを複数個振って、どれだけ同じ目が揃ったかで達成値 (出目の良さ) を表現します
- 例えば、ダイスを5個振り、結果が $1, 1, 2, 4, 6$ の場合は、1の目が2個揃っているので、達成値は2です

ロールの記法は $NM6$ を用います:
- N: 振るダイスの数 (= 能力値)
- M: マッチ判定 (Match)
- 6: 6面ダイス

略称:
- 達成値の略称は $MtV$ (Match Value) を用います
- 詳細については [system/01-roll/](system/01-roll/) を参照してください

## 4. プレイヤーキャラクターを作る

- ルゥレーンにはヒューム、エルフ、ドワーフ、ビースト、ドラゴニュートなど、多彩な種族が生息しています
- 冒険に旅立つ前に、まずあなただけのプレイヤーキャラクター (PC) を作成しましょう

キャラクターメイキングの手順:
1. 名前、種族、性別を決めます
2. 種族に応じて、キャラクターのステータスをダイスロールで決めます
3. キャラクターの年齢を決めます
4. 年齢に応じて、キャラクターの才能と背景を決めます
5. 信仰する神を決めます
6. キャラクターのお金を使用して、装備やアイテムを購入します

クラスとレベルについて:
- ルゥレーンTRPGには、職業やレベルといった概念はありません
- PCが使える魔法や奇跡、持てる装備やできる行動を考え、あなたのキャラクターを作成してください

各種手順とデータ:
- プレイヤーキャラクターのステータスと作成手順についての詳細:
    - [system/02-player-character/](./system/02-player-character/) を参照してください
- 種族についての詳細:
    - [system/03-species/](./system/03-species/) を参照してください
    - 種族によって、ステータスや特性、背景や魔法が異なります
- 魔法や奇跡についての詳細:
    - [system/04-spells/](./system/04-spells/) を参照してください
    - 種族や信仰している神によって、行使できる魔法や奇跡が変化します
- 冒険者の背景についての詳細:
    - [system/02-player-character/03-background.md](./system/02-player-character/03-background.md) を参照してください
    - 冒険者になる前の背景により、ステータスや所持金が変化します
- 所持品のルールと、各種アイテム・装備品についての詳細:
    - [system/05-items/](./system/05-items/) を参照してください

PCが出来上がったら、冒険の舞台へと旅立ちましょう。

- セッションには、GMとPLがそれぞれ下記のものを持ち寄ります
    - GM: シナリオシート、プレイに必要な小道具
