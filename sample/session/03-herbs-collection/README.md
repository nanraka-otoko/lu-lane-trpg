# 薬草採取の護衛

## シナリオの概要

```
ルゥレーンでの暮らしに薬草は欠かせない。
傷を癒し、死を遠ざけるポーションは、薬草を素材として作られる。

薬草はハーブであり、香辛料としても用いられる。保存の効く食材だ。

だが、ポーションの生成にはその薬草の鮮度が求められる。
生命を癒す薬には、生命力が満ちた素材が必要なのだ。

混沌の手勢に対抗するには、ポーションを備蓄しなければならない。
薬草採取の護衛は、冒険者の社会貢献として最も一般的なもののひとつだ。

今日ももちろん、冒険者ギルドには依頼が来ている。
新人の仕事としては悪くない選択肢だろう。
```

---

## セッション情報

- 想定PC数: 3～4人
- 想定脅威度: 1～2
- 想定プレイ時間: 2～3時間

### 1PCあたりの報酬の目安

| 項目              |  WLT |
| :---------------- | ---: |
| 依頼報酬          |    2 |
| 戦利品 (ドロップ) | 1～2 |
| 戦利品 (採取)     | 1～2 |
| 合計              | 4～6 |

### セッション所要時間の目安

| フェイズ       | 時間         |
| :------------- | :----------- |
| 冒険準備       | 15分         |
| 目的地への移動 | 15分         |
| 森でのキャンプ | 30分 ～ 45分 |
| 薬草採取       | 15分         |
| 野盗の撃退     | 30分 ～ 45分 |
| 帰還・報告     | 10分         |
| 合計           | 約2～3時間   |

### 備考

- 簡易的なシナリオです
- 「05 行商人の荷馬車の護衛」などのシナリオと組み合わせ、ボリュームアップしても構いません
- ギルドから「今日はこんな依頼があります」といった形式で複数の依頼を提示し、PLに選ばせる方法を採用しても良いでしょう

## 冒険準備

### 情報収集

君たちはギルド職員から依頼人を紹介され、説明を受ける。

依頼の内容はこの通りだ。

1. 森に自生している薬草を採取しにいく
2. 採取班は3人
3. 森で一晩キャンプし、翌朝に薬草を採取して帰還する
4. 報酬は一人あたりWLT 2
5. キャンプの食料は採取隊が持つ

詳しい話を聞ける。

1. 採取隊のメンバーは、薬草を持つために武装をしていない
2. いざというときのためにポーションを持っているが、冒険中に買うならWLT 1を要求する

> GM向け: PLがさらなる情報を求めた場合、目標値2の対人判定を要求する。
>
> 成功すると:
> 1. 森には狼が出ることがある。群れで来ることは少ないが、夜営時は注意が必要
> 2. 採取地までは街道を半日、そこから森に入って2時間ほど
> 3. 前回の採取でも護衛がついたが、特に問題はなかった

---

## 目的地への移動

- 街道を半日歩く。天候は穏やかで、特に問題は起きない
- 道中、採取班との会話を通じてロールプレイの時間を取る

### 採取班のメンバー

採取班は以下の3人で構成されている。

- リーゼ (班長): 中年の薬師。事務的だが信頼できる人物。薬草の知識が豊富で、質問すれば丁寧に教えてくれる
- マルト (見習い): 若手の薬師見習い。明るい性格で、冒険者の仕事に興味津々。やたらと話しかけてくる
- ゲルト (荷運び): 寡黙な男。元冒険者だが膝を痛めて引退し、今は荷物係をしている

> GM向け:
> - NPCの名前や人物像は自由に変更して構わない
> - 採取班のメンバーは非戦闘員として扱い、戦闘には参加しない

### 森への道

- 街道を離れ、森に入る。木々が密になり、足元も悪くなっていく

探索と移動:
- PCは目標値2の探索判定を行う
- 誰か一人でも成功すると、スムーズに進行できる
- 全員が失敗した場合、目標地点までの移動が遅れ、野営の時間を2時間失う

---

## 森でのキャンプ

- 採取地の近くに到着する頃には、辺りは薄暗くなっている
- 採取は翌朝に行うため、ここで一晩キャンプを張る
- 野営は8時間を使用するが、移動が遅れた場合は6時間となる

### 食事と会話

野営:
- 採取班が食事を用意してくれる
- PCは食料を消費しなくても良いが、手持ちの食料を供与することで、さらに野営ロールに+2dのボーナスが得られる

焚き火を囲んで、採取班との会話を楽しめる

- マルトは冒険者の武勇伝を聞きたがる。話してやると喜ぶ
- リーゼに薬草のことを聞くと、採取地の薬草の特徴や、ポーションの調合について教えてくれる
- ゲルトに話しかける場合、目標値2の対人判定を要求する。成功すると:
    - ゲルトは冒険者時代の話をしてくれる。大した冒険者ではなかったが、護衛の仕事を長くやっていた
    - 「夜の森では焚き火を絶やすな。獣は火を嫌う」と助言してくれる

> マルトの会話:
> - 「なぁなぁ、あんたらこれまでどんな冒険してきたんだ？ ダンジョンとかも行ったのか？ 中ってどうなってる？」
> - 「俺は今はこうして薬師の見習いをやってるけどさ、いつかダンジョンに潜ってみたいぜ！」

> リーゼの会話:
> - 「薬草のことについて知りたいですか？ そうですね、薬草の用途は主に、保存料、香辛料、そして調合の素材です」
> - 「薬草は生命力の豊かな場所に生えるんです。都市部での栽培もこれまでに研究されてきましたが、成功した例はないか、コストが見合わないという結論が主流なんです」
> - 「だからこうして皆さんに、護衛をお願いししているわけですね」
> - 「実入りは良い仕事ですが、それよりもやっぱりポーションの調合が好きなんです。僕は」
> - 「僕はこう、葉が親指くらいの大きさの薬草を好んで採取していますね。僕の調合手順と相性が良いんです」

> ゲルトの会話:
> - 「俺の過去の話か？ よくある話だ、深手を負った冒険者が引退した後、こうして近しい仕事をするのは」
> - 「大した冒険者じゃなかったよ。護衛の仕事を長くやっていた。リーゼに聴かせる武勇伝なんかも、持っていないさ」
> - 「夜の森では焚き火を絶やすなよ。獣は火を嫌う」

### 襲撃

- キャンプでは、夜間の見張りをPCが分担する
- 見張りの割り振りをPLに相談させる

> 野営の扱い:
> - 野営の処理に従い、PCは4時間以上寝ないと、翌日にペナルティが生じる
> - 野営は8時間を使用できるため、GMは見張りの順序をPLに宣言させること

遭遇:
- 深夜、空腹の狼がキャンプに近づいてくる
- ウルフパック x (PC数 x 2) と遭遇する
- 4時間経過時点で見張りをしていたPCは、目標値3の感知判定を行う
    - 焚き火を炊き続けている場合は、目標値が2となる
- 成功すると、狼たちの接近に気が付き、睡眠中の仲間を起こせる
- 失敗した場合、奇襲を受けて戦闘を開始する

野営ロール:
- 戦闘終了後、PCたちは眠りについて良い
- PCたちは起床時、野営ロールを振り、ボーナスを得る

---

## 薬草採取

- 翌朝、野営を撤収して採取地についた冒険者と依頼人たちは、薬草の採取を開始する
- 採取班が手際よく薬草を採取していく
- PCたちは護衛として周囲を警戒しつつ、2時間ほどの採取作業を見守る

探索:
- 護衛の合間に、PCは探索判定を行っても良い
- 探索判定のMtVを利用して戦利品を獲得できる

| 戦利品       | 目標値 | 価値 | 重量 | 説明                                                                            |
| :----------- | -----: | ---: | ---: | :------------------------------------------------------------------------------ |
| 薬草         |      2 |    ? |    0 | 依頼人が探している薬草。2個で価値:1としてリーゼが買い取る                       |
| 珍しいキノコ |      4 |    2 |    0 | 奇妙な色をしたキノコ。目標値3の知識判定で、食用の珍しいキノコであることがわかる |

> 珍しいキノコ:
> - リーゼたちは見たことがないというキノコ。
> - 目標値3の知識判定に成功すると、これが高値で取引されるキノコであることがわかる
> - 失敗した場合、このキノコを鑑定するには都市に戻ってからWLT 1を支払う必要がある

## 野盗の撃退

- 採取を終え、森を出て街道に戻る帰路。木々が開けた場所で、前方に人影が立ちはだかる
- クロスボウを持った野盗のリーダーが姿を見せ、声をかけてくる
- 「おい、そこの連中。荷物を置いていけ。素直にすれば痛い目は見ないぜ」
- 茂みに隠れていた配下の野盗が姿を現す。採取班のマルトとリーゼは、怯えて動けない
- ゲルトは、二人を庇うように立ちながら、冒険者たちに目配せをする

### 遭遇前の選択

- 野盗のリーダーは話を続ける
- 「荷物が嫌なら、手持ちの武器と有り金を全部置いていきな。通行料ってやつだ」
- 「従わねぇなら……分かってるだろうな？」
- 野盗のリーダーはクロスボウをちらつかせ、配下の野盗たちは下卑た笑いを浮かべる

PCにはいくつかの選択肢がある:
- 交渉する: 目標値4の対人判定に成功すると、PCの手持ちWLTをすべて差し出すことで見逃してもらえる
- 荷物を渡す: 採取した薬草をすべて失う。依頼報酬は得られない
- 戦う: 戦闘を開始する

遭遇:
- 戦闘する場合、スリ x 2、野盗 x 3、野盗のリーダー x 1 との遭遇となる
- この野盗との遭遇では、PCは奇襲を実行できない

> GM向け:
> - 野盗のリーダーは後衛、野盗たちは前衛で配置する。
> - 野盗のリーダーはPCの前衛にクロスボウを射掛けてくる
> - スリは後衛に配置し、ナイフを投擲する。前衛が減ると、リーダーが前進を指示する
> - PC数が5人以上の場合、敵後衛にならず者の用心棒を配置し、最初のターンで敵前衛に移動させると良い
> - 最初のラウンドでは攻撃しないためPCが多少有利になるが、危険な敵を増やすことでバランスを取れる
>
> 野盗たちの結末:
> - PCたちが野盗を気絶させて捕獲を希望した場合、GMは認めて良い
> - その場合、ギルドや衛兵への引き渡し時に、野盗たちへの戦利品判定に+2dのボーナスを得て戦利品を獲得して良い
> - 野盗たちはそれぞれ重量:3の所持品として扱う。持ちきれない場合、その野盗は捨て置くか、殺害しても良い
> - NPCたちは野盗を殺すことを提案する
> - こちらも殺される可能性が高かったことと、連れ歩くこと自体が危険だと主張する
> - 野盗たちは犯罪者であり、彼らを死亡させても冒険者は罪に問われない

---

## 帰還・報告

- 都市に帰還する
- リーゼは礼を述べ、採取した薬草を使ってポーションを調合する予定なことを教えてくれる
- マルトは別れ際に「次も護衛お願いしますね！」と手を振る
- ゲルトは無言で頷く
- ギルドに報告すると、依頼の報酬が支払われる

## 結末

### 成功

- 依頼達成の報酬: 1PCあたり WLT 2
- 冒険で得た戦利品を換金できる
- 冒険者たちは1点のEXPを獲得する

### 野盗にPCの所持品を渡した場合

- 野盗にPCの所持品を渡した場合は、依頼者からの報酬を受け取れる
- 依頼達成の報酬: 1PCあたり WLT 2
- 戦利品は得られない
- 野盗に渡した所持品やWLTをすべて失う
- 冒険者たちは1点のEXPを獲得する

### 野盗に依頼人の荷物を渡した場合

- 野盗に依頼人の荷物を渡した場合は、依頼者からの報酬は受け取れない
- 戦利品は得られない
- 冒険者たちは1点のEXPを獲得する

### 獣との戦いに敗北

- 獣との戦いで敗北し、全滅した場合、すべてのPCは運判定を行う
    - 死亡しているPCは達成値に-1の補正を受け、死の結末のみを受け入れる
- 達成値0
    - 君は死亡した
    - 君はウルフパックたちの食事となり、蘇生されることもなかった
- 達成値1
    - 君は死亡した
    - しばらくした後、死体が回収される
    - 望むのなら、蘇生の奇跡を受けることができる
    - 蘇生を受ける場合、WLTを10失い、CONとMNDを恒久的に-1点して復活する
- 達成値2以上
    - 君は一命をとりとめ、なんとか都市に撤退した
- いずれの場合でも、君たちは依頼者を守ることはできなかった
- 報酬は受け取れない
- 冒険者たちは1点のEXPを獲得する

### 野盗との戦いに敗北

- 野盗との戦いで敗北し、全滅した場合、すべてのPCは運判定を行う
    - 死亡しているPCは達成値が0となり、死の結末のみを受け入れる
- 達成値0
    - 君は死亡した
    - 野盗たちは君の所持品をすべて剥ぎ取り、その辺に捨て置いた
    - 君の死体は回収されることなく、野鳥や獣たちに食われた
- 達成値1
    - 君は気絶し、野盗に捕まった
    - 君は身代金としてWLT10を失うことで解放される
    - 身代金が払えない場合、野盗の一味となる。APPを恒久的に-2し、2点のEXPを獲得し、解放される
- 達成値2以上
    - 君は一命をとりとめ、なんとか都市に撤退した
- いずれの場合でも、君たちは依頼者を守ることはできなかった
- 報酬は受け取れない
- 冒険者たちは1点のEXPを獲得する
