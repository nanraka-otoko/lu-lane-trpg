# オークの集落の攻略

```
オークは決して人族に劣る存在ではない。
そんなことを町中で主張したら、たちまち指をさされて笑われるだろう。
本気でそんなことを主張したら、魔族の手先として排斥されるかもしれない。
もはや都市部の人間にとっては、オークなどは冒険者に狩られるだけの動物に過ぎない。

だが、オークは決して、人族に劣る存在ではない。
大抵の人族よりも強靭な身体に、群れをなして狩りをする知性、そして秩序の文明に対する明確な敵意。
人族の少なくない人口が、オークによって狩られている。

彼らは文明に参画するための知性がないのではなく、自ら野生の中で生きることを選んでいる。
そして今日まで種として血を繋いでいるということは、野生の中で生きる術を継いでいるということだ。

オークたちは原始の時代の集落を築き上げ、部族単位で生活を営む。
規模が大きくなると集落が人口を許容できなくなり、人族の村や街を襲撃して新たな根拠地とする。

この習性は秩序文明にとっては厄介なものであり、故にオークの集落を先んじて攻めることが肝要である。
オークは廃村や遺跡を根拠地とする。つまり、冒険者の出番である。
```

## 冒険準備

### 情報収集

ギルドからの説明:
1. 街道沿いの森で、商隊がオークの襲撃を受けた
2. 商隊は護衛の冒険者によって無事だったが、物資のいくつかは盗まれた
3. オークたちは森の奥へと逃げ去っていった
4. 冒険者たちはそのオークを追跡した結果、古い廃村に集落を築いていることを発見した
5. ギルドはこの報告を受け、集落の攻略戦を計画した
6. PCたちは攻略部隊の一員として、作戦に参加する

依頼の条件:
1. オークの集落を襲撃し、オークたちを討伐する
2. 全滅させる必要はないが、可能な限り数を減らす
3. オークたちを捕獲する必要はない。すべて殺傷して良い
4. ギルドからは一人あたりWLT 4の報酬を支払う
5. 冒険で得られた戦利品は、各自の自由にして良い

集落の規模について:
- 20～30体ほどの規模の集落と推測される
- 煙が立ち上っており、ある程度文化的な生活を営んでいる
- 狼を手懐けている様子である

攻略部隊について:
- PC以外に、もう1つのPTが参加する
- 壮年のダークエルフ「ヤニ」がリーダーを務める冒険者PT「シガレッツ」である
- シガレッツは同じ冒険者ギルドに所属している。依頼を受領すると、顔合わせとなる

> 冒険者PTについて:
> - GMの判断で、攻略に参加する冒険者PTを別のPTとしても良い
> - 過去のセッションなどでPLに馴染みのある冒険者たちがいる場合は、そうしたPTを登場させて良い
>     - その場合は、このシナリオでシガレッツと記載された箇所を、該当の冒険者PTとして読み替えること
> - 知り合いの冒険者がPTを組んでいない場合は、シガレッツの協力者として登場させるのも良いだろう
> 
> PCたちの後日談:
> - PCたちが「オークの集落を発見した」「魔族の根拠地の地図を見つけた」といった経験がある場合、発見者をPCたちとしても良い
> - その場合はPCが作戦を主導し、NPCの冒険者PTが攻略に参加する
>
> オークたちの数:
> - ギルドの推測は当たっており、このシナリオには合計で30体のオークが登場する

### 作戦会議

- ギルドの一室にて、攻略部隊の冒険者PTと作戦会議を行う
- オークの集落を発見したのは、この冒険者PTである

冒険者PT「シガレッツ」のメンバー:
- ダークエルフの禁止魔術師、ヤニ (女性、壮年)
- 熊獣人の重戦士、シケモク (男性、壮年)
- ケンタウロスの天文学者、チェーン (女性、壮年)
- エルトメンヒェンの遊び人、タール (女性、壮年)

> NPCのキャラクターシート:
> - シナリオに付属のキャラクターシートを参照すること

集落の場所:
- 都市部から1日と半日ほど街道を進んだ先の森に、オークたちが潜んでいる
- 森の中には古びた廃村がある。街道から森をさらに半日ほど踏み分ける必要がある
- 街道まで1.5日かけて移動し、その後森の前で野営をする。その翌日に森に踏み入る算段である

オークの情報:
- 商隊を襲ったオークの内、1体が森に逃げていった
- オークを追跡した結果、集落を発見した
- オークたちは弓や槍で武装しており、狼を手なづけている
- 大剣を持った個体もいる。集落なので上位個体がいる可能性が高い

作戦について:
- 集落の正面からオークを相手取るPTと、その隙に裏手から攻めるPTとに分かれる
- 正面部隊はオークを引きつけて集落の外に設置したバリケードで耐久戦を行う
- 裏手から奇襲したPTは、集落内に残ったオークを討伐し、その後に正面部隊と戦っているオークを挟撃する

分担について:
- シガレッツは正面部隊を希望する
    - 耐久力の高い冒険者が揃っており、耐久戦に分があると、ヤニは主張する
- どちらを分担するかは、PCたちが決めて良い

## 街道の移動

- 冒険者ギルドを出発し、ギルドが手配した馬車で目的地まで移動する
- 討伐戦が目的であるため、馬車には他の乗り合わせや御者などはおらず、自分たちで操縦して移動する
- 1.5日後、17:00頃に森の前にたどり着く

野営:
- PCたちはここで野営を行う
- 翌日の早朝4:00に森に侵入し、オークの集落を攻める算段である
- 各々準備した道具で野営を行うことができる

> 寝床について:
> - シガレッツのメンバーであるチェーンが、混交魔法「バウアー」を習得している
> - 一晩分の寝床は、土でできた小屋の中に用意される
> - PCたちは快適な寝床を得られ、野営ロールに+2dのボーナスを得ることができる
>
> シガレッツ以外の冒険者と行動している場合:
> - GMは、PCたちのNPCと交流次第で、野営ロールにボーナスを与えても良い

## 森の探索

### 街道からの出発

- 馬車を街道の脇に係留し、不要な道具を荷台に置いて森に侵入する
- 馬車は特に襲撃されることはないが、PCたちに何か心配事がある場合は、GMは好きにさせて良い

### 森の中の進行

- 8時間ほどかけて森の中を進む
- 途中、オークの斥候と出くわす
- オークは2体。5匹の狼を引き連れている

分業:
- お互いの腕試しと称し、NPCのPTはオークか狼か、どちらか片方ずつをそれぞれのPTで受け持つことを提案する
- PCは好きな方を選択する
- 選択したエネミーとの遭遇となる

遭遇A:
- オーク x 2

遭遇B:
- ウルフパック x 5

> 戦闘について:
> - 基本的には余裕で勝てるエネミーであり、ほとんど脅威にはならない
> - 小手調べ感覚で戦闘を行わせ、PLたちに余裕ムードを演出させると良い

戦闘終了後:
- PCたちは目標値3で知識判定を行う
- 成功すると「斥候が帰還しないことが発覚すると、襲撃が感づかれる」ことに思い至る
- ここからは急いでオークの集落に向かう必要がありそうだ

> 時間経過について:
> - 修理判定や治療判定、戦利品判定を行うと、時間を所要してしまう
> - 時間を所要した場合は、オークの集落でのPCたちのイニシアチブ判定にペナルティを与える
> - 戦闘後、10分を所要する判定を行った場合、-1dのペナルティが付与される
>     - このペナルティは-6dまで累積する
>     - 修理判定のような1時間を所要する判定を行った場合、ペナルティは-6となる
> - このペナルティについては、知識判定に成功したPCに対しては、処理の内容まで説明して良い

### オークの集落への到着

- オークの集落に到着する
- ボロボロに倒壊した廃村の廃屋を改修して住み着いている様子だ
- 遠目でもオークたちが10体以上いることがわかる
- 廃村の中央からは、煙のようなものが上がっている
    - どうやら村の中央で焚き火を焚いているようである

襲撃:
- PCは正面部隊、奇襲部隊のうち、どちらを担当するかを決める
- それに従い、集落を襲撃する

## オークの集落の襲撃 - 正面部隊

### バリケードの設置

- PCたちは協力して、集落の外にバリケードを設置することができる
- PCは代表者を選出して、知識判定、探索判定、筋力判定を行う
    - それぞれの判定は、一人のPCのみが行うことができる
    - 同じPCは、異なる判定を実行できない
- すべての判定のMtVを合計し、バリケードを作成・設置する

| MtV合計 | バリケード                       |
| ------: | :------------------------------- |
|     ～3 | バリケードは作成できない         |
|    4～7 | 「小さなバリケード」を作成できる |
|     8～ | 「バリケード」を作成できる       |

> バリケードの材質:
> - 森の中に落ちているような倒木や草木を用いて作ったバリケードである

### 防衛戦

- 村の正面にはオークの門番が2体立っている
- このオークに発見されると、オークたちがPCを捕捉し、追いかけてくる
- オークは次々と集落から出撃し、PCを包囲しようとする

遭遇:
- オーク x 4、オークシューター x 3

バリケード:
- 戦闘開始時、PCは設置したバリケードを自陣の前衛とするか、後衛とするかを選べる

増援:
- 1ラウンド目終了時: オーク x 2、ウルフパック x 3
- 2ラウンド目終了時: オーク x 3、オークシューター x 2
- 3ラウンド目終了時: オーク x 2、オークシューター x 2、オーク・ウォリアー x 1
- 増援はすべて敵後衛に登場する
- オーク・シューターは後衛に留まって遠隔攻撃を行い、それ以外のエネミーは前衛に移動してPCに攻撃を仕掛ける

> 増援の調整:
> - PCたちが厳しい場合は、増援が登場するラウンドを次に回しても良い

### 集落内の部隊の救援

- PCが正面部隊を担当した場合、NPCの部隊が集落内で苦戦している
- 集落に侵入し、オークの頭目と戦っているNPCに加勢する
- 加勢時、イニシアチブ判定に+4dのボーナスを得る
- イニシアチブ判定のMtVが4以上の場合、PC側の奇襲での遭遇となる

遭遇:
- オーク x 2、オーク・シューター x 1、オーク・ドリフター x 1、オーク・プリースト x 1

> NPCの扱い:
> - オークの残りを引き受けているため、PCたちの戦闘には参加しない
> - 加勢したPCが劣勢の場合、3ラウンド目終了時を目処に、PCの後衛エリアからNPCたちを参戦させる
>     - この場合、NPCはMPを0点に、HPをCON点(最大HPの半分)にし、防具の装甲を0点にして加勢させること

## オークの集落の襲撃 - 奇襲部隊

### 見張りの暗殺

- 正面部隊がオークたちを引き付けたのを確認し、集落の裏手から頭目に奇襲を仕掛ける
- 道中、弓を持ったオークの見張りを2体発見する
- 暗殺を希望するPCは隠密判定を行う
    - 目標値3の隠密判定に成功すると、そのPCは見張りを奇襲できる
    - 誰か1人でも隠密判定に失敗した場合は、見張りに気づかれ、全PCが通常の遭遇となる
- 隠密判定に自信がないPCは、隠密判定をしなくても良い
    - その場合、失敗することはないが、そのPCは奇襲のボーナスを得られない
- 全員が隠密判定をしない場合は、イニシアチブ判定に+2dのボーナスを得ての遭遇となる

遭遇:
- オーク・シューター x 2
- この遭遇では、奇襲したPCは最初のラウンドで2ターン分の行動を行うことができる
- オーク・シューターはどちらも敵前衛に配置される
- オーク・シューターの2体は、最初のラウンドで敵後衛に移動し、撤退を宣言する
    - 撤退のルールに従い、そのラウンド中はオーク・シューターはすべての判定に-2dのペナルティを受ける
- 1ラウンド目の終了時、オーク・シューターは戦闘から撤退する

オーク・シューターを逃がした場合:
- オーク・シューターは頭目に奇襲を報告する
- 逃がしたエネミーが次の戦闘に参加し、PC側はイニシアチブ判定に-2dのペナルティを受ける

毒を与えている場合:
- 強度1点以上の毒を付与している場合、撤退した見張りは毒によって死亡する
- この場合は、その見張りは逃さなかったものとして扱って良い

### 頭目への奇襲

- 集落の中央で、正面部隊に対し追撃の指示を出している頭目と、その取り巻きに対して奇襲を行う
- 見張りのオーク・シューターを倒している場合、PCはイニシアチブ判定に+4dのボーナスを得ての遭遇となる
    - このイニシアチブ判定のMtVが4以上の場合、PC側の奇襲での遭遇となる
- 見張りのオーク・シューターを逃がした場合、PCはイニシアチブ判定に-2dのペナルティを受けての遭遇となる
    - この場合、MtVが4以上でも奇襲は発生しない

遭遇:
- オーク x 3、オーク・シューター x 2、オーク・ドリフター x 1、オーク・プリースト x 2
- 見張りを逃がしている場合は、見張りも戦闘に参加する
    - HPなどの状態は、直前の戦闘の状況を引き継ぐ
- オークとオーク・ドリフターは前衛に配置し、残りのエネミーは後衛に配置される
- 後衛のオーク・プリーストは、最初のラウンドで「切り伏せよ」をオーク・ドリフターに、「蹂躙せよ」を味方全員に行使する
    - 続けて予備行動で「戦士よ、立ち上がれ」を行使し、傷ついた味方を回復する
    - 2ラウンド目は「戦士よ、立ち上がれ」のみを行使する
    - MPが尽きた場合は、前衛に移動し、槍でPCを攻撃する
- オーク・ドリフターは斧槍によってPCの前衛を複数体同時に攻撃する
    - 自身のHPが減少している場合は、予備行動で「戦士よ、立ち上がれ」を自身に行使する
    - オーク・ドリフターは味方を回復することはない

### 正面部隊の救援

- PCが奇襲部隊を担当した場合、NPCの部隊がオークを引きつけて耐久戦を行っている
- オークたちの背後から挟撃を仕掛け、討滅する
- 加勢時は、PC側の奇襲での遭遇となる

遭遇:
- オーク x 3、オーク・ウォリアー x 1、オーク・シューター x 3

> NPCの扱い:
> - オークの残りを引き受けているため、PCたちの戦闘には参加しない
> - 加勢したPCが劣勢の場合、3ラウンド目終了時を目処に、PCの後衛エリアからNPCたちを参戦させる
>     - この場合、NPCはMPを0点に、HPをCON点(最大HPの半分)にし、防具の装甲を0点にして加勢させること

## 撤退する場合

- 戦闘中にPCが撤退した場合、そのまま森を抜けて街道まで引き返すことになる
- NPCたちは運判定を行い、MtVが1以下のNPCは撤退できずに取り残される
- 撤退の結末に進む

## 戦利品の獲得

- 集落の壊滅後、オークたちから戦利品を獲得できる
- 森の探索中に討伐したエネミーは、その時点で戦利品判定をしていない場合は戦利品を獲得できない
- オークの集落にて討伐したエネミーに対しては、戦利品判定を行うことができる
- 戦利品判定が可能な対象は下記のとおりである

戦利品判定の対象:
- ウルフパック x 2
- オーク x 10
- オーク・シューター x 5
- オーク・プリースト x 1
- オーク・ウォリアー x 1
- オーク・ドリフター x 1

> 取り分:
> - 登場するエネミーを、PCたちのPTの活躍に合わせて分割したもの
> - PCが「オークプリーストを2体戦利品判定したい」などと要求した場合は、上位種との交換なら、GMは応じても良い
> - もしくはPCたちが非常に大きな活躍をしていた場合は、目標値3の対人判定を行わせ、成功した場合は1体までのエネミーをPCに渡しても良い
>
> 所持数制限:
> - エネミーの数が多いため、すべての戦利品を持ち帰ることは想定されていない
> - 持ち帰ることのできなかった戦利品は、獲得できない
>     - 後日回収しに来ることもできない

## 集落の探索

- 戦闘終了後、集落を探索できる
- 探索判定を行い、MtVに応じて様々なものを発見する
- MtV 2: オークの作戦会議室
- MtV 3: オークの子供
- MtV 4: 集落の宝物庫

### オークの作戦会議室

- オークが作戦会議をしていたであろう天幕
- 目標値2の探索判定に成功すると、オークの地図を発見できる

| 戦利品       | 目標値 | 価値 | 重量 | 説明                                                     |
| :----------- | -----: | ---: | ---: | :------------------------------------------------------- |
| オークの地図 |      - |    3 |    0 | オークたちが作成していた地図。迷宮の在り処が記されている |

### オークの子供

- 物置小屋のような建物に、オークの子供たちが隠れている
- 子供は5体いる
- 子供といっても、ハーフリングの成人ほどはある体格である
- 武装はしておらず、怯えた様子で冒険者たちを睨んでいる

殺傷する場合:
- PCがオークの子供を殺傷する場合、宣言するだけで倒したこととして良い
- オークの子供に対しては、下記の戦利品テーブルで戦利品判定を行うことができる

| 戦利品     | 目標値 | 価値 | 重量 | 説明                                           |
| :--------- | -----: | ---: | ---: | :--------------------------------------------- |
| 小さな宝石 |      5 |    1 |    0 | 河原で見かけるような綺麗な石の、その大きなもの |

見逃す場合:
- オークの子供たちは森に逃げていく
- オークの子供を全員見逃した場合、PCたちは運判定を行う
- 運判定のMtVが1以下の場合、その者はAPPを恒久的に1点失い、能力「オークとの対話」を獲得する

| 特性・能力     | 効果                                                                                               |
| :------------- | :------------------------------------------------------------------------------------------------- |
| オークとの対話 | オークに対する対人判定が可能。基本的には無意味であるが、GMの裁量でそのPCはオークと意思疎通ができる |

> APPの減少:
> - 魔族を逃がすことは魔族に与することであり、秩序文明に対する背信行為である
> - 不運にもAPPが減少したPCは、魔族に少しだけ近い立場になり、オークとの対話が可能となる

### 集落の宝物庫

- オークたちが略奪した物品を押し込めていた宝物庫
- 廃屋の地下室を改装していたようだ
- 戦利品判定を行い、戦利品を獲得できる

## 結末

### 勝利

- オークの集落の討伐後、街道に出て野営を済ませた冒険者たちは、ギルドに帰還する
- PCたちは報酬金としてWLT 4を獲得し、1点のEXPを得る

### 撤退

- 君たちはオークの潜む森から撤退し、街に帰還する
- オークを1体でも討伐している場合、WLT 1の報酬金を獲得し、1点のEXPを得る
    - 1匹のオークすら狩ることのできない冒険者には、WLTとEXPは与えられない
- 帰還できなかったPCとNPCは、敗北の処理を行う

### 敗北

- すべてのPCは運判定を行う
- MtV 2以下:
    - 君は死亡した
    - 君の死体は魔族の集落に運ばれ、ひと通りの冒涜を受けた後、魔族の食料となった
    - 君の死体の大腿骨は、彼らのペットである狼たちの、お気に入りのおもちゃとなったようだ
- MtV 3:
    - 君は死亡したが、後に冒険者たちの手によって遺体が回収された
    - 望むのなら、蘇生の奇跡を受けることができる
    - 蘇生を受ける場合、WLTを10失い、CONとMNDを恒久的に-1点して復活する
    - 所持品をすべて失う
- MtV 4以上:
    - 君は一命をとりとめ、なんとか街道まで逃げ出した
    - 通りかかった商隊に拾われ、街に帰還する
    - 2種類までの所持品を選び、持ち帰る。それ以外の所持品をすべて失う
- 冒険者たちは1点のEXPを獲得する
