# 地下水路の害獣駆除

## シナリオの概要

```
この依頼を受けるということは、君たちは新米の冒険者なのだろう。

首都ルゥースの地下に広がる広大な上下水の処理機構は、はるか昔にヒュームとドワーフ、ハーフリングの手に造られて久しい。
だが今では、度重なる修理と拡張によって、この地下水路の構造は半ば迷宮と化している。

地下水路には生活排水が大量に流れ込んでおり、これらを餌にする害獣の棲家となっている。
君たちの仕事は、とりわけ生活圏に近い上層に登ってきた害獣を駆除することだ。

水路の下層には迂闊に近づいてはならない。
魔力を伴う不浄は、下水の生態系の中で濃縮されていき、その淀みは恐るべき脅威となる。

上層に登ってくる害獣とは、つまるところそうした魑魅魍魎に追われた者たちに過ぎない。
君たちは、それらを狩る立場だ。

だが、忘れてはならない。
ギルドが斡旋した依頼ということは、冒険者が命を賭して挑むべき脅威ということなのだ。
```

## 冒険準備

### 情報収集

君たちはギルド職員から依頼の説明を受ける。

1. 地下水路の上層に潜んでいる害獣を駆除してほしい
2. これまでに確認されている魔物は、ヴァンプバット、ラージラット、スライムなど
3. 命の危険があるので、下層には立ち入らないこと
4. 指定区画をすべて見回ること。害獣がいなければ、戦闘をしなくても報酬は受け取れる
5. ギルドから、簡易的な地図を渡す。指定区画はそこに記されている
6. 地下水路の鍵を渡す。これは合鍵だが、紛失した場合はWLT 1の賠償が生じる

依頼の条件:

1. 地下水路の上層の害獣を駆除すること
2. 指定区画をひと通り巡り、ギルドに報告したら一人あたりWLT 1を報酬として支払うこと
3. 害獣を10匹以上を駆除した場合、追加で一人あたりWLT 1を報酬として支払うこと
4. 期間は今日1日、ギルドの窓口が閉じるまでに報告をすること
5. 冒険で得られた戦利品は、各自の自由にして良いこと

巡回の条件:
- 巡回地点には巡回者の名前を記録する必要がある
- 巡回地点の壁には、紙束が杭に刺さっている。それが過去の巡回者の署名である
- 冒険者は手持ちの紙に代表者の名前を記入し、そこの杭に紙を重ねるように縫い付ける必要がある
- 巡回の証明として、自分たちの3回前に巡回した者の名前を読み、ギルドへ報告する
- 名前は異なる場合もあるので、1箇所回って完了、とはならないので注意が必要である

追加調査
- PLがさらなる情報を求めた場合、目標値2の対人判定を要求する
- 成功した場合、下記の情報を得る
    1. 水路は薄暗い。巡回路には光源はあるが、準備として灯りを持ち込んだほうが良い
    2. 雨水と汚水の合流式の下水路である。衛生状態が悪いので、毒や瘴気への対策があると良い
    3. 厳しい場合は1回地上に戻るのも手。1度までの撤退であれば、1日の期間で可能なので、無理をしないように
    4. 地下水路の詳しい様子は、他の冒険者に聞いてみるのもいいかもしれない
    5. 巡回路を外れた先にも害獣がいるかもしれない。強制はしないが、倒してくれると助かる

先輩からのアドバイス:
- 先輩の冒険者に地下水路のことを聞く場合、目標値3の対人判定を要求する
    - この手の依頼で向かう地下水路は毎回異なるため、同じ現場に行ったことのある冒険者を探すのは難しいからだ
- 成功した場合、下記の情報を得る
    1. 地下水路には古代の仕掛けが残っていて、動かすと水路の足場を上げたり落としたりできる
    2. 上手く使えば、効率的に区画を回れるかもしれない
    3. 奥の方、3番目の巡回地点から西に少し進んだところに、危険なスライムがいた
    4. 自信があれば挑むのも良いが、対策は怠らないようにした方が良い
    5. 少なくとも、自分たちは撤退を決めた。今なら勝てるだろうけど、駆け出しには荷が重そうだと当時は感じた
    6. 下水に装備を落とすと、価値が落ちてしまう。石鹸を持ち込んでおけばすぐに洗える

### R1: 入口

- ギルドからほど近い都市の外れに行くと、鉄格子の嵌められた、水路の入口がある
- 依頼受託時に預けられた鍵を使うことで、鉄格子の中に入れる
- 地下へと向かう通路がある
- その先には長い階段があり、地下水路へとつながっているようだ

---

### S1: 階段

- 長い階段を降りる
- 段々と、下水から立ち上る悪臭が漂ってくる
- 匂いに敏感な者は、その悪臭に顔をしかめるだろう
- 2階建ての家ほどの高さがある階段を降りると、地下水路にたどり着く

---

### R2: 水路の開始地点

- 階段を降りてすぐの場所。下水の匂いが強く漂う
- 水路には下水が流れるくぼみの横に、人が歩ける側道がある
- 下水に落ちない限りは、この側道を歩くことができそうだ
- 道は、まっすぐ進んでいる

照明:
- 下水道には、等間隔で魔石を利用した光源が設置されている
- 明かりに困ることはないだろう

> 下水に落ちた場合:
> - 下水は酷い汚泥であり、ここに身体が落ちた場合は強度4の病気を受ける
> - さらに、酷い汚泥にさらされた装備品は、価値を1失う
>     - 携行している武器、盾、防具のすべてが対象となる
>     - 複数携行している場合は、すべての装備品が価値を1点失う
>     - 価値が0以下の装備品は、それ以上価値を失わない
>     - 装備品以外のアイテムは価値を失わない
>     - 石鹸を持っている場合、1時間以内に使用すれば価値を取り戻せる
> - このルールは、今後の下水探索で常に適用する
> - PLが下水のペナルティの開示を要求する場合、目標値3の知識判定を行う
>     - 成功すると、PLは下水に落ちた際の効果を把握できる
>     - MtV 2で失敗した場合、病気を受けることと、装備品が汚損することだけはわかる
> 
> 水判定:
> - 下水は水としても扱うことができる
> - 水魔法や氷魔法のコストを低減する

### R16 白骨死体のある水路:

- 何の変哲もない水路だが、通路の片隅に白骨死体がある
- 白骨死体を調べる場合、目標値2の知識判定を行う
- 成功すると、この白骨死体はヒュームのものであること、そう古くない死体であることがわかる
- MtV 3で成功すると、こうした死体はスライムなどによって肉を溶かされることでできたものであることもわかる
    - PCたちは、この付近にスライムがいるのではないかと推測することができる

遭遇:
- P10→P18と進行する場合、P18で遭遇となり、逆に進行した場合はP10での遭遇となる
- スライム x 3との遭遇となる

増援と奇襲:
- スライムが1体、天井に潜んでいる
- 戦闘開始時、PCたちは感知判定を行う
    - MtV 3で成功すると、そのPCは天井に潜むスライムに気がつくことができる
    - 戦闘前にPCたちがスライムの奇襲に警戒するようなロールプレイをしていた場合、この目標値を2としても良い
- スライムは1ラウンド目の敵後衛ラウンドで、ランダムな対象のPCの頭上から「捕食」による強襲をかける
    - この時の対象は、他のスライムからの攻撃を受けていないPCを優先的に選択すると良い
    - スライムは、この時対象にしたPCと同一のエリアに出現する
- スライムの存在に気がついている場合、この攻撃への回避判定は、さらに+DEXx2dのボーナスを得て振って良い
    - さらに、気がついていた場合はこのスライムを包囲することができる
    - 包囲されたスライムは、包囲ペナルティ2以上が生じていると、攻撃のダイスを失うため何もできなくなる
- スライムの存在に気がついていない場合、この攻撃への回避判定を行うことはできない
    - この場合、PC陣営に出現したスライムをPCが包囲することはできない
    - このスライムは、次のラウンドでは強襲したPCに対して「捕食」で追撃を仕掛ける
