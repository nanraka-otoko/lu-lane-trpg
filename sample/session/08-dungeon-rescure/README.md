# 冒険者の救援

## シナリオの概要

```
君たちは冒険者としての実績を上げて、栄誉や信頼を勝ち得て来た頃だろう。
様々な冒険の中で、時に死線をくぐり抜けたこともあっただろう。

「ゴブリンが出たんだってさ！ さっさと行こうぜ、俺達なら楽勝だろ？」

冒険者ギルドにいると、そんな会話が聞こえてくる。
駆け出しの冒険者のようだ。彼らはこれから、村人に目撃されたというゴブリンの討伐に向かうのだそうだ。

冒険者の仕事とは命がけの仕事だ。
下級の魔族、小規模な迷宮。
多くの冒険者たちは帰還するが、戻らぬ者たちもいる。

彼らは決して弱かった訳では無い。
愚かだったわけでも、ないのかもしれない。
だが敗れた。

先発の冒険者が敗北するということは、彼らでは敵わない脅威がそこにあるということだ。
そして、それを討ち果たすのもまた、冒険者の仕事なのである。
```

## 冒険準備

### 情報収集

ギルドは実績のある冒険者を対象に、先発のパーティの救援部隊を募っている。
依頼の内容はこうだ。

1. 既知の小規模な迷宮に現れたゴブリンの群れを討伐するために向かった先発のPTが全滅した
2. 逃げ帰った冒険者は、「迷宮は全然小規模じゃなかった！」と語る
3. ゴブリンを数体撃破したが、その後にオークに襲われたとのこと
4. PTは全滅し、1人だけが逃げ帰ってきた
5. ギルドは直ちに救援部隊を組織し、救援に向かわせたい
6. そのため、君たちが適任と判断して依頼を差し出した

この話を聞くと、生き残りの冒険者を案内される。

冒険者は、以下のように語る:
1. 冒険者は「シルナシ」という、バードフォークの女性
    - 愛称は「ルナシー」という。PCが聞くと教えてくれる
2. シルナシはPTの斥候を務めていた
3. 撤退できたのは、自分の運が良かったから
4. PTは4人。3人を迷宮に残して撤退してきた
5. 残りのメンバーの詳細を説明する
    - 若年のマーフォーク女性の修道女、ウワブレ
    - 若年のマイセリアンの男戦士、ゼンマシ
    - 壮年のリザードマンの男戦士、ゴワメン

> データについて:
> - 冒険者PTのデータは、シナリオに添付したキャラクターシートを参照のこと

全滅したシチュエーションについても語る:
1. 迷宮の上層、B1Fでの探索中、B2Fに逃げるゴブリンを見つけた
2. そいつを追いかけたところ、オークの奇襲を受けた
3. PTは散り散りになり、自分だけは逃げ帰ることができた
4. 残りのメンバーがどうなっているのかはわからないが、すぐに救援に向かいたい

同行者:
- シルナシは、救援隊として冒険に向かう君たちに同行することを希望する
- PCたちはシルナシを同行させても良いし、ギルドで待機させても良い

> GM向け:
> - PCの戦力を増強する意図でもシルナシが同行する
> - PLが拒否する場合は同行させなくても良い

### 迷宮の地図と情報

- シルナシから迷宮の地図と、情報が得られる

```
=== 1F ===

                        +----+      +----+
                        |    |------|    |
                        +-+--+      +----+
                          |
                          |   ↓ B1F
                          |
=== B1F ===
                          |
+----+      +----+      +-+--+
|    |------|    |--  --|    |
+----+      +-+--+      +-+--+
              |           |
              |           |
              |           |
            +-+--+      +-+--+
            | A  |      | D  |
            +-+--+      +----+
              |
              |  ↓ B2F
              |
=== B2F ===
              |
            +-+--+      +----+      +----+      +----+
            |    |------|    |------| B  |------| C  |
            +----+      +-+--+      +----+      +-+--+
                          |                       |
```

A:
- 迷宮の最深部と思われていた場所
- 実際には、ゴブリンの一体がこの奥に逃げていった
- 回転式のドアとなっていて、石壁のように見える壁を押すことで、くるりと開いた

B:
- 実際にオークに襲われた場所
- 逃げ出したゴブリンを討伐した後、ここを探索している途中に、「背後から」オークの軍団に襲われた
- シルナシは倒れた仲間を見捨て、迷宮の奥へと逃げた

C:
- シルナシが隠れていた場所
- オークの資材置き場のような場所であり、隠れる場所として使った
- 背の高い棚に登り、天井付近に隠れていた
- オークたちが去った後、もと来た道を辿って迷宮を脱出した
- 脱出したとき、Bの部屋にはもう、仲間たちの姿はなかった

D:
- 上層を探索していたときに発見した部屋
- 略奪したであろう樽やら木箱やらが雑多に積み上げられていた部屋
- おそらくやつらの宝物庫なのだろうと思っていた
- 今思うと、あれだけの物品はゴブリンだけでは持ち寄れなかったはずだ
    - それに気がついていたら、オークを警戒できたかもしれない、とシルナシは続ける

### 迷宮への移動

- ギルドから早馬の使用許可が得られる
- PCたちは早馬に乗り、迷宮の近傍にある村へと移動する
- 村への移動は半日ほど所要する

> 移動手段について:
> - ケンタウロスや飛行種族など、素早い移動手段を持つPCについては、ロールプレイでそれらの移動方法を採用させても良い
> - 合計でかかる移動時間は、シナリオの進行上では1時間として扱う
>
> 本来の距離:
> - 荷馬車などに随伴し、2日かけて移動する距離の村である

### 移動描写の省略

- GMは「PCたちがたまたま村に立ち寄っていて、そこで依頼を受けた」ということにしても良い
- その場合は、ギルドでのやり取りを省略し、そのままシルナシからの依頼を受けて迷宮に向かう

## 迷宮近傍の村

- ゴブリン退治を依頼していた村
- 村にたどり着く頃には、日が落ちて夜になろうとしている
    - 時刻にして17:00である
- 村人に迎えられ、馬を小さな厩舎に預ける

### 村人からの情報提供と協力

- シルナシの話と大きく変わることはない
- 村人たちは冒険者たちの身を案じており、宿を用意してくれる
- PCは村で休息を取るか、そのまま迷宮に向かうかを選べる
- 休息を取る場合は、翌朝4:00に村を発つ
- そのまま迷宮に向かう場合は、せめて食事だけでもと食料を渡してくれる

| 戦利品 | 目標値 | 価値 | 重量 | 説明                                                                                             |
| :----- | -----: | ---: | ---: | :----------------------------------------------------------------------------------------------- |
| 食料   |      - |    0 |    0 | 村人がくれた食料。堅焼きのパンと干し肉、チーズ。料理しなくても食べられるようにとの気遣いのようだ |

> 時間の制限:
> - このシナリオでは、時間経過によって迷宮内の冒険者たちの結末が変動する
> - 迷宮から帰還したシルナシ、迷宮からなんとか脱出したゼンマシは、時間に依らず救出できる
> - ゼンマシは迷宮の前で倒れており、夜間に行動することで早期に救出ができ、野営によりHPとMPを回復させることができる
>     - ゼンマシは特性「群体」によって生還している
>     - ゼンマシの子実体は、オークたちの「前菜」として食されている
> - ウワブレは気絶した状態で迷宮のオークたちに囚われている。
>     - マーフォークである彼女はオークたちの「ごちそう」であり、生きた状態で保管されている
>     - ウワブレは翌日の18:00に食料にされる
> - ゴワメンは死亡しており、翌日の12:00に食料にされる

### 迷宮への移動

- 迷宮のある山林へと入っていく
- 迷宮までの道のりはおよそ半日足らずと聞いている

探索と迷宮への移動:
- 目標値3で探索判定を行い、成功すると山林を素早く抜けることができる
    - 誰か一人でも成功すれば良い
- 成功した場合、迷宮への移動は6時間で行える
- 失敗した場合、迷宮への到着が4時間遅れ、10時間を所要する

夜に行動している場合:
- 暗視のないキャラクターは、探索判定の目標値が4となる
- 光源を持っている場合は、探索判定の目標値が3となる代わりに-2dのペナルティを受ける

### 迷宮の入口

- 夜に出発した場合は24:00に、翌朝に出発した場合は10:00に迷宮の入口に到着する
- 探索判定に失敗している場合、夜出発では4:00、翌朝出発では14:00での到着となる

倒れている冒険者:
- 迷宮の入口にたどり着くと、マイセリアンの若年男性が迷宮の近くで倒れているのを発見する
- この男性はシルナシの仲間の「ゼンマシ」である
    - シルナシが同行している場合は、即座に名前が明らかになる
    - シルナシが同行していない場合は、依頼を受ける際に聞いた情報で正体を推測する
- ゼンマシは衣類を着けておらず、裸の状態で地面に伏している

ゼンマシの状態:
- HPが-3点の状態で気絶している
- 特性「群体」により、菌糸の本能でなんとか迷宮から脱出している状態
- 所持品をすべて失っている
- HPを1以上まで回復すると、目を覚ます

ゼンマシの情報:
- オークたちに襲われた状況は、シルナシの話した内容と同じ
- おぼろげな感覚だが、菌糸体を逃がすことで、なんとか迷宮の外まで出ることができた
- ゴワメンとウワブレはそのままオークにやられて、迷宮の奥に引きずられていった

### 野営

- PCは迷宮に突入する前に、野営をすることができる
- 夜間から行動している場合、すでに24:00 ～ 4:00となっている

夜間に行動し、野営をせず迷宮に入る場合:
- PCたちは不眠により、すべての判定に-1dのペナルティを受ける
- 食事がある場合は、それを食べることで飢餓のペナルティを回避できる

野営を行う場合:
- PCは翌日の6:00～10:00まで野営を行い、その後に迷宮に突入する
    - 野営の時間はPCの任意に決めて良い
- 睡眠時間が6時間未満の場合は、短時間の野営となり、睡眠時間の不足で野営ロールに-2dのペナルティが与えられる
- 睡眠時間が4時間未満の場合は、極めて短時間での野営となり、睡眠時間の不足で野営ロールに-4dのペナルティが与えられる
- 詳細は野営のルールを参照すること
- この野営における襲撃は発生しない
    - ただし、GMはPCに対して十分な見張りをロールプレイさせることが望ましい

### 時間経過のルール

- 部屋を探索・通過するたび、10分の時間を所要する
    - 部屋を探索せずに素通りする場合は、10分を所要する
    - 部屋を探索してから移動する場合は、20分を所要する
- 通路の移動には時間を所要しない
    - 通路の移動は、部屋の移動時間に含める

迷宮への侵入時刻:
- 0:00 夜間行軍して最速で迷宮に到着し、野営なしで突入する場合の時刻
- 4:00 夜間行軍し、遅れて迷宮に到着し、野営なしで突入する場合の時刻
- 6:00～10:00 夜間行軍し、野営した後に突入する場合の時刻
- 10:00 村で休息を取り、最速で迷宮に到着した場合の時刻
- 14:00 村で休息を取り、遅れて迷宮に到着した場合の時刻

時間経過で起きるイベント:
- 12:00 R10の隠し食料庫に置かれたゴワメンの死体が食料にされる
- 18:00 R18の迷宮の最奥部で囚われているウワブレが食料にされる
- 食料にされたNPCは遺体ごと失われ、救助・蘇生はできない

### H3 隠し扉 / R10 隠し食料庫:

H3 隠し扉:
- R11の壁が回転式のドアとなっていて、石壁を押すことで回転して開く

R10 隠し食料庫:
- R11から分厚い壁を隔ててすぐ隣にある部屋
- この部屋に侵入すると、オークとの遭遇となる

遭遇:
- オーク x 3、オーク・シューター x 2、オーク・ブッチャー x 1
- オーク・ブッチャーは解体用の大斧を持った、シナリオ専用のエネミーである

> オークの作戦:
> - オークはこの迷宮の隠し扉を利用し、R11からP6に進んだ者を背後から奇襲している
> - 迷宮の奥に逃げるゴブリンは、わざとP6でアラートの罠を起動し、R10にいるオークに合図を送っている
> - 「隠し扉を見つけて迷宮の奥に来たやつは、そのすぐ先にも新たな隠し扉などあるはずがない」と疑わないだろう、というオークの知略である

ゴワメンの遺体:
- 12:00までにこの部屋に侵入した場合、ゴワメンの遺体が石机の上に寝かせられている
- ゴワメンは死亡している。遺体を回収することができる
- 蘇生手段がある場合は、この場で蘇生しても良い
- 12:00以降でこの部屋に侵入した場合、ゴワメンの遺体は解体されて食われている
    - リザードマンの鱗皮や骨のみが無惨に散らかっている
- 12:00ちょうどにたどり着いた場合は、ゴワメンがオーク・ブッチャーに処理される直前ということにして良い
    - この場合、ゴワメンの遺体は回収できる

### P12 大扉 / R18 迷宮の最奥部

P12 大扉:
- 鉄製の大扉
- 開けるには目標値4の筋力判定に成功する必要がある
    - この判定は協力判定として行うことができる
- 失敗した場合、開けるために体力を消耗してしまう
    - 次の遭遇でのイニシアチブ判定に-4dのペナルティを受ける

R18 迷宮の最奥部:
- 一際大柄なオークと、その取り巻きたちとの戦闘

遭遇:
- オーク・チャンピオン x 1
