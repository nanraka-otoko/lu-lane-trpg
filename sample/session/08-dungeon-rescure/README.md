# 冒険者の救援

## シナリオの概要

```
君たちは冒険者としての実績を上げて、栄誉や信頼を勝ち得て来た頃だろう。
様々な冒険の中で、時に死線をくぐり抜けたこともあっただろう。

「ゴブリンが出たんだってさ！ さっさと行こうぜ、俺達なら楽勝だろ？」

冒険者ギルドにいると、そんな会話が聞こえてくる。
駆け出しの冒険者のようだ。彼らはこれから、村人に目撃されたというゴブリンの討伐に向かうのだそうだ。

冒険者の仕事とは命がけの仕事だ。
下級の魔族、小規模な迷宮。
多くの冒険者たちは帰還するが、戻らぬ者たちもいる。

彼らは決して弱かった訳では無い。
愚かだったわけでも、ないのかもしれない。
だが敗れた。

先発の冒険者が敗北するということは、彼らでは敵わない脅威がそこにあるということだ。
そして、それを討ち果たすのもまた、冒険者の仕事なのである。
```

---

## セッション情報

- 想定PC数: 3〜4人
- 想定脅威度: 3～5
- 想定プレイ時間: 3〜4時間

| :---------------- | ----: |
| 依頼報酬          |  2～5 |
| 戦利品 (ドロップ) |  2～4 |
| 戦利品 (探索)     |  2〜5 |
| 合計              | 6〜14 |

| フェイズ     | 時間         |
| :----------- | :----------- |

- ギルドまで帰還した冒険者から依頼を受ける形となりますが、森の中や村で偶然に冒険者と出会ったことにしても構いません

## 冒険準備

### 情報収集

ギルドは実績のある冒険者を対象に、先発のパーティの救援部隊を募っている。
依頼の内容はこうだ。

1. 既知の小規模な迷宮に現れたゴブリンの群れを討伐するために向かった先発のPTが全滅した
2. 逃げ帰った冒険者は、「迷宮は全然小規模じゃなかった！」と語る
3. ゴブリンを数体撃破したが、その後にオークに襲われたとのこと
4. PTは全滅し、1人だけが逃げ帰ってきた
5. ギルドは直ちに救援部隊を組織し、救援に向かわせたい
6. そのため、君たちが適任と判断して依頼を差し出した

この話を聞くと、生き残りの冒険者を案内される。

冒険者は、以下のように語る:
1. 冒険者は「シルナシ」という、バードフォークの女性
    - 愛称は「ルナシー」という。PCが聞くと教えてくれる
2. シルナシはPTの斥候を務めていた
3. 撤退できたのは、自分の運が良かったから
4. PTは4人。3人を迷宮に残して撤退してきた
5. 残りのメンバーの詳細を説明する
    - 若年のマーフォーク女性の修道女、ウワブレ
    - 若年のマイセリアンの男戦士、ゼンマシ
    - 壮年のリザードマンの男戦士、ゴワメン

> データについて:
> - 冒険者PTのデータは、シナリオに添付したキャラクターシートを参照のこと

全滅したシチュエーションについても語る:
1. 迷宮の上層、B1Fでの探索中、B2Fに逃げるゴブリンを見つけた
2. そいつを追いかけたところ、オークの奇襲を受けた
3. PTは散り散りになり、自分だけは逃げ帰ることができた
4. 残りのメンバーがどうなっているのかはわからないが、すぐに救援に向かいたい

同行者:
- シルナシは、救援隊として冒険に向かう君たちに同行することを希望する
- PCたちはシルナシを同行させても良いし、ギルドで待機させても良い

> GM向け:
> - PCの戦力を増強する意図でもシルナシが同行する
>     - なるべく同行させること
> - PLが拒否する場合は、目標値4の対人判定で説得する
>     - 成功した場合、シルナシはPCたちに一任してギルドで待機となる
>     - 失敗した場合は、PCたちはシルナシの覚悟に押され、同行を認める

ギルドからの依頼と報酬の説明:
- 件の迷宮の踏破と魔族の討伐が依頼の主目的である
- 迷宮探索の報酬は一人あたりWLT 2。これをPCたちが受け取る
- 追加で、ギルドからは人員救助のために報酬を出す
    - これは迷宮の追加の探索分を兼ねた報酬である
    - 冒険者を1人救助するごとに、一人あたりWLT 1を出す
    - 3人の冒険者をすべて救出できた場合は、通常の報酬と合わせてWLT 5が報酬となる
- 冒険者たちの容態は、死体でも問わない
    - 蘇生可能な状態で回収できた場合は、救出できたものとみなす
- 遺品を回収した場合は、もとの持ち主の冒険者に返却する
    - その冒険者を救出できなかった場合は、PCたちが獲得できる
- シルナシが同行する場合、冒険中に得られた戦利品は、PCたちとシルナシとで分配する
    - 先発の冒険者たちは死亡している可能性が非常に高いため、その蘇生費用も兼ねてのシルナシへの分配となる
    - 端数が出た場合は、シルナシに優先して分配する
    - 例: PCが4人でWLT 6の戦利品を得た場合は、シルナシ含めてWLT 1ずつ5等分し、端数のWLT 1はシルナシに分配する
    - これはシルナシが希望したことではなく、冒険者ギルドからの指示である
    - シルナシが同行しない場合は、戦利品はすべてPCたちで分配して良い

- 実際にオークに襲われた場所

- 上層を探索していたときに発見した部屋

- おそらくやつらの宝物庫なのだろうと思っていた
- 今思うと、あれだけの物品はゴブリンだけでは持ち寄れなかったはずだ
    - それに気がついていたら、オークを警戒できたかもしれない、とシルナシは続ける

### 迷宮への移動

- ギルドから早馬の使用許可が得られる
- PCたちは早馬に乗り、迷宮の近傍にある村へと移動する
- 村への移動は半日ほど所要する

> 移動手段について:
> - ケンタウロスや飛行種族など、素早い移動手段を持つPCについては、ロールプレイでそれらの移動方法を採用させても良い
> - 合計でかかる移動時間は、シナリオの進行上では1時間として扱う
>
> 本来の距離:
> - 荷馬車などに随伴し、2日かけて移動する距離の村である

### 移動描写の省略

- GMは「PCたちがたまたま村に立ち寄っていて、そこで依頼を受けた」ということにしても良い
- その場合は、ギルドでのやり取りを省略し、そのままシルナシからの依頼を受けて迷宮に向かう

## 迷宮近傍の村

- ゴブリン退治を依頼していた村
- 村にたどり着く頃には、日が落ちて夜になろうとしている
    - 時刻にして17:00である
- 村人に迎えられ、馬を小さな厩舎に預ける

### 村人からの情報提供と協力

- シルナシの話と大きく変わることはない
- 村人たちは冒険者たちの身を案じており、宿を用意してくれる
- PCは村で休息を取るか、そのまま迷宮に向かうかを選べる
- 休息を取る場合は、翌朝4:00に村を発つ
- そのまま迷宮に向かう場合は、せめて食事だけでもと食料を渡してくれる

| 戦利品 | 目標値 | 価値 | 重量 | 説明                                                                                             |
| :----- | -----: | ---: | ---: | :----------------------------------------------------------------------------------------------- |
| 食料   |      - |    0 |    0 | 村人がくれた食料。堅焼きのパンと干し肉、チーズ。料理しなくても食べられるようにとの気遣いのようだ |

> 時間の制限:
> - このシナリオでは、時間経過によって迷宮内の冒険者たちの結末が変動する
> - 迷宮から帰還したシルナシ、迷宮からなんとか脱出したゼンマシは、時間に依らず救出できる
> - ゼンマシは迷宮の前で倒れており、夜間に行動することで早期に救出ができ、野営によりHPとMPを回復させることができる
>     - ゼンマシは特性「群体」によって生還している
>     - ゼンマシの子実体は、オークたちの「前菜」として食されている
> - ウワブレは気絶した状態で迷宮のオークたちに囚われている。
>     - マーフォークである彼女はオークたちの「ごちそう」であり、生きた状態で保管されている
>     - ウワブレは翌日の18:00に食料にされる
> - ゴワメンは死亡しており、翌日の12:00に食料にされる

### 迷宮への移動

- 迷宮のある山林へと入っていく
- 迷宮までの道のりはおよそ半日足らずと聞いている

探索と迷宮への移動:
- 目標値3で探索判定を行い、成功すると山林を素早く抜けることができる
    - 誰か一人でも成功すれば良い
- 成功した場合、迷宮への移動は6時間で行える
- 失敗した場合、迷宮への到着が4時間遅れ、10時間を所要する

夜に行動している場合:
- 暗視のないキャラクターは、探索判定の目標値が4となる
- 光源を持っている場合は、探索判定の目標値が3となる代わりに-2dのペナルティを受ける

### 迷宮の入口

- 夜に出発した場合は24:00に、翌朝に出発した場合は10:00に迷宮の入口に到着する
- 探索判定に失敗している場合、夜出発では4:00、翌朝出発では14:00での到着となる

倒れている冒険者:
- 迷宮の入口にたどり着くと、マイセリアンの若年男性が迷宮の近くで倒れているのを発見する
- この男性はシルナシの仲間の「ゼンマシ」である
    - シルナシが同行している場合は、即座に名前が明らかになる
    - シルナシが同行していない場合は、依頼を受ける際に聞いた情報で正体を推測する
- ゼンマシは衣類を着けておらず、裸の状態で地面に伏している

ゼンマシの状態:
- HPが-3点の状態で気絶している
- 特性「群体」により、菌糸の本能でなんとか迷宮から脱出している状態
- 所持品をすべて失っている
- HPを1以上まで回復すると、目を覚ます

ゼンマシの情報:
- オークたちに襲われた状況は、シルナシの話した内容と同じ
- おぼろげな感覚だが、菌糸体を逃がすことで、なんとか迷宮の外まで出ることができた
- ゴワメンとウワブレはそのままオークにやられて、迷宮の奥に引きずられていった

> 合流:
> - ゼンマシはPCたちについていくことを希望する
> - PCが拒否する場合は、ゼンマシは状況を伝えるために村に帰還する
>
> GM向け:
> - ダンジョンのエネミーが強力なため、ゼンマシも探索に参加させることが望ましい
> - PCの回復用のリソースを大きく奪うため、PCに余力がない場合は帰還をさせても良い

### 野営

- PCは迷宮に突入する前に、野営をすることができる
- 夜間から行動している場合、すでに24:00 ～ 4:00となっている

夜間に行動し、野営をせず迷宮に入る場合:
- PCたちは不眠により、すべての判定に-1dのペナルティを受ける
- 食事がある場合は、それを食べることで飢餓のペナルティを回避できる

野営を行う場合:
- PCは翌日の6:00～10:00まで野営を行い、その後に迷宮に突入する
    - 野営の時間はPCの任意に決めて良い
- 睡眠時間が6時間未満の場合は、短時間の野営となり、睡眠時間の不足で野営ロールに-2dのペナルティが与えられる
- 睡眠時間が4時間未満の場合は、極めて短時間での野営となり、睡眠時間の不足で野営ロールに-4dのペナルティが与えられる
- 詳細は野営のルールを参照すること
- この野営における襲撃は発生しない
    - ただし、GMはPCに対して十分な見張りをロールプレイさせることが望ましい

### 時間経過のルール

- 部屋を探索・通過するたび、10分の時間を所要する
    - 部屋を探索せずに素通りする場合は、10分を所要する
    - 部屋を探索してから移動する場合は、20分を所要する
- 「探索」とは、探索判定を行うことを意味する
    - 判定を行わない調査や目視については、時間を所要しないものとして良い
    - 戦闘があった場合も、戦利品判定をしない場合は時間を所要しないものとして良い
- 通路の移動には時間を所要しない
    - 通路の移動は、部屋の移動時間に含める
    - 通路内で能動的に探索判定を行う場合は、10分を所要する

迷宮への侵入時刻:
- 0:00 夜間行軍して最速で迷宮に到着し、野営なしで突入する場合の時刻
- 4:00 夜間行軍し、遅れて迷宮に到着し、野営なしで突入する場合の時刻
- 6:00～10:00 夜間行軍し、野営した後に突入する場合の時刻
- 10:00 村で休息を取り、最速で迷宮に到着した場合の時刻
- 14:00 村で休息を取り、遅れて迷宮に到着した場合の時刻

時間経過で起きるイベント:
- 12:00 R10の隠し食料庫に置かれたゴワメンの死体が食料にされる
- 16:00 R18の迷宮の最奥部で囚われているウワブレが殺害される
- 18:00 R18の迷宮の最奥部で死亡しているウワブレが食料にされる
- 食料にされたNPCは遺体ごと失われ、救助・蘇生はできない

> GMのヒント:
> - 1時間=6回の移動・探索として管理すると良い
> - 突入時刻をもとに、時間経過で生じるイベントが何回の移動・探索で発生するかをあらかじめ計算しておくと、プレイする際にわかりやすいだろう
>
> 探索の大成功:
> - 探索判定のMtVが、目標値を越えている場合は、その探索は素早く行えたこととして良い
> - 例えば、目標値3の探索判定でMtV 4を出したPCがいた場合、その探索は短時間で完了し、10分の所要時間がなかったものとして良い

### 選択のジレンマ

- R18での戦闘を終えた後、救出したNPCの安全のため、PCたちはすぐに帰還することになる
    - 未探索の領域がある場合も、PCはそのエリアを探索できない
    - GMは1部屋程度までなら追加の探索を認めても良いが、すべてのエリアを回ろうとする場合は、強制的に帰還させること
- R1、R4、R9などの脇道や隠し部屋には戦利品が隠されており、NPCたちの安全と引き換えに探索することで、それらを獲得できる

### R3 入口 / P1 通路:

R3 入口:
- 迷宮の入口
- 大きな岩を掘ったような横穴が入口となっている
- 3mほどの高さの天井で、横幅は3m程度
- 中に入ると、照明がなくて薄暗い
- 入口の大岩をくり抜いた先の壁は石レンガでできているようだ
- 暗視のない者は、照明がない場合、この迷宮では暗闇によってすべての判定に-2dのペナルティを受ける

P1 通路:
- まっすぐ続く通路
- R3から道幅はほとんど変わらず、天井も3mほどの高さがある
- 道はわずかに傾斜しており、少しずつ地下に向かっているようだ
- 通路の床は水たまりで満ちている
- この道には汚泥の床の罠があり、踏むと防具の価値を失ってしまう

汚泥の床:
- 先頭を歩くPCは目標値3の感知判定を行う
    - 成功すると、水たまりにまぎれて汚泥だまりがあることに気がつく

| :----- | -----: | ---: | ---: | :--------- |

> 鍵:
> - R20の私室を開くための鍵
> - オーク・チャンピオンの私室である
> - 使用せずに迷宮外に持ち出した場合は、特に用途はなくなる

迷宮からの脱出:
- 迷宮の最奥部に潜むオーク・チャンピオンを討伐したPCたちは、救出したNPCたちをつれて迷宮を脱出する
- PLが追加の探索を望む場合は、GMは1部屋の探索までなら認めても良い
    - 例えば、見つけた鍵を利用して私室の扉を空けても良い
    - 探索によって隠し部屋を発見した場合は、そこも探索して良い
    - それ以上の探索は、救助対象のNPCを危険に晒すため、冒険者の義務として許可されない
- PLがどうしても追加の探索をしたいと言う場合は、目標値4での対人判定を行わせる
    - 成功した場合は、追加で1部屋までの探索を認める
    - これは、帰還が遅れた冒険者がギルドに言い訳をした、という判定である
    - 失敗した場合は、PCたちは強制的にギルドに帰還する

## 結末

### オーク・チャンピオンを討伐した場合

- 依頼の主目的である迷宮の踏破・魔族の討滅が果たされたと認定される
- PCたちは依頼の報酬であるWLT 2を受け取る
- すべてのPCは、1点のEXPを獲得する

### 魔族の手紙を持ち帰った場合:

- ギルドが解読したところ、これは魔族の指揮官であるオークとのやり取りをしている手紙のようだ
- 魔族の根拠地を絞り込むための情報として、ギルドはこの手紙をWLT 3で買い取る
- PCたちは戦利品にWLT 3を加える
    - 一人ひとりの報酬ではなく、戦利品扱いであるため、PT内で分配する必要がある

### シルナシが死亡した場合

- 依頼主の死の代価として、蘇生費用をPCが負担する
    - PCたちがそれぞれWLTを供出し、WLT 10を支払う
- シルナシの死体を回収できなかった場合は、特に費用の負担は生じない

### ゼンマシの救助

ゼンマシが生還した場合:
- PCたちはWLT 1を受け取る

ゼンマシが死亡した場合
- PCたちはWLT 1を受け取る
    - ゼンマシは救助対象ではあるが、死体での回収でも構わないというのが依頼の内容である
    - 冒険中、ゼンマシが何かしらの要因で死亡した場合でも、死体を回収できていれば救出扱いとなる
- 蘇生費用はPCたちは払わなくて良い
- ゼンマシが死亡し、死体も放置した場合は、追加の報酬は得られない

### ゴワメンの救助

ゴワメンの死体を回収できた場合:
- PCたちはWLT 1を受け取る
- 冒険中にゴワメンを蘇生した場合は、蘇生費用が浮く
    - 後述の追加EXPを得る基準でWLT 10点分有利になる

ゴワメンの遺骨を回収した場合:
- 報酬は発生しないが、ギルドからは感謝される
- 冒険中に取り戻したゴワメンの所持品は、PCたちが戦利品として獲得しても良い

### ウワブレの救助

ウワブレを生存した状態で救助した場合:
- PCたちはWLT 1を受け取る
- ウワブレの蘇生費用が浮き、追加のEXPを受け取る基準が易化する

ウワブレの死体を回収した場合:
- PCたちはWLT 1を受け取る

ウワブレの遺骨を回収した場合:
- 報酬は発生しないが、ギルドからは感謝される
- 冒険中に取り戻したウワブレの所持品は、PCたちが戦利品として獲得しても良い

### アイテムの返却

- 冒険中に先発の冒険者たちのアイテムを取り戻した場合、PCたちはそれらのアイテムを返却する
- 持ち主が不明のアイテムについては、すべてPCたちが獲得する
- 持ち主が蘇生不可能な状態な場合は、PCの戦利品としてして扱う

> 所持品の返却:
> - 死亡したNPCは蘇生費用を払う必要があり、その返済が必要である
> - 冒険者として再起するためにも、彼らの所持品を返却する義務がギルド所属のPCには生じる
> - 所持品が一杯だったり、重すぎて持てない物品については、無理に持ち帰る義務はない
>     - あくまでも、持ち帰れたら返してあげる、という程度の努力義務である

### NPCを全員救出し、かつPCが返却したアイテムの合計WLTがNPCの蘇生費用以上だった場合

- NPCの蘇生費用は1人あたりWLT 10である
- 返却したアイテムの合計WLTが、その蘇生費用以上だった場合は、NPCたちは借金なしで蘇生できる
- PCたちは深く感謝される
- 偉大な救出劇を成し遂げたPCたちは、追加で1点のEXPを得る

> NPCが全員蘇生済の状態で生還した場合:
> - 蘇生費用はWLT 0となり、この結末の条件を満たすため、追加のEXPが得られる

### 報酬の合計

この冒険で得られる最大の報酬:
- 主目的の達成: WLT +2, EXP +1
- 冒険者の救助: WLT +3
- アイテムの返却と蘇生費用の確保: EXP +1

### 敗北

- すべてのPCは運判定を行う
- MtV 2以下:
    - 君は死亡した
    - 君の死体は魔族の食料となり、蘇生を受けることはできなくなった
- MtV 3:
    - 君は死亡したが、後に冒険者たちの手によって遺体が回収された
    - 望むのなら、蘇生の奇跡を受けることができる
    - 蘇生を受ける場合、WLTを10失い、CONとMNDを恒久的に-1点して復活する
    - 所持品をすべて失う
- MtV 4以上:
    - 君は一命をとりとめ、なんとか逃げ出してギルドに帰還する
    - 2種類までの所持品を選び、持ち帰る。それ以外の所持品をすべて失う
- 冒険者たちは1点のEXPを獲得する

NPCの結末:
- 生存しているNPCも同様に判定を行う
- 死亡しているNPCは、MtV 2の結末となる
