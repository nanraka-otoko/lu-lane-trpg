# 行商人の荷馬車の護衛

## シナリオの概要

```
街道を行く荷馬車は、文明圏を結ぶ物流の動脈だ。
布、穀物、金物、塩、薬。生活に必要なあらゆる物資が、行商人の手で運ばれている。

だが街道は安全ではない。
野盗、魔獣、崩れた路面、壊れた車軸。荷馬車を止めるものは幾らでもある。

護衛の依頼は、冒険者にとって最も馴染み深い仕事のひとつだ。
報酬は悪くない。道中の食事も付く。何より、旅の経験になる。

今日もギルドに、護衛の依頼が出ている。
隣町までの2日間。大したことはないだろう。
```

---

## セッション情報

- 想定PC数: 3～4人
- 想定脅威度: 1～2
- 想定プレイ時間: 2～3時間

### 1PCあたりの報酬の目安

| 項目               |  WLT |
| :----------------- | ---: |
| 依頼報酬           |    2 |
| 戦利品 (ドロップ)  | 1～2 |
| 合計               | 3～4 |

### セッション所要時間の目安

| フェイズ           | 時間         |
| :----------------- | :----------- |
| 冒険準備           | 15分         |
| 街道の旅 - 1日目   | 30分 ～ 45分 |
| 野営               | 20分 ～ 30分 |
| 街道の旅 - 2日目   | 30分 ～ 45分 |
| 隣町到着           | 10分         |
| 合計               | 約2～3時間   |

### 備考

- 簡易的なシナリオです
- 「03 薬草採取の護衛」などのシナリオと組み合わせ、ボリュームアップしても構いません
- ギルドから「今日はこんな依頼があります」といった形式で複数の依頼を提示し、PLに選ばせる方法を採用しても良いでしょう

## 冒険準備

### 情報収集

君たちはギルド職員から依頼人を紹介される。

行商人のハンス。陽気な中年の男だ。
日に焼けた顔に人懐っこい笑みを浮かべ、PCたちに握手を求めてくる。

依頼の内容はこの通りだ。

1. 荷馬車を隣町まで護衛してほしい
2. 道のりは街道を2日間
3. 荷は布と雑貨。特に高価なものはない
4. 報酬は一人あたりWLT 2
5. 食事はハンスが用意する

> GM向け: PLがさらなる情報を求めた場合、目標値2の対人判定を要求する。
>
> 成功すると:
> 1. この街道で先月、野盗が出たという噂がある
> 2. 街道は整備されているが、最近の雨で路面が荒れている箇所がある
> 3. ハンスは慣れた行商人だが、以前に一度だけ荷を奪われたことがある。それ以来、護衛をつけるようにしている

---

## 街道の旅 - 1日目

- 朝、街を出発する。ハンスが御者台に座り、老馬がゆっくりと荷馬車を引く
- ハンスは道中よく喋る。行商の苦労話、各地の名産品、街道沿いの宿の評判など、話題は尽きない

> GM向け: 
> - ハンスは気さくなNPCとして運用して良い
> - PLが話しかければ何でも答えるし、PLが黙っていても適度に話題を振る
> - PCの種族や装備に興味を示し、「それはどこで手に入れたんだ？」などと聞いてくる

### イベント1: 壊れた車輪

- 昼過ぎ、荷馬車が路面の窪みに嵌まり、車輪の軸が外れる
- ハンスは困った顔で車輪を見つめる
- 「すまない、ちょっと手を貸してくれないか」

車輪の修理には2つの方法がある:
- 目標値4の筋力判定: 荷馬車を持ち上げて車軸を嵌め直す
    - 協力判定が可能
- 目標値3の修理判定: 道具を使って丁寧に修理する
- PCごとに、どちらか一方のみ判定可能
- 誰か一人でも成功すれば修理が完了し、30分ほどで出発できる

全員失敗した場合:
- ハンスと協力して時間をかけて修理する。2時間のロスが生じる
- 到着が遅れ、野営の時間が2時間減る。次の野営ロールに-2dのペナルティを与える

### イベント2: 街道の分岐

街道が二手に分かれている:
- 右の道: 本街道。遠回りだが整備されている
- 左の道: 近道。森の中を抜ける旧街道。半日ほど短縮できる
- ハンスは「近道は使ったことがない。お前たちに任せる」と言う

探索:
- 目標値3の探索判定に成功すると、下記情報が得られる
    - 左の道には獣の足跡が多い。森の中を抜けるため、野生動物との遭遇リスクが高い
    - MtV 4以上の場合、この足跡は大柄の猪であることもわかる
    - 右の道は旅人の足跡が多く、安全そうだ
- 失敗した場合、どちらの道も特に違いはなさそうに見える

遭遇:
- 左の道を選んだ場合、ホーンドボア x 2と遭遇する
- この遭遇に対し、PC全員は目標値3の感知判定を要求し、失敗すると奇襲を受ける
- 獣の足跡を発見している場合、感知判定の目標値は2となる
- 戦闘後、近道を通ったため、野営の時間が2時間増える。次の野営ロールに+2dのボーナスを得る
- 戦利品は、荷馬車に預けることができる

## 野営

- 日が暮れ、街道沿いの開けた場所で野営する
- ハンスは手慣れた様子で焚き火を起こし、鍋で煮込み料理を作る
- 「ぜひに食ってくれ。お前らには明日も護衛してもらうわけだからな」
- PCは食料を消費しなくて良いが、美食を求める場合は食料を供与しても良い

### 焚き火の夜

- 焚き火を囲んで、ハンスとの会話が可能

ハンスに話しかけると:
- 行商人になったきっかけ: 父親が行商人だった。自然と跡を継いだ
- 隣町について: 港がある交易都市で、活気がある。ギルドの支部もある
- 冒険者への思い: 「お前たちがいるから、俺たちは安心して荷を運べる。感謝しているよ」

> ハンスの会話:
> - 「父が行商人だったんだ。それで自然と、俺も行商人をしていたよ。気がついたらね」
> - 「この荷馬車は、新調したものだけどな。父から譲り受けた荷馬車は、以前、街道で奪われちまった」
> - 「港町への行商は儲かるからな。野盗たちも分かっているんでしょう。我々を襲うことに旨味があるってことを」
> - 「俺たちはお前たち冒険者のおかげて、こうして安心して商売ができるんだ。感謝しているよ」

### 襲撃

- キャンプでは、夜間の見張りをPCが分担する
- 見張りの割り振りをPLに相談させる

> 強行軍のペナルティ
> - 野営の処理に従い、PCは4時間以上寝ないと、翌日にペナルティが生じる
> - 野営は8時間を使用できるため、GMは見張りの順序をPLに宣言させること

遭遇
- 深夜、空腹の狼がキャンプに近づいてくる
- ウルフパック x (PC数 x 2) と遭遇する
- 4時間経過時点で見張りをしていたPCは、目標値3の感知判定を行う
    - 焚き火を炊き続けている場合は、目標値が2となる
- 成功すると、狼たちの接近に気が付き、睡眠中の仲間を起こせる
- 失敗した場合、奇襲を受けて戦闘を開始する
- 戦闘終了後、PCたちは眠りについて良い
- PCたちは起床時、野営ロールを振り、ボーナスを得る

## 街道の旅 - 2日目

- 翌朝、出発する
- 隣町まであと半日の距離だ。天気は良い

### イベント3: 野盗の襲撃

- 街道を進んでいると、急に声が聞こえた
- 「ハハッ！ 今日もツイてるぜ。こうしてバカどもがノコノコと来てくれてよ！」
- 野盗たちが姿を現す。野盗の一人が話を続ける
- 「さ、その荷馬車を置いていきな。たんまり溜め込んでるんだろ？」
- 野盗たちには統制がないが、自身ありげな表情を浮かべている
- 「お前ら、逆らわない方が良いぜ？ なんたって俺らにはこの方がいるんだからよォ！」
- そう言うと、大剣を携えた大柄な男が姿を現す
- 「このお方はよォ！ 元冒険者で"剛力のジョー"って呼ばれてたスゴいお方なんだよ！ 死にたくねぇならさっさと失せな！」
- 野盗たちはワイワイと騒いでいる。"剛力のジョー"は、静かに話してくる。
- 「そういうことだ。金目のものを置いていくなら、手出しはしない」

PCにはいくつかの選択肢がある:
- 交渉する: "剛力のジョー"にWLTを10を支払うことで、見逃してもらえる
    - 目標値:4の対人判定に成功すると、額をWLT 6までまけてくれる
- 荷物を渡す: 荷馬車を失う。依頼報酬は得られない
- 戦う: 戦闘を開始する
    - 戦闘する場合、野盗 x 4、ならず者の用心棒 x1 と戦闘となる
    - この野盗との遭遇では、PCは奇襲を実行できない

> GM向け:
> - ならず者の用心棒と野盗の3人は前衛に、もう1人の野盗は後衛に配置する
> - 後衛の野盗はナイフを投げて攻撃を行い、前衛の野盗たちもPCの後衛に対してナイフを投げる
> - PCの人数が多い場合、後衛に野盗を追加する

## 隣町到着

- 野盗を退け、荷馬車は無事に隣町に到着する
- ハンスは荷降ろしを済ませ、PCたちに礼を言う
- 「いやあ、助かった。お前たちがいなかったらどうなっていたことか」
- ハンスは約束通り報酬を支払い、加えて握手を求めてくる
- 「また護衛が必要な時は、お前たちに頼むよ」

> GM向け:
> - 隣町にギルドの支部がある場合、そこで依頼完了の報告ができる
> - 帰路の護衛を別途依頼される展開にしても良いが、このシナリオの範囲外とする

## 結末

### 成功

- 依頼達成の報酬: 1PCあたり WLT 2
- 冒険で得た戦利品を換金できる
- 冒険者たちは1点のEXPを獲得する

### 交渉に応じた場合

- 依頼達成の報酬: 1PCあたり WLT 2
- 戦利品は得られない
- 冒険者たちは1点のEXPを獲得する

### 獣との戦いに敗北

- 獣との戦いで敗北し、全滅した場合、すべてのPCは運判定を行う
    - 死亡しているPCは達成値に-1の補正を受け、死の結末のみを受け入れる
- 達成値0
    - 君は死亡した
    - 君は野生動物たちの食事となり、蘇生されることもなかった
- 達成値1
    - 君は死亡した
    - しばらくした後、死体が回収される
    - 望むのなら、蘇生の奇跡を受けることができる
    - 蘇生を受ける場合、WLTを10失い、CONとMNDを恒久的に-1点して復活する
- 達成値2以上
    - 君は一命をとりとめ、なんとか都市に撤退した
- いずれの場合でも、君たちは依頼者を守ることはできなかった
- 報酬は受け取れない
- 冒険者たちは1点のEXPを獲得する

### 野盗との戦いに敗北

- 野盗との戦いで敗北し、全滅した場合、すべてのPCは運判定を行う
    - 死亡しているPCは達成値が0となり、死の結末のみを受け入れる
- 達成値0
    - 君は死亡した
    - 野盗たちは君の所持品をすべて剥ぎ取り、その辺に捨て置いた
    - 君の死体は回収されることなく、野鳥や獣たちに食われた
- 達成値1
    - 君は気絶し、野盗に捕まった
    - 君は身代金としてWLT10を失うことで解放される
    - 身代金が払えない場合、野盗の一味となる。APPを恒久的に-2し、2点のEXPを獲得し、解放される
- 達成値2以上
    - 君は一命をとりとめ、なんとか都市に撤退した
- いずれの場合でも、君たちは依頼者を守ることはできなかった
- 報酬は受け取れない
- 冒険者たちは1点のEXPを獲得する
