# 下級魔族の討伐

## シナリオの概要

```
君たちは、新米の冒険者だ。
様々な夢や野望、目的のために。もしくは日銭を得るために。
首都ルゥースの西に位置する、この冒険者ギルドの門を叩いたのだ。

掲示板には、たくさんの依頼や迷宮の情報が所狭しと張り出されている。
ギルド職員は、新人向けの依頼を君たちに案内した。

「下級魔族の討伐」

西の村の近くの迷宮に魔族が住み着いたので、討伐してほしい。
ルゥレーンでは、よくある依頼だ。

かつて攻略された小規模の迷宮を、混沌の手勢が棲家とする。
攻略済の迷宮と言っても、秩序の文明から離れた森林、山中、平原などにそれらは点在している。
故に共和国の警備兵ではすべてを監視することは難しい――そうした問題を解決するのが、冒険者の存在意義のひとつだ。

君たちは、この依頼を受けることにした。
```

---

## セッション情報

- 想定PC数: 3～4人
- 想定脅威度: 1～2
- 想定プレイ時間: 2～3時間

### 1PCあたりの報酬の目安

| 項目              |  WLT |
| :---------------- | ---: |
| 依頼報酬          |    2 |
| 戦利品 (ドロップ) | 1～2 |
| 戦利品 (探索)     | 1～2 |
| 合計              | 4～6 |

### セッション所要時間の目安

| フェイズ   | 時間       |
| :--------- | :--------- |
| 冒険準備   | 15分       |
| 村の訪問   | 15分       |
| 迷宮探索   | 60～120分  |
| 帰還・報告 | 15分       |
| 合計       | 約2～3時間 |

### 備考

- 各種判定や、迷宮探索における様々なギミックを搭載した小規模なダンジョンです
- 冒険の準備を含め、2～3時間程度の所要を想定しています
- キャラクターメイキングの説明に1時間を所要すると仮定した場合、4時間程度を所要することに注意してください

## 冒険準備

### 情報収集

ギルドは依頼者の村人から、事前に情報を得ている。
1. 村の近くに1匹のゴブリンが現れた
2. 村人は農具でゴブリンを追い払った
3. ゴブリンは森の奥に逃げていった
4. その方角には、かつて攻略された迷宮がある

依頼の条件:
1. 迷宮内に潜んでいるであろうゴブリンたちを討伐すること
2. 報酬は一人あたり、WLT 2 をギルドから支払うこと
3. 冒険で得られた戦利品は、各自の自由にして良いこと

> さらなる情報: 目標値2の対人判定に成功すると、ギルド職員が下記のことを教えてくれる
> 1. 村に現れたゴブリンは斥候である可能性が高い
> 2. リーダー格がいるだろう
> 3. 村についた後、2日程度で完了できる規模の依頼であるとの見込み
> 4. 依頼者の村人とは良い関係を築くとよい

> 先輩冒険者の助力:
> - キャラクターメイキング時にWLTが0で装備が買えないPCがいた場合に限り、GMは下記のことをしても良い
> - ギルドは物品の貸与を、特に駆け出し相手には禁止している
> - 先輩の冒険者たちであれば、中古の武器を貸してくれるかもしれない
> - 若年の冒険者に限り、目標値2の対人判定に成功したら、価値1の武器を1つ貸してもらえる
> - 借りた武器は、セッション終了時に返却する

### 物品調達

- 出発までに1時間の猶予がある
- PCたちは、それぞれの判断で価値1までの品を購入できる
    - 価値0のアイテムも購入できる

> ギルドからの前借りは不可:
> - ギルドは、駆け出しに対しては前借りを固く禁止している
> - 依頼の失敗や死亡、踏み倒しなどで過去に損害が生じており、PCに対しても前借りは一切認めない

## 村を訪れる

- 荷馬車で1日。西の村に到着する

村人からの情報:
1. ゴブリンは追い払って以来現れていない
2. 一行を一晩泊める用意がある

目標値2の対人判定に成功すると:
- 劣化したポーションを1つ入手
- 食料セットを1つ入手

対人判定に失敗した場合:
- 迷宮の往復には時間がかかるから、食料があった方が良い
- 森での狩猟する手もあるが、可能なら食料を持っていくことを推奨する
- WLT 1で、PT全員分の食料セットを買うことができる

| 戦利品             | 目標値 | 価値 | 重量 | 説明                                                                                      |
| :----------------- | -----: | ---: | ---: | :---------------------------------------------------------------------------------------- |
| 劣化したポーション |      - |    - |    0 | 劣化したヒールポーション。このセッションに限り、ポーションとして使用可能。HPを2点回復する |
| 食料               |      - |    - |    0 | 1PT分の食料                                                                               |

## 迷宮への移動

- 村から迷宮まで徒歩で半日。森の中を進む
- 森の中でPCは目標値2の探索判定を行う
    - 誰か一人でも成功すると、良質な薬草を1つ発見する

| 戦利品     | 目標値 | 価値 | 重量 | 説明                                                                                      |
| :--------- | -----: | ---: | ---: | :---------------------------------------------------------------------------------------- |
| 良質な薬草 |      - |    - |    0 | 新鮮な状態でしか使えない薬草。このセッションに限り、非戦闘状態で使用可能。HPを2点回復する |

> 複数のPCが探索判定に成功しても、良質な薬草は合計で1つまでしか発見できない

森を進むと、迷宮の入口が見えてくる:
- 迷宮の入口は森の中に口を開けた石造りの洞窟
- 入口にゴブリンの足跡が多数見られる
- 目標値2の探索判定に成功すると、ゴブリンの足跡から10匹以上は存在することがわかる

> ゴブリンの足跡:
> - 出入りしているのが10匹以上ということは、実際にはもっといる可能性が高い、ということである
> - GMはこのことをPCに伝えなくても良いが、高い達成値で成功したPCにはそれとなくヒントを与えても良い

---

### 野営

迷宮の入口にたどり着いた頃、辺りはすっかりと暗くなっている:
- PCたちはここで休憩を実行できる
- 休憩せずに乗り込んだ場合、PCたちは不眠と空腹によりすべての判定に-2dのペナルティを受ける

PCは食料を利用して休息を実行できる:
- 処理の詳細は `system/06-adventuring/02-camp.md` を参照すること
- この休息で襲撃は発生しない

> 休憩の有無について:
> - この処理が冗長に感じる場合は、GMの判断で休息をスキップしても良い

### R1: 入口

- 薄暗い、石造りの部屋。下へと続く階段が見える
- 暗視の能力を持たない者は、光源がないと階段の先を見通すことができない
- 階段の先を覗くと、部屋があるように見える

> 光源:
> - GMは、PLに対してたいまつやライトの使用を推奨しても良い
> - PCに光源を得る手段がない場合、目標値2の運判定により、たいまつを1本与えても良いだろう

---

### S1: 階段

- 薄暗い階段。奥に進むに連れ、地上の光が遠ざかっていく
- 手元の明かりだけが、冒険者たちの頼りとなる

> - この階段には特に何もない

---

### R2: 狭い部屋

- 狭い部屋。隅を見ると、土や泥の汚れに苔やキノコが繁殖している

> 探索:
> - PCが探索判定や感知判定を希望する場合、GMは目標値3で振らせる
> - 成功すると、PCはネズミの鳴き声を聞き取る

---

### P1: 通路

- 暗い通路。光源がなく、暗視を持たない者には-2dのペナルティが生じる。

遭遇:
- 通路を進むと、ラージラットxPC数と遭遇する
- 光源または暗視がある場合、通常の遭遇となる
- 視界が確保できていない場合、目標値2の感知判定を行い、失敗すると奇襲を受ける

---

### R3: 広間

- ゴブリンたちの棲み処。焚き火の跡、食べ残し、散乱した寝床
- 部屋に入る前に目標値2の感知判定に成功すると、ゴブリンたちの存在を察知できる

> 感知:
> - 達成値が3以上の場合、5～6体のゴブリンがいる気配を察知できる
> - 達成値が4以上の場合、さらに「隠れ潜んでいそうなゴブリンの気配」も感じる

遭遇:
- 部屋に入ると、ゴブリンx4、ゴブリンアーチャーx2と遭遇する
- 身構えながら部屋に入ることで、イニシアチブに+1dのボーナスを得る
- この遭遇では、PCたちはゴブリンを奇襲することはできない
- 戦闘開始から3ラウンド目の開始時、通路P3からゴブリンが1体加勢する
    - 増援のゴブリンは敵エリアの後衛に出現し、自身のターンで前衛に移動する

> 調整:
> - PCが過度に有利な場合、増援のゴブリンを増やしても良い
> - 逆に最初のラウンドでPCが過度に不利な状態となっている場合、増援をなくしても良い

探索:
- 戦闘後、部屋を探索できる
- 目標値2の探索判定に成功すると、下記の情報を得る
    - P4への通路には、足跡が少ない
    - P2への通路には、ゴブリンの足跡が多数あり、往復している

---

### P4: 通路

- 狭い通路。冒険者たちは、1列に並んで歩く必要がある

> - GMはここでPLに隊列の順序を聞いておくと良い

トラップ:
- 通路にはトリップワイヤーが仕掛けられている
- 先頭を歩くPCは、目標値3の感知判定に成功すると、罠を感知できる
- 失敗すると、先頭を歩いているPCに矢の罠が作動する
- 罠の矢は正面から飛来する
- 矢は対象に防護・回避不可で5点のダメージを与える

> 罠の検出:
> - 罠の検出の目標値は、厳しめの基準となっている
> - PCが能動的に探索判定をする場合、目標値2で検出させて良い
> - 能動的に探索するのであれば、先頭以外のPCも判定に参加できる
> - トリップワイヤーを見抜いているのであれば、この通路は判定なしで安全に出入りできる
>
> クロスボウ:
> - INTを用いた探索判定で達成値が3以上の場合、PCは「ダートトラップにはクロスボウが使われている」ことがわかる
> - GMは、罠を解体できた場合、クロスボウを獲得できるかもしれない、という情報を与える
> 
> トラップの解体
> - 罠を解体する場合、目標値3での解除判定を要求する
> - 達成値が1以下の場合、解除に失敗し、罠が作動する
> - 達成値が2の場合、罠は動かないが、解除はできない
> - 達成値が3の場合、罠を解除できる
> - 達成値が4以上の場合、罠に仕掛けられたクロスボウを入手できる

### R6: 宝物庫

- ゴブリンが略奪した物資が貯め込まれた部屋
- そのほとんどはガラクタだが、探索することでなにか価値のあるものが見つかるかもしれない

探索:
- 目標値2の探索判定に成功すると、PCは略奪品の山を見つける
- 略奪品の山にはタイニーミミックが潜んでいる
- PLが略奪品の山に対して知識判定を能動的に要求した場合、GMは許可して良い
- 目標値2の知識判定に成功すると、タイニーミミックの存在に気がつく

> 探索判定での代用:
> - 探索判定で達成値が4以上の場合、知識判定に成功したものとして扱って良い

遭遇:
- PCが略奪品の山に近寄った場合、タイニーミミックx3との遭遇となる
- タイニーミミックの存在に近寄ったPCに目標値3の感知判定を要求する
- 成功した場合、タイニーミミックに気が付き、通常の遭遇となる
    - PC側は、近寄ったPCのみがイニシアチブロールを可能
    - 近寄ったPCは前衛に、近寄っていないPCは後衛に配置される
- 失敗した場合、奇襲を受けた状態での遭遇となる
- 知識判定ですでにタイニーミミックに気がついている場合は、PCが奇襲を行うことができる
- 戦闘後、略奪品の山に対して戦利品ロールが可能

| 戦利品           | 目標値 | 価値 | 重量 | 説明                                                   |
| :--------------- | -----: | ---: | ---: | :----------------------------------------------------- |
| 小銭             |      1 |    1 |    0 |                                                        |
| ヒールポーション |      2 |    1 |    0 |                                                        |
| レザーアーマー   |      3 |    1 |    1 | 状態が良好なもの。ゴブリンでは着られなかったのであろう |

### P2: 通路

- 広間の奥に続く通路。ゴブリンの足跡が往復している
- 目標値2の探索判定に成功すると、ゴブリンの小便溜まりが見つかる
    - 冒険者は、嫌な気持ちになる

### R4: 寝床

- ボロ布を敷き詰めたゴブリンたちの寝床
- 目標値2の探索判定に成功すると、ゴブリンたちの足跡はP5に向けて多数往復していることがわかる
- P3に向けては、あまり足跡がないが往復はしている

### P3: 通路

- 寝床を抜けた先の通路
- 薄暗く、悪臭が漂う
- 奥に行き着くと、R5のゴミ溜めに出る

### R5: ゴミ溜め

探索と戦利品獲得:
- 動物の骨や皮、食べ残しのような腐肉が捨てられたゴミ溜め
- ゴミの中にはもしかしたら何かがあるのかもしれないが、手を触れることすら躊躇われる汚物だ
- 戦利品を探す場合は、PCごとに戦利品ロールを行える
- 戦利品ロールをしたPCは強度2の病気を受ける

| 戦利品       | 目標値 | 価値 | 重量 | 説明                       |
| :----------- | -----: | ---: | ---: | :------------------------- |
| ゴミ         |      1 |    0 |    1 | チクショウ！最悪な気分だ！ |
| きれいな指輪 |      4 |    3 |    0 |                            |

> 備考:
> - 病気の状態異常は生命力判定で対抗できる
> - PLが望む場合は再度の戦利品ロールを許可しても良いが、そのPCは次は強度3の病気を受ける
>
> 探索判定:
> - 目標値3の探索判定で、ゴミ溜めの中にきらりとひかるものを発見させても良い
> - この情報がある場合、この戦利品判定に+1dのボーナスを与える

---

### P5: 通路

- 寝床から先へとつながる通路。長い通路の先に、木でできた3mほどの高さの扉がある
- ゴブリンたちの足跡は、この扉の先に向かっている

> 扉に対する探索:
> - PLが感知判定を希望する場合、目標値3で判定させて良い
> - 扉の奥にいるゴブリンの人数が5～6匹だとわかる
> - この判定に成功しても、GMはPCに奇襲を許可してはならない。扉を開けた時点で、ゴブリンたちはPCを感知するからである
> - 達成値が2で失敗した場合、具体的な数はわからないが、ゴブリンの声が聞こえる

---

### R7: 奥の間

- 開けた空間に、何らかの生物の骨を組んでできた玉座があり、ゴブリンが座っている

遭遇:
- 扉を開けるとゴブリンリーダーx1、ゴブリンx3、ゴブリンアーチャーx3、ゴブリンシャーマンx1、レッドキャップx1との遭遇となる
- PCはこの戦闘で奇襲できない

> 調整:
> - PCがかなり有利な状態でこの場所に侵入した場合、GMの判断で前衛のゴブリンを1体追加する、レッドキャップを増やすなどして難易度を上げても良い

- ゴブリンリーダーは後衛に構えており、前衛のゴブリンが撃破されるまで前衛には移動しない
- ゴブリンシャーマンは後衛に待機し、前衛のPCに対してエナジーショットを使用する
- すべてのゴブリンを倒したら、依頼の目標は達成となる

探索:
- ゴブリンの討伐後、PCが探索判定を要求したら、GMは応じる
- 目標値3の探索判定に成功したら、PCは骨の玉座の裏に、1mほどの高さの隠し通路を見つける

> 補足:
> - PLが骨の玉座を調べようとしている場合は、目標値を2にして探索判定をさせて良い

### H1: 隠し通路

- 1mほどの高さの通路であるため、伏せて侵入する
- この通路を2mほど進むとR8にたどり着く

### R8: 隠し部屋

戦利品獲得:
- 隠し部屋には、ゴブリンリーダーが溜め込んでいた戦利品が存在する
- 戦利品判定を行い、戦利品が得られる

| 戦利品     | 目標値 | 価値 | 重量 | 説明                                                  |
| :--------- | -----: | ---: | ---: | :---------------------------------------------------- |
| ガラクタ   |      1 | 0～1 |    0 | 獲得時、2M6を振る。価値はMtV-1となる                  |
| 小さな宝石 |      2 |    2 |    0 |                                                       |
| 古びた地図 |      3 |    ? |    0 | 目標値2の知識鑑定を行う。失敗した場合、未鑑定品となる |

> 古びた地図:
> - 目標値2の知識判定により鑑定可能
> - 成功した場合、この地図は別の迷宮の在り処であることがわかる
> - 価値3の戦利品として売却可能
> - 鑑定できない場合、ギルドにWLTを1支払うことで鑑定が可能
> - その後、価値3の戦利品として売却できる。(差額で2のWLTを得られる)

## 帰還と報告

- 迷宮を出て村に戻る
- 村人が歓迎してくれる
- 村で一泊し、翌日荷馬車で首都に帰還する

報酬の獲得:
- ギルドに報告すると、一人あたりWLT 2の報酬が支払われる
- 古びた地図を持っている場合は、ギルドに報告するとWLT 3がPTに支払われる
- 未解読の場合、ギルドに鑑定料としてWLT 1を支払うことで、その後に売却できる
    - 鑑定しない場合、価値:0のゴミとして、君たちはこの地図を処分する

## 結末

### 成功

- 依頼達成の報酬を得た後、戦利品を換金できる
- 冒険者たちは1点のEXPを獲得する

### 撤退

- PCたちは冒険を諦めて途中で撤退することができる
- ゴブリンを5体以上倒して撤退した場合: WLT 1の報酬を受け取る
- ゴブリンを5体以上倒せなかった場合: 報酬は発生しない
- 一度でも戦闘をしていた場合、冒険者たちは1点のEXPを獲得する
    - ネズミとも戦闘せずに逃げ帰った者には、EXPは与えられない
- 獲得していた戦利品を、報酬とは別に換金できる

### 敗北

#### 死亡したPC

- 運判定を行う
- MtV 0～1
    - 君の死体はゴブリンたちに無惨に食い尽くされ、回収されることなく迷宮の塵となった
- MtV 2以上
    - 後に踏み込んだ冒険者によって、君の死体が回収される
    - 望むのなら、蘇生の奇跡を受けることができる
    - 蘇生を受ける場合、WLTを10失い、CONとMNDを恒久的に-1点して復活する
    - 所持品をすべて失う

#### 気絶したPC

- 運判定を行う
- MtV 0
    - 君は死亡した
    - 君の死体はゴブリンたちに無惨に食い尽くされ、回収されることなく迷宮の塵となった
- MtV 1
    - 君は死亡した
    - 後に踏み込んだ冒険者によって死体が回収される
    - 望むのなら、蘇生の奇跡を受けることができる
    - 蘇生を受ける場合、WLTを10失い、CONとMNDを恒久的に-1点して復活する
    - 所持品をすべて失う
- MtV 2
    - 君は気絶していた
    - ゴブリンたちに酷い扱いを受けたが、なんとか生きている
    - 後に踏み込んだ冒険者によって救助され、一命をとりとめた
    - 所持品をすべて失う
- MtV 3以上
    - 君は命からがら逃げ出した
    - 所持品のうち、ひとつだけを回収することができる
    - 村に帰還して報告する。ほどなくして救助隊が駆けつけるだろう
- 冒険者たちは1点のEXPを獲得する

> GM向け:
> - 運判定を補助する装備品がある場合は、所持品喪失前に運判定をさせても良い
