# 井戸の底に眠るもの

```
ルゥレーンの各地には井戸が点在する。
魔法により飲水を得ることができても、生活用水や農業用水は常に不足している。
井戸は、古代から今日まで、各所で掘られ、活用されている。

だが井戸も無限ではない。
枯れた井戸は木板などで封印され、やがて忘れ去られていく。

そうした古井戸には、時として淀みが蔓延する。
農村で古井戸から魔なる者が湧き出た例は、古今問わず多くの事例があるのだ。
故にその探索は、冒険者の仕事となる。

そして、そのありふれた依頼は、今日もギルドの掲示板に張り出されていた。
```

---

## セッション情報

- 想定PC数: 3～4人
- 想定脅威度: 1～3
- 想定プレイ時間: 1～2時間

### 1PCあたりの報酬の目安

| 項目              |  WLT |
| :---------------- | ---: |
| 依頼報酬          |    2 |
| 戦利品 (ドロップ) | 0～1 |
| 戦利品 (宝)       | 1～2 |
| 合計              | 4～6 |

### セッション所要時間の目安

| フェイズ       | 時間       |
| :------------- | :--------- |
| 冒険準備       | 15分       |
| 目的地への移動 | 15分       |
| 井戸の探索     | 1時間      |
| 帰還・報告     | 10分       |
| 合計           | 約1～2時間 |

### 備考

- 簡易的なシナリオです
- ギルドから「今日はこんな依頼があります」といった形式で複数の依頼を提示し、PLに選ばせる方法を採用しても良いでしょう
- 都市部から農村への移動が含まれますが、街の近くの井戸の探索という建付けにしても構いません

## 冒険準備

### ギルドからの情報

- 君たちは依頼の説明を受ける

依頼内容:
1. 農村の井戸から腐臭が漂い始めた
2. ずいぶんと昔に枯れ、今では使われていない井戸である
3. 異変が起きたのは1週間前
4. 疫病が蔓延したら危険なので、井戸の異変を調査してほしい
5. 報酬は一人あたりWLT 2
6. 村へは荷馬車が出ているので、1時間後に出発してほしい

## 目的地への移動

- 荷馬車に揺られ、2日間の距離を移動する
- 食事は馬車の保存食がもらえる
    - これは依頼の内容に含まれているもので、ギルドからの支給である

## 農村

- 農村に到着すると、依頼者の村長と、井戸の異変に最初に気がついた農民が声をかけてくる

情報:
- 村の外れ、畑の近くにある古井戸から、1週間前から異臭が漂ってきた
- 肉の腐る匂い。土葬された死体が雨で掘り起こされたときのような匂いがする
- 今は蓋をしている状態
- 探索に降りる場合は、縄梯子を上からかける

聞き込み:
- 目標値3の対人判定に成功すると、情報を入手できる
- 村は昔から農村であり、古井戸を使って水を汲み上げていた
- 川からの灌漑ができてからは井戸は使われなくなった

## 井戸

### 井戸の入口

- 井戸には蓋がかかっている
- 外すと、腐臭が漂う
- 井戸の底は、見えないほどに深く、暗い
- 村人が縄梯子をかけてくれる。それを伝って底に降りれる

暗視がある場合:
- 井戸の底に、横穴が見える
- 明かりを投げ込むことで、暗視のないPCでも視認が可能

### 井戸の底

- 20mほどの深さの井戸を降りると、人一人が通れるほどの大きさの横穴がある
- 石積のレンガの一部が崩れているようだ
- 暗闇が続いており、暗視のない者は何かしらの明かりが必要となる

> 明かりを持たない冒険者たちへ:
> - PCが一人も明かりを持っていない場合、GMは一度村に戻り、たいまつを分けてもらうよう指示しても良い
> - その場合、1本のたいまつをもらうことができる

---

## 井戸の底の探索

- 井戸の底は、シンプルな構造となっている

```
[入口] - [遭遇1] -+- [イベント1] - [イベント3] - [遭遇2]
                  |
                  +- [イベント2]
```

---

### 入口

- 井戸の横穴の入口
- 崩れた石レンガのアーチをくぐると、腐臭がより一層濃くなる
- この横穴はそこまで天井が高くないため、飛行可能なキャラクターも、飛行することができない
- 目標値3の探索判定に成功すると、奥から何かが徘徊するような音が聞こえる

---

### 遭遇1: スライムとアンデッドとの遭遇

遭遇:
- ゾンビx3
    - ゾンビの1体はHPが残り3点の状態で遭遇する
- スケルトンx3
- スライムx1

エネミーの操作:
- スライムは後衛でゾンビを捕食しており、1ラウンド目でゾンビを捕食して成長を行う
    - 何かしらの手段で捕食前にゾンビを撃破された場合、ゾンビを撃破したPCを狙って行動を行う
- 残りのエネミーは前衛に配置され、PCの前衛に攻撃を仕掛ける

> 戦闘について:
> - 井戸の底は狭い空間だが、戦闘上では特にエリアの制限などは設けないものとして良い

探索:
- ゾンビに知識判定をした場合、たしかに酷い匂いを纏っているが、これが異臭の原因ではないことがわかる
- PLが探索を希望した場合、GMはそのまま後続の進行の判定をここで実行させて良い

進行:
- 進行方向が、二手に分かれている
- 目標値3の探索判定に成功すると、以下のことがわかる
    - 正面方向には、スライムが這いずったと思わしき痕跡がある
    - 右手の方向には苔が生えており、何かが行き来した痕跡は少ない
- 正面方向に進むと、イベント1に進む
- 右手に進むと、イベント2に進む

---

### イベント1: スライムの沼

- 道を進んでいると、足元がぬかるみ始める
- 先頭を歩くPCは、目標値2の感知判定を行う
    - 成功すると、スライムの沼に気がつく
    - 失敗すると、スライムの沼に脚を突っ込んでしまい、3点のダメージを受ける
- スライムの沼は行く手を阻んでおり、このままでは無事に進むことが難しそうだ

強行突破:
- このスライムの沼は、そのまま足を踏み入れると膝下程度の深さがある
- 強行突破する場合、6点のダメージを受け、防護を-2する
    - この効果は、一方方向に通過するごとに適用される (復路でもダメージを受ける)
- この横穴の天井は低く、飛行可能な者であっても飛行して突破することはできない

コアを破壊する:
- 目標値3の知識判定に成功すると、スライムの沼の核を発見する
- 核に対し、何かしらの遠隔攻撃手段で攻撃することで、スライムを撃退できる
- 撃退したスライムは石や土の隙間に浸透していき、やがて消滅する

スライムを焼く:
- 火炎属性ダメージで10点のダメージを与えることで、スライムを焼き払うことができる

> 選択肢に困る場合:
> - PLが突破方法に悩んでいる場合、GMはPCに知識判定をさせて良い
> - 目標値3で成功した場合、GMは選択肢を提示する
> - 失敗した場合は、GMはPLに強行突破を強制的に選択させても良い
> - 解法は提示した手段以外でも、PLの創意工夫によって認めても良い

### イベント2: 白骨死体の書斎

- 横穴の分かれ道を進むと、黄緑色の淡い光が見えてくる
- その光の元に進むと、光を放つキノコが群生した、小部屋に出る
- 小部屋には古い羊皮紙や本などが置かれている
- 部屋の隅には白骨死体が横たわっている。白骨死体は、ボロボロの服を身にまとっている

探索:
- 目標値2の探索判定に成功すると、呪術家の手記を見つける
- この書斎からめぼしい道具などを探す場合は、戦利品判定を行う

| 戦利品               | 目標値 | 価値 | 重量 | 説明                                       |
| :------------------- | -----: | ---: | ---: | :----------------------------------------- |
| ランプマッシュルーム |      1 |    0 |    0 | 光るキノコ。「魔法の触媒」として代用できる |
| 呪物の杖             |      2 |    1 |    1 |                                            |
| 呪術書               |      4 |    5 |    0 |                                            |

呪術家の手記:
- ボロボロの手記
- ほとんどのページは掠れて読めないが、一部のページだけは読み解くことができる

ヴェルベナの連中は皆バカだ。禁止魔法の研究にこそ、次の時代の光があるというのに。

### イベント3: 増殖する腐肉

- 道の途中、通路を巨大な腐肉が塞いでいる
- この腐肉からは、井戸の外や入口で感じていた、異常な匂いを感じる
- 異臭の原因はこの腐肉と見て間違いはないだろう
- PCたちは、この腐肉を排除することで、異臭の原因を断つことができる
    - この腐肉の破壊が、今回の依頼の目的となる

腐肉の破壊:
- 腐肉はHP30、防護0の標的として扱う
    - 腐肉は1ラウンドごとにHPを5点回復する
    - 火炎属性ダメージを与えたラウンドは、HPが回復しない
- PCたちは各自の攻撃手段でこの腐肉を攻撃する。HPを0以下にすることで、腐肉を破壊して先に進めるようになる
- 腐肉の具体的な情報を得たい場合、目標値3で知識判定を行う
    - 成功した場合、GMはこの腐肉の処理についてPLに説明する

> 腐肉の正体:
> - 呪術と死霊術の行使に失敗し、肉体が無限に増殖を繰り返すようになった研究者の成れの果てである
> - PCがその正体の看破を望む場合、目標値4の知識判定に成功したら、GMはその真実を教えても良い
> - 奥に潜むタイニーコープスたちは、この腐肉の塊を喰らい続けることで、今日まで生きながらえていた

井戸の奥へ:
- 腐肉を破壊すると、井戸はさらに奥へと続いている
- PCが望む場合、この先に進むことができる

### 遭遇2: 子供ゾンビとの遭遇

- 井戸の奥に進むと、黄緑色のキノコが発光している部屋に行き着く
- そこには犬に取り囲まれた、一人の子供が眠っていた
- 彼らに近づくと、アンデッドたちが目を覚まし、冒険者たちを襲う
- タイニーコープスたちとの遭遇となる

遭遇:
- タイニーコープスx1
- ゾンビ犬x4
- スケルトンアーチャーx2

エネミーの操作:
- ゾンビ犬が4体、最初のラウンドでPC側の前衛に移動し、噛みつきを行う
    - GMは、ゾンビ犬の攻撃手段の射程が0である点に注意すること
- 続けてタイニーコープスがPC側の前衛に移動して噛みつきを行う
    - これを阻止するためには、PC側がゾンビ犬を減らしておき、タイニーコープスの移動を包囲で止める必要がある
    - タイニーコープスがPC前衛に移動できない場合は、射程:1の攻撃手段でPCの前衛を攻撃する

探索:
- 戦闘終了後、探索が可能
- 目標値2の探索判定に成功すると、古びた玩具を発見する

| 戦利品     | 目標値 | 価値 | 重量 | 説明                               |
| :--------- | -----: | ---: | ---: | :--------------------------------- |
| 古びた玩具 |      - |    - |    0 | 「アレン」と名前が書かれた古い玩具 |

## 帰還

- 腐臭の原因となっていた腐肉を取り除いた君たちは、村人に歓迎される
- 村の共同浴場を無償で貸し出され、君たちは冒険の汚れを拭うことだろう
- 村で一泊し、翌日に冒険者ギルドに向けて帰還する

古びた玩具:
- 古びた玩具を獲得している場合、村人に見せることができる
- アレンの名前を教えると、翌日に村長が「200年前にこの村にいた子どもの名前だった」と教えてくれる
- アレンの家族の墓地に、その玩具が供えられるという
- 冒険者たちは、墓で祈りを捧げることができる
- 祈りを捧げたPCは、運判定を行う
    - MtV 3以上を出したPCは、敬虔な祈りが真実神に届き、1点のEXPを獲得する

報酬の獲得:
- ギルドに報告すると、一人あたりWLT 2の報酬が支払われる

## 結末

### 成功

- 依頼達成の報酬を得た後、戦利品を換金できる
- 冒険者たちは1点のEXPを獲得する

### 撤退

- PCたちは冒険を諦めて途中で撤退することができる
- 最初の遭遇をした後に撤退した場合: 冒険者たちは1点のEXPを獲得する。報酬は発生しない
- 最初の遭遇をせずに撤退した場合: 冒険者たちはEXPを獲得できず、報酬も発生しない

### 敗北

- すべてのPCは運判定を行う
    - 死亡しているPCは、死の結末のみを受け入れる
- MtV 0:
    - 君は死亡し、後にアンデッドとなって井戸の底をさまよった
    - アンデッドとなった君は、やがて次に現れた冒険者に屠られた
- MtV 1:
    - 君は死亡した
    - 後に死体が回収され、蘇生を受けることができる
    - 所持品をすべて失う
    - 蘇生を受ける場合、WLTを10失い、CONとMNDを恒久的に-1点して復活する
- MtV 2:
    - 君はこの井戸から脱出した
    - 所持品のうち、ひとつだけを回収することができる
- 冒険者たちは1点のEXPを獲得する
