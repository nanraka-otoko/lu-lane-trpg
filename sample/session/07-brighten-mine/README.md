# マナに灯る鉱山

```
ルゥレーンの各地には、とにかく地下洞窟がある。
そこはかつての鉱山であったり、炭鉱であったり、はたまた古代の種族の住処であったりする。
地下洞窟のいくつかは、龍のねぐらだったものだ、とする学説もあるほどだ。

廃坑などの捨てられた洞窟には魔法生物や魔族の棲家となるケースもあり、集落の近くに棲み着いた場合は冒険者に討伐が依頼される。
鉱山の採掘中に危険な地下洞窟に当たったり、原生生物が目覚めたり、といった事例も、各地で数多く報告されている。

いずれにせよ、ルゥレーンにはとにかく洞窟があり、冒険者の出番もあるのである。

さて、今日も今日とて、君たち冒険者一行は冒険者ギルドに集まっていた。

酒場は今日も大賑わい。飲む者、食う者、笑う者。
理由をつけては喜び騒ぐのが冒険者というものだ。

しかし彼らが語る冒険譚の裏側には、冒険の機会を提供する弱者たちの姿があることも忘れてはならない。
酒場に併設されたギルドには、今日も今日とて依頼が舞い込む。

失せもの探し、行商人の用心棒、ゴブリン退治。
君たち一行にはときめかないような依頼ばかりがやってくる。

そんな君たちのもとに、ギルド職員がやってくる。
ドワーフの魔石鉱夫からの依頼のようだが、どうやら訳ありのようだ。

厄介な依頼と踏んだならば、君たちは拒否することもできる。
その場合は今日2番目の"大物"の依頼である、ゴブリンの討伐に向かうことになるだろう。
```

## 冒険準備

### 情報収集

ギルドは依頼者のドワーフの鉱夫から、事前に情報を得ている:
1. 村の魔石鉱山に、変な魔物が住み着いた
2. 鉱夫仲間たちが何人か食われてしまった
3. 鉱石を食う芋虫みたいなやつだった
4. 鉱山は現在、長老の命令で閉鎖している
5. 冒険者たちには、あの怪物を討伐してほしい
6. 村はギルドから3日ほど馬車で行った場所にある、タルシュ村という村落である

知識判定:
- PCはここで、目標値4で知識判定を行う
- 成功すると、鉱石を食らうミミズ状の魔物、マインワームの存在に思い至る

ロールプレイで魔物の情報を聞き出す場合:
- 「大きさは？」「特徴は？」「色は？」など、PLが魔物の特徴を聞いた場合は、知識判定が有利になる
- 魔物の詳細を聞いた場合は、先ほどの知識判定の目標値を-1する
- すでにMtV 3で失敗していた場合は、その知識判定の目標値を-1して成功したこととし、そのPCは魔物の正体に思い至る

依頼の条件:
1. 魔石鉱山に住み着いた怪物を討伐すること
2. 報酬は一人あたり、WLT 3をギルドから支払うこと
3. 冒険で得られた戦利品は、各自の自由にして良いこと
    - ただし、魔石鉱山は村の資源であるため、むやみに荒らさないこと

その他の情報:
1. 村にはこれから1時間後に出発する荷馬車に同乗してもらう
2. 荷馬車の代金は、護衛料ということでギルドが交渉済である
    - 冒険者に対し、特にこの護衛についての報酬金は出ないが、運賃は無料である
    - 村までの街道は比較的安全であるため、その護衛の相場も割安である
3. 荷馬車は都市の外で待機しているので、出発するなら急いでほしい
4. 必要な道具類がある場合は、現地の村で調達してほしい
    - 報酬金のWLT 3の範囲内であれば、前払いの形式で手配するように依頼主と話をつけてある

- 鉱山の地図について聞いた場合は、地図を見せてくれる

地図の情報についての説明:
- 怪物が現れたのは、鉱山の奥部の第三鉱区である
- 鉱山には、西口から入る
- 西口から入ってすぐの広場を直進すると、通路に出る。さらにそこから進むと大広間に出る
- 大広間の分岐のうち、南東方向の通路を進むと第三鉱区に出る
- 広場から分岐すると第一鉱区に、大広間からの分岐で第二、第三鉱区へとつながっている
- 大広間から東の側道に進むと、通路がさらにT字に分岐する
- 直進するとエレベータの区画に、下層に降りる方向へと曲がると第三鉱区に出る
- 第四鉱区の奥に横穴を掘ってあり、そこを進むと第五鉱区に行き着く
- エレベーターは第五鉱区との間で稼働しているが、東口に上がる機材は現在は動いていない

東口とエレベーター:
- 東口は昔に使っていた縦穴であり、現在は使っていない
- エレベーターも放置されている状態
    - 機構は生きているが、動力は通っていない
    - 魔石式の動力であり、動力を入れれば動くだろうが、そもそも東口を使ってない状態
- かつては東口側にも集落があったが、人口減に伴って西側の集落に村人を寄せた
- もともと、外部への輸出も西側の方が有利だったこともあり、西口だけしか使わなくなった
- 鉱山からの鉱石の運搬には、魔石動力のトロッコを用いている
- それぞれの鉱区からは大広間へのトロッコが引いてあり、さらに大広間と広場を結ぶトロッコも引いてある
- 広場とエレベーター区画にもトロッコが引かれている

第一鉱区:
- かつて掘られていた区画
- すでに魔石は枯れており、現在は使われていない

第二鉱区:
- 比較的浅い層に掘られた鉱区
- 現在も使っているが、第三鉱区を主としている

第三鉱区:
- 最深部に位置する鉱区
- 質の良い魔石が多く取れることから、現在でも稼働している鉱区
- 段々と収量が減ってきており、廃棄が検討されている

第四鉱区:
- 魔石以外の鉱物も産出した鉱区
- 汎用金属やヘビーメタルが産出していた
- その後、第五鉱区が拡張され、この鉱区は実質的に通路として用いられるようになった
- 現在は稼働していない鉱区

第五鉱区:
- 第四鉱区の奥で稼働している、比較的新しい鉱区
- 汎用金属の鉱石や魔石が産出する
- 深さ30mごとに横堀りされた、水平坑道である
- 各層は階段に加え、昇降機で移動可能
- 昇降機はエレベーターホールの区画と接続しており、この昇降機は稼働している
- 昇降機で鉱物を持ち上げた後、トロッコで運搬している

鉱山への侵入について:
- 西口からは歩いて入れるので、そちらから行った方が良い
- 東口は暗く、エレベーターも稼働していないので避けた方が良い
- もしも東口から入る場合は、100m以上の垂直登攀が必要となる
    - 飛行種族の場合は飛んで降りられるが、狭い縦穴のため、羽や翼をぶつけないように注意が必要である

昇降機の使用方法や修理方法について:
- PLが昇降機の使用方法や修理方法について質問した場合にのみ回答がある
- 東口、第五鉱区、エレベーターホールには操作盤があり、そのレバーを引くことで昇降機を呼ぶことができる
    - 操作は、先にレバーを引いた側が優先される
- 昇降機内にはレバーがあり、それを引くことで目的の場所に移動できる
    - 昇降機内の操作は、エレベーターホールや各種行き先の昇降場の操作よりも優先される
- 東口用の昇降機は動いていないが、動力を足せばすぐに動かせる
    - 現在は商売的にもここに魔石を投入して維持する必要がないので、使わなくなっている
    - 触媒「魔石」に相当する魔石を、合計でMP10点分も入れれば、昇降機が稼働するはずである

> 昇降機の修理の情報:
> - 東口の昇降機の修理方法を知っている場合は、「エレベーターホール」で行う知識判定に自動成功する

## 鉱山の探索

### 鉱山の構造

```
[西口]                        [東口]
  |                             |
[広場]------[通路1]             |
  |           |                 |
[通路2]     [第一鉱区]          |
  |                 |           |
[大広間]----------[通路4]-----[エレベーターホール]
  |  |              |                |
  |  +----[通路3]   |                |
  |         |       |                |
[第二鉱区]  |       |                |
            |       |                |
            |     [第四鉱区]-------[第五鉱区]
            |            |
          [第三鉱区]---[通路5]
```

事前情報との差異:
- 第一鉱区と通路4の間、第三鉱区と第四鉱区の間に、フロウド・マインワームが掘り進めた道ができている
- 通路5にはフロウド・マインワームの巣穴があり、そこに卵が産み付けられている
    - フロウド・マインワームは、この巣穴を防衛するために鉱山の入口を支配している

鉱山内:
- 鉱山内は灯りの届かない地下であり、暗視の能力を得ていないものは、すべての判定に-2dのペナルティを受ける
- 暗視を持つドワーフが利用する坑道であるため、その灯りは最低限である
- ドワーフは意図して火気を持ち込んでいない
    - これはガスなどで爆発した経験から得た知恵である
    - この鉱山ではガス爆発などのギミックはないが、鉱山の照明事情に関するフレーバーとして活用しても良い

- PLは西口か東口か、好きな入口から侵入できる

### 西口からの侵入

- 「立ち入り禁止」と書かれた看板がある
- 高さ3mほどの大きな横穴であり、ここから鉱山に入ることができる
- しばらく進むと横穴が狭くなり、幅1m、高さ2mほどの通路となる
- この通路は下方向に向けて傾斜しており、鉱脈に向けて下っていることがわかる

> 半強制:
> - 西口からの侵入を基本的なルートとして想定している
> - このルートから侵入した場合、即座にシナリオボスとの戦闘となり、その後に鉱山からの脱出を行う

---

### 東口からの侵入

エレベーターの縦穴:
- こちらの入口から侵入する場合、PCたちはエレベーター用の狭い縦穴を抜けていく
- -4dのペナルティを受けた状態で、目標値3の汎用行動判定を行う
    - 成功すると、縦穴を降りることができる
    - 登攀キットを所持している場合は、目標値を2とする。自動成功はしない
    - 失敗すると落下し、50mの墜落によるダメージを受ける
- 飛行種族の場合は、-4dのペナルティのもとで目標値2の汎用行動判定として良い
    - 穴は狭く、羽ばたくための十分なスペースが確保できない
    - 加えて灯りもないため、飛行種族であっても大きなペナルティを受けることは免れない
- PLが東口からの侵入を希望した場合は、GMはこのルールを開示して良い
    - それでもPLが希望する場合は、こちらからの侵入を許可する

探索:
- 目標値3の探索判定に成功した場合、ルミナスポーションが見つかる
- ポーションを使用した場合、縦穴が十分に明るくなり、暗闇のペナルティを無視できる
- PLたちは、このポーションを獲得した後に西口に戻ってから再度侵入しても良い

| 戦利品             | 目標値 | 価値 | 重量 | 説明                                |
| :----------------- | -----: | ---: | ---: | :---------------------------------- |
| ルミナスポーション |      - |    1 |    0 | 使用した地点を光源にする。1時間持続 |

> 半強制:
> - 東口は、通常の侵入ルートとして想定されていない
> - 魔法や道具を利用することで降下することも不可能ではないが、MPやWLTといったリソースを大幅に消費する
> - GMはPLにリスクが生じることを伝え、それでも希望する場合にのみ、こちらからの侵入でマスタリングする
> - このルートから侵入した場合、広場のシナリオボスを討伐し、その後に普通に鉱山から脱出できる

### エレベーターホール

- 昇降機のある区画
- 第五鉱区への昇降機と、東口への昇降機とがある
- 第五鉱区への昇降機は可動をしており、操作することで移動が可能

第五鉱区の昇降機:
- 稼働している昇降機
- レバーを引くことで、昇降機を呼ぶことができる
- 昇降機の操作は昇降機内のレバーを利用する
    - 行き先は「エレベーターホール」「上層」「中層」「下層」と表示されている

東口の昇降機:
- 停止している昇降機
- 昇降機の近くに「動力炉」と書かれた魔導機械が設置されている
- 目標値3の知識判定に成功すると、修理方法がわかる
    - ドワーフの場合は知識判定に+4dのボーナスが得られる
    - 動力だけが落ちている状態であり、「動力炉」に燃料を追加すれば動作する
    - 触媒「魔石」に相当する魔石を、合計でMP 10点分投入することで再稼働する
- MtV 4以上で成功すると、昇降機は使われていないため、動かしたらトラブルが起きるかもしれないと感じる
    - 途中で止まる可能性がありそうだ、ということがわかる
- この判定は、昇降機の情報を事前に聞いている場合は、自動成功する
    - その場合でも、GMは観察のために判定を行わせて良い
    - 事前に情報がある場合、MtV 4での大成功が、MtV 2で可能となる
- 目標値2の修理判定で、昇降機の修理ができる
    - 修理に成功した場合、「東口への脱出」の運判定に+2dのボーナスが得られる
    - MtV 3以上で成功した場合、運判定へのボーナスが+6dになる
    - ここでの修理は、壊れた状態からの修理でないため、アクシデントの完全な予防はできない (予防的処置であり、根本の処理ではない)

掘削のルール:
- 崩落した入口は防護5、HP50の標的として扱う
- 1度の攻撃判定に対し、1時間を所要する
- 2人同時に攻撃ができるが、その場合は両者の攻撃判定に-2dのペナルティを受ける
- 攻撃判定をしたものは、HPを1点失う
- 標的のHPが1点減少するごとに、1mを掘り進めることができる
- 標的のHPが0以下になったとき、崩落した入口が開通し、脱出できる

> 標的機のHPの開示について:
> - 崩落現場の距離は、PCには未開示の情報として扱う
> - 目標値4の知識判定に成功すると、およそ50mの崩落であると観察からの洞察を得られたこととしても良い
> - 25mまで掘り進めた段階で、後半分程度掘れば開通しそうだ、という情報を開示しても良い

時間について:
- 西口からの脱出を試みた時点で 15:00 を回ったこととして良い
- 1日で脱出できない場合、鉱山内での野営を行う必要がある

### 東口への脱出

- エレベーターホールで東口の昇降機を修理した場合、昇降機を使って脱出できる
- 昇降機に乗り込んだPC全員で運判定を行い、最も高いMtVでアクシデント表を参照する
    - すでに予防的な修理を行っていた場合は、運判定にボーナスが得られる
    - 乗り合わせていないPCはこの判定を行うことはできない

| 運判定MtV | 東口昇降場への距離 | 汎用行動判定の目標値 | 描写・説明                                                                                    |
| --------: | -----------------: | -------------------: | :-------------------------------------------------------------------------------------------- |
|     4以上 |                 0m |                    0 | 昇降機は東口にたどり着いた                                                                    |
|         3 |                10m |                    2 | 昇降機が、後少しのところで止まった。少しの登攀で、十分に登れそうだ                            |
|         2 |                20m |                    3 | 昇降機が止まった。ある程度の高さがあるが、なんとか登れそうだ                                  |
|         1 |                30m |                    3 | まだ先が視えないところで昇降機が止まった。登攀するにも、汎用行動判定に-2dのペナルティを受ける |
|         0 |                50m |                    3 | 昇降機が、半ばで止まってしまった。登るにも下るにも、汎用行動判定に-4dのペナルティを受ける     |

汎用行動判定の処理:
- MtV 4以上の場合、そのまま脱出できる
- MtV 0～3の場合、垂直登攀が必要となる
    - 汎用行動判定に成功することで、そのPCは脱出できる
- 成功したPCはそのまま昇降場まで移動できる
    - 一人でも脱出できた場合は、そのPCの介助や、助けを呼びに行ったことにし、残りのPCも脱出できる
- 失敗した場合、昇降距離の半分の高さから落下する
    - 墜落のダメージを受ける
    - 着地を試みる場合は、その汎用行動判定に-4dのペナルティを受ける

停止した昇降機の修理:
- 内側からは直せない状態である
    - 昇降機に乗り込んだPCは、そのまま脱出を試みるか、外部からの修理を待つ必要がある
    - すべてのPCで乗り込んでいた場合は、修理が不可能となり、直接の脱出を試みるしかない
- エレベーターホールまたは東口にて修理が可能
- 修理する場合は、目標値3で修理判定を行う
    - この判定には1時間を所要する
- 成功すると、昇降機が再稼働する
    - 一度でも停止→修理を行ったら、以降、昇降機はトラブルなく稼働し続ける

### 鉱山内での野営

- 鉱山内は薄暗く、居心地が悪い
- 野営ロールには下記のボーナスとペナルティが生じる

| 状況                   | ボーナス/ペナルティ |
| :--------------------- | ------------------: |
| 鉱山内での就寝         |            常に -2d |
| 寝具のない状態での就寝 |                 -2d |
| 種族がドワーフ         |                 +2d |
| 種族がダークエルフ     |                 +2d |

食料の調達:
- 鉱山内には食料がほとんど自生しておらず、食料の獲得は困難を極める
- PLが食料調達判定を希望する場合、その目標値は5となる

料理:
- 大広間には調理器具があり、そこで食材を料理することができる
- 道具「料理キット」として使用可能。野営ロールに+2dのボーナスが得られる

> 野営について:
> - 西口からの脱出を図る場合に採用すると良い
> - PCたちのHPとMPを回復させつつ、鉱山内で生活するロールプレイを行わせると良い
> - シナリオ上、食料は無限に獲得できるが、3日以上の休息をした場合は「そろそろ食料について心配になってきた」などと描写しても良い
>     - その場合は、不安により野営ロールに-2dのペナルティを与えるなどしても良いだろう

## 結末

### フロウド・マインワームを撃破した場合

- 依頼の標的であるフロウド・マインワームを倒した場合、依頼の報酬としてWLT 3をそれぞれ獲得する
- すべてのPCは1点のEXPを獲得する

> 撃破せずに、何かしらの手段で逃げて生還した場合:
> - 依頼不達成となり、報酬は得られない
> - マインワームの異常個体の情報を仔細に報告することで、情報料としてWLT 1を受け取ることができる
> - 怪物を倒さなかったPCたちには、EXPは与えられない

### 西口が崩落した状態で東口から脱出した場合

- 故障していた東口の昇降機を修理したことにより、感謝料としてWLT 1を追加で獲得する

### 崩落した西口を掘り進めて脱出した場合

- 本来であれば鉱夫たちでやる復旧作業を肩代わりしてくれたことにより、感謝料としてWLT 1を追加で獲得する

> 崩落現場を掘り抜き、さらに東口から脱出した場合:
> - どちらの感謝料も得られる。WLT 2を追加で獲得する
>
> 東口から侵入して西口から戻った場合:
> - 特に追加の報酬は得られない
> - 本来であれば発生したはずの崩落を未然に予防したことにはなるが、それはIFの物語であり、このルートを辿ったPCの功績とは認められない

### 第三鉱区の魔石の鉱脈について報告した場合

- 今回の件で、村に新たなる鉱山資源がもたらされた
- 怪物の襲撃は鉱夫に被害を与えたが、それと同時に恵みも与えていた
- 村人たちは喜び、亡くなった鉱夫たちへの慰霊と、新たな恵みへの感謝のため、宴会を行う

酒宴と飲み比べ:
- 宴会は朝まで続き、キツい酒と濃い料理に舌鼓を打ちつつ、激しい胸焼けに襲われることだろう
- ドワーフの酒飲みたちは、飲み比べを始める。PCはこの飲み比べに参加することができる
- 参加したPCは生命力判定を行う
- MtV 4の場合、飲み比べで健闘することができる
    - EXPを追加で1点獲得する
- MtV 5以上の場合、飲み比べに勝利することができる
    - 大いなる酒飲みとしての名誉が得られる。APPを恒久的に+1する
- それ以外の場合は、普通に酔いつぶれ、翌朝に宿屋で目が覚める

> 生命力判定について:
> - PCたちは、ここで判定にボーナスを与える魔法や奇跡を使用することができる
> - GMは判定後の処理についてPLにあらかじめ説明し、MtV 5以上の大成功を競わせると盛り上がるだろう

### 食料庫から酒樽を獲得した場合

- 鉱山の脱出後、直ちに盗みが発覚する
- 安酒ということで賠償は免除される
- 宴会が開かれた場合、すべてのPCは飲み比べに強制参加させられる
- 飲み比べの生命力判定のMtVが1以下の場合、致命的なまでに酔いつぶれてしまう
    - CON、MND、LUCのいずれかを選び、恒久的に-1する

> 窃盗のペナルティ:
> - 酒樽の窃盗を行った場合、飲み比べへの参加にリスクが生じる
> - MtV 5以上を出した場合は、通常の酒宴と同様にAPP+1の効果が得られる

### 食料庫から火酒を獲得した場合

- 鉱山の脱出後、獲得した本数を目標値に、PCの代表者が運判定を行う
- 運判定に失敗すると盗みが発覚し、2倍の価値で賠償することになる
- 発覚後、野盗のせいにする場合は、獲得した本数-1を目標値に対人判定を行う
    - 成功すると賠償を回避できる
    - 失敗した場合、そのまま賠償する
    - 目標値-1のMtVで失敗した場合、等倍の価値での賠償で済む
- 賠償を拒否した場合、PCたちのAPPを恒久的に-1する
    - 盗賊行為は信用を損なう行為であり、その分のマイナスである

### フロウド・マインワームが生存している鉱山内で敗北

- すべてのPCは運判定を行う
- MtV 2以下:
    - 君は死亡し、フロウド・マインワームの死霊術によってアンデッドと化した
    - 秩序文明の敵対者となった君の肉体は、後に生者の手によって滅ぼされた
- MtV 3:
    - 君は死亡し、死体は鉱山内に捨て置かれた
    - 後に踏み込んだ冒険者によって死体が回収される
    - 望むのなら、蘇生の奇跡を受けることができる
    - 蘇生を受ける場合、WLTを10失い、CONとMNDを恒久的に-1点して復活する
    - 所持品をすべて失う
- MtV 4以上:
    - 君は一命をとりとめ、鉱山内で隠れ潜んでいた
    - 後に踏み込んだ冒険者により救出される
    - 装備品や所持品は失わずに帰還できる
- 冒険者たちは1点のEXPを獲得する

### フロウド・マインワームが討伐された後の鉱山内で敗北

- すべてのPCは運判定を行う
- MtV 0:
    - 君は死亡した
    - 君の死体は運悪く、鉱山内で生じた小規模な崩落に巻き込まれ、蘇生不可能なほどの損傷を負った

    - 君は死亡し、死体は鉱山内に捨て置かれた
    - 後に踏み込んだ冒険者によって死体が回収される
    - 望むのなら、蘇生の奇跡を受けることができる
    - 蘇生を受ける場合、WLTを10失い、CONとMNDを恒久的に-1点して復活する
    - 所持品をすべて失う
- MtV 3以上:
    - 君は一命をとりとめ、鉱山内で隠れ潜んでいた
    - 後に踏み込んだ冒険者により救出される
    - 装備品や所持品は失わずに帰還できる
- 冒険者たちは1点のEXPを獲得する
