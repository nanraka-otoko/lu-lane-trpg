# 迷宮探索

```
迷宮探索の時間だ！

ルゥレーンには、数多の迷宮が存在する。それは冒険者にとってのロマンだ。
無論、秘宝を秩序の文明に持ち帰ることは、ギルドが推奨することでもあるし、市井の者が望むことでもある。

迷宮の情報はギルドに張り出されるが、探索の優先権は発見者と、その所属する支部に割り当てられる。
迷宮探索にはギルドを通じて国からも報酬が出る。秘宝の探索は、秩序文明にとってそれだけ重要なことなのだ。

君たちは、冒険者ギルドに張り出されていた一件の迷宮に目をつけた。
```

---

## セッション情報

- 想定PC数: 3～5人
- 想定脅威度: 3～5
- 想定プレイ時間: 4～6時間

### 1PCあたりの報酬の目安

| 項目              |    WLT |
| :---------------- | -----: |
| 依頼報酬          |   4～8 |
| 戦利品 (ドロップ) |   3～7 |
| 戦利品 (宝)       |  3～10 |
| 合計              | 10～25 |

### セッション所要時間の目安

| フェイズ       | 時間       |
| :------------- | :--------- |
| 冒険準備       | 10分       |
| 目的地への移動 | 20分       |
| 迷宮の探索     | 4～5時間   |
| 帰還・報告     | 10分       |
| 合計           | 約4～6時間 |

### 備考

- 4階層、45部屋からなるダンジョンハックです
- オールドスクール的なダンジョンですが、システム自体が軽量であるため、1セッションで完了する想定です
- 時間内に終わらない場合は、セッションを分割しても良いですし、撤退・報告をして報酬を得たこととしても構いません

## 冒険の準備

### 冒険者ギルドにて

- 新たな迷宮が発見され、PCたちが所属している冒険者ギルド支部が優先探索権を得た
- 支部は、所属する冒険者たちに迷宮の探索を斡旋している
- 迷宮の深度や難易度は不明だが、オークの出入りが確認されている
    - 中位の魔族が棲まう可能性が高いと推測される
    - そのため、中堅冒険者に対して迷宮の探索を依頼することとなった

ギルドから提示される迷宮探索の条件:
1. 迷宮の地図を作成すること
    - 探索の状況に応じ、報酬が追加で支払われる
    - 迷宮の規模にもよるが、最低でWLT 1、最大でWLT 4程度は支払われるのが通例である、と職員が説明する
    - 報告内容によって提示する額が変化するが、概ね1階層を探索するとWLT 1程度のレートである
2. 戦利品は冒険者のものとして良いこと
    - ただし、今回の探索で持ち帰ったもののみに所有権が与えられる
    - 今回の探索で発見したが、持ち帰れなかったものは、後続の回収隊預かりとなり、PCたちの所有権は認められない
3. 秘宝が発見された場合、それはギルドが所有権を持つこと
    - 冒険者のものではないため売却はできないが、報酬金が上乗せされる
    - 冒険者たちはギルドの代理人としての側面を持つため、秘宝を発見した場合はギルドの一員として振る舞え、ということ

> 地図の作成について:
> - 道具の持ち込みを宣言していなくても、暗黙の携行品としてペンと紙を持っていることとして良い ( `system/05-items/` を参照のこと)
> - PCが地図の作成を宣言することが望ましいが、ロールプレイなしでも地図を作成しているものとして扱って良い

迷宮の情報:
1. 迷宮の位置情報とルートは、発見者から報告されている
2. 支部から荷馬車で出発し、3日程度の距離の村を拠点とする
3. そこから1日かけて山中に歩くと、迷宮がある
4. 迷宮探索に1～2日を要するため、村からの往復で3～4日を所要する
5. 村からの往復にて、4日以内に戻らない場合、異常があったとギルドは判断する
    - そのため、迷宮探索の時間制限は2日間である
6. 移動にかかる費用や食料はギルドが負担する

荷物持ちについて:
- 荷運びの要員が欲しい場合、ギルドにお金を支払うことで、人手を雇うことができる
- 一日あたりWLT 1で一人の人足を雇うことができる
- 今回は村から迷宮への往復で見込み3～4日所要するので、4日の見込みとなり、後払いでWLT 4を所要する

> 荷物持ちの処理について:
> - ステータスはエネミーリストの人間「一般人」を参照すること
> - アイテムを10個まで携行可能であり、携行重量5まで持ち歩くことができる
> - 荷物持ちは戦闘に参加しない
> - 戦闘に参加させた場合、荷物持ちは冒険者として扱われるため、荷運びの報酬に加え、冒険で得た報酬も等分することになる
>     - PLが戦闘参加を希望した場合、GMはこのことを警告として提示すること
>     - その上で戦闘参加を希望する場合は、許可しても良い
> - 荷物持ちが死亡した場合、冒険者はその遺体の回収と蘇生に責任を負う

## 迷宮への移動

- 迷宮への移動には、特に演出などは設けない
    - 迷宮探索部分が時間の大部分を占めるためである
    - GMの裁量で、荷馬車の道中や、村での会話、迷宮までの道のりを描写したり、冒険の要素を加えても良い

> 簡易的な処理:
> - 迷宮前での野営を実行する場合は、PCに目標値3で探索判定をさせても良い
> - 成功した場合、そのまま迷宮まで進める
> - 失敗した場合、迷宮に到着する時間が余計に2時間かかり、野営ロールに-2dのペナルティを受ける
> - MtV 4以上の場合、素早く迷宮前に到着できたため、野営ロールに+2dのボーナスを得る

### 迷宮前での野営

- 迷宮に到達した冒険者たちは、迷宮の入口近くで、その出入りを見張りながら野営を行う
- 時刻は20:00。あたりはすっかり暗くなってきている。今から野営の準備をすれば、6～8時間は眠れそうだ

> 野営のオミット:
> - GMが野営を不要と判断した場合は、この処理をスキップしても良い

オークの斥候:
- 24:00頃、野営をしていると、迷宮の入口から2体のオークが現れる
- 起床しているPCは、目標値3の感知判定を行い、成功するとオークの斥候を感知できる
    - 迷宮の入口を重点的に監視していた場合、目標値を-1する
    - オークは暗視を持たないため、たいまつを片手に行動しているため、暗視を持たないPCでも感知判定にペナルティは生じない
    - 見張りのPCが居ない場合、GMは最もLUCが高いPCで、目標値4で運判定をサイレントロールする。成功したら、そのPCがオークの斥候を感知できる
- オークの斥候は程なくして野営地に接近する
- オークの斥候を感知できなかった場合、オークの斥候は野営地を発見する
- オークの斥候は迷宮に引き返し、仲間を連れて02:00に野営地を襲撃する
- オークの斥候を感知した場合、PCたちは斥候を襲撃できる
    - オーク x 2 との遭遇となる。イニシアチブロールに+4dのボーナスを得る
    - PLが奇襲を希望する場合、目標値3の隠密判定を要求する。成功した場合、奇襲できる
    - 失敗した場合、イニシアチブロールに+4dでそのまま遭遇する

オークの襲撃:
- オークの斥候が迷宮に引き返した場合、オークたちは仲間を連れて野営地を襲撃する
- 02:00に襲撃が生じる
- オークの襲撃部隊はオーク x 3、オーク・シューター x 2で構成されている
- 見張りをしているPCは目標値2で感知判定を行い、成功するとオークの襲撃部隊を発見する
- 襲撃部隊を発見した場合、PCは仲間を起こし、戦闘の準備を行えるため、通常の遭遇となる
- 発見に失敗した場合、イニシアチブロールに-4dのペナルティを受けての遭遇となる
    - この場合、イニシアチブロールを行えるのは、見張りのPCに限定される
- 見張りのPCが居ない場合、PCは奇襲を受けての遭遇となる

> 斥候を撃破した場合について:
> - 襲撃部隊で追加されるオーク x 1とオーク・シューター x 2はセッション内に登場しなくなる
> - 希少なドロップ品などはないため、基本的には斥候を排除した方が有利である

戦闘後:
- そのまま野営を続けることができる
- 起床時、野営ロールを行い、各PCは野営の効果で回復やボーナスを得る

## 迷宮の探索

- PCたちは迷宮に突入する

時間のルールについて:
- 突入は8:00の時刻で行われる
- 部屋や通路を探索したり通過したりするたび、10分の時間を所要する
    - 部屋を探索せずに素通りする場合は、時間は所要しない
- 20:00を過ぎると、PCたちは「疲れや眠気を感じ始める」
    - GMは「一度地上に撤退し、野営を行うべきである」と推奨すること
    - 地上に移動するまでの移動にも時間を所要するため、20:00を実質的なタイムリミットとして扱うと良い
- 24:00を過ぎると、PCたちは強行ペナルティを負う
    - 1日に4時間以上の睡眠を取らない場合、すべての判定に-1d
    - 1日間、食事を摂らない場合、すべての判定に-1d
- 時間の管理は厳密に行わなくても良い
    - 移動や探索の回数が厳密に釣り合わなくても問題はない
- 迷宮探索に費やせる時間は2日間まで
    - 村に冒険者が帰還しない場合、異常自体発生としてギルドに連絡が行く
    - 時間をオーバーした場合、ギルドでの緊急対応が生じるため、その対応費用として報酬がWLT 2減額される

> 時間とマスタリングの考え方:
> - 8:00 - 20:00 を 1 term とし、12時間=72回の移動・探索と考えると良い
>     - 戦闘後、戦利品獲得を行うと10分を所要するため、戦利品を得る場合はさらに1回の探索として扱う
>     - 治療判定や修理判定には1時間を要するため、6回分の探索として扱うと良い
> - 1F～B2Fまでをすべて探索すると、およそ40～60回の移動・探索が生じるようになっている
> - 時間の厳密な管理をしたくない場合、野営については、B3Fの攻略の途中で一度「君たちは疲れや眠気を感じ始めた」としてPL側に休憩を提案させると良い

迷宮内での野営について:
- 迷宮内は快適な空間ではなく、常に警戒が必要なストレスフルな空間である
- 迷宮内で野営を行う場合、野営ロールに-4dのペナルティを受ける
    - 迷宮内の安全が確保された部屋や場所で野営する場合は、-2dのペナルティとなる

> 時間と休息について:
> - このダンジョンは規模が大きいため、リソース回復の機会として野営を推奨している
> - 処理が冗長だと感じる場合、この処理を無視しても良い
> - その場合、GMは「撤退して一夜を明かした」こととし、HPを1点回復し、MPを全回復させても良い
>     - このルールを採用する場合、PCからのルールの適用を提案されたら、無条件で応じる
>
> NPCの休息について:
> - この迷宮には救出可能なNPCが存在する
> - 探索1日目に出会えない場合、NPCたちは独自に休息を取っていたこととして良い
>     - つまり、NPCに対しては強行ペナルティを考慮しないものとしても良い

### R1 迷宮の入口 / S1 階段

- 迷宮の入口には、壁掛けのたいまつがかかっている
- 一見、火の消えることのないたいまつのように見えるが、壁から外すと途端に炎を失う仕掛けだ
    - この手の迷宮にはよくある遺物である
    - 便利だからと再現が試みられているが、実はまだ原理が不明とされる
- いずれにせよ、この迷宮はある程度の明るさがあるようだ。少なくとも、この入口と階段においては明かりは不要である
- 階段は、まっすぐ続いている
- 階段を降りると、R3に行き着く

> ロールプレイ用の判定:
> - GMは知識判定を求め、壁掛けたいまつについてのフレーバーを読み上げても良い
> - その際は、成功したPCが知っていたこととしてロールプレイさせると良い
>
> 迷宮の材質:
> - この迷宮は、石レンガ造りの壁や床でできている


## B1F

### R3 踊り場のような小部屋 / S2 階段

- 長いまっすぐの階段を降りると、踊り場のような小さな部屋に出る
- 正面には、まだ階段が続いている
- 左右には、仁王立ちする騎士の石像が立っている
    - 石像は、高さが3m程度ある
- 階段を降りて右手 (西側) の石像は、盾と剣をそれぞれの手に持っている
    - その股の間には、扉が備えられているようだ
- 左手 (東側) の石像は、両手剣を地面に突き刺し、その柄を上から抑えるようにして立っている

P1への扉:
- この扉は、普通に開けることができる

H1への隠し扉:
- この扉は、東側の石像の両手剣の裏に隠れている

石像の調査:
- 目標値3の知識判定、または目標値4の探索判定に成功すると、東の石像の剣の裏に、扉があることがわかる
    - この目標値は、PCが能動的に石像に調査を仕掛けた場合、-1して良い
    - ドワーフかダークエルフの場合、この判定には+2dのボーナスが得られる
- この石像の両手剣は脆くなっており、衝撃を与えることで崩すことができる
- 両手剣を崩すと、隠し扉の先に行くことができる → H1

S2 階段:
- まっすぐの階段が続いている
- 進むと R5 の部屋に出る

### P1 通路 / R2 小部屋

P1 短い通路:
- 特に何もなく、R2 に進む

R2 小部屋:
- 小部屋の中には、開け放たれた宝箱が見える
- 宝箱は所々が錆びた鉄のフレームの木箱である
- 中身を覗くと、何も入っていないことがわかる
- 目標値2で宝箱に何かしらの判定を行うと、蝶番はピッキングによって強引に開けられていることがわかる

部屋の探索:
- 目標値2の探索判定に成功すると、部屋は何者かに荒らされていることがわかる
- MtV 3以上の場合、埃についた足跡は、靴によってつけられたことがわかる

> 先に来ていた者:
> - この迷宮には、PCたち以外にも冒険者が立ち入っている
> - その冒険者PTは4人構成であり、すでに壊滅している
> - 1人はR21に立てこもり、1人は気絶した状態でその者と一緒にいる
> - 残りの2人はR42で死亡している

### H1 隠し通路 / R4 小部屋

H1 隠し通路:
- 土塊の両手剣を崩し、扉を進む
- 短い通路があるだけである。そのまま R4 に進む

R4 小部屋:
- R2 と対になるような部屋である
- 埃が堆積しており、人の気配はない
- 部屋には剣や盾、弓の入った武器掛けがあり、正面には空いていない宝箱が置いてある

- 使えそうな武器が置いてある

R21 小部屋:

- 兎獣人の少女「シチミ」と、ハーフリングの女性「ベニ」が隠れ潜んでいる

    - HP: -3、MP: 0

    - HP: 4、MP: 0

冒険者たちの救助:

- PCはベニのHPを1点以上まで回復することで、気絶したベニを救助できる
    - 救助しない場合、二人からの信頼や協力は得られない
    - 情報をシチミから聞き出そうとしても、ベニを救助するまでは応じてくれない
- この部屋は安全そうに見えるので、迷宮探索が終わるまでここで待機するよう指示しても良い
    - 迷宮からPCが離脱する際に二人を救助すると宣言した場合は、連れ出せたこととして良い

助力:
- ベニを救助すると、二人はPCたちに協力をしてくれる

マップ情報の共有:
- 二人から、先行したPTが探索した迷宮の情報が得られる
- R5 / R6 / R11 / R12 / R13 / R14 / R7 / R8 / R27 / R26 / R25 / R32 / R31 / R38 / R37 のマップを開示する
- R31とR32の間にH4に隠し通路がある。物陰に隠れるように人一人が通れる通路がある

先行したPTについての情報:
- ヒュームの戦士「ナミモリ」とヒュームの軽戦士「ツユダク」がいる

- 戦闘はR37の、階段を含めた十字路の部屋で行われた

同行:
- 救助された二人は、PCたちへの同行を希望する
- 同行させる場合、PCが二人を任意に操作して良い
    - 救助の対価として、金銭や所持品などを要求しても良いが、法外な要求をすると、あとでギルドに密告される可能性がある
    - 合計で価値:5以上の金品を接収した場合、後にギルドに報告をされ、冒険者資格を剥奪される
- 装備品や携行品は、冒険中に得た戦利品などを貸し与えても良い
    - 迷宮内で得られた防具などを貸し出せる、ということ
    - 戦力として期待するのであれば、必要なことだろう

> GM向け:
> - このダンジョンは危険度が高く、NPC二名を連れた状態での最深部での戦闘を想定しています
> - 体制を立て直すための野営ロールをPCが希望する場合、許可しても良いです

> - ベニとシチミは、ナミモリから「ギルド職員からいい話を聞いたんだ」と言われ、この迷宮にやってきている
>     - この情報は、ベニとシチミに対して事情を聞き出すことで得られる
>
> NPCのロールプレイについて:
> - NPCの操作はPLが担当しますが、ロールプレイはGMが主導してください
