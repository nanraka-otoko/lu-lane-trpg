# 訓練迷宮を踏破せよ！

```
ようこそ、ルゥレーンの世界へ！
君たちは、この世界に旅立つ冒険者の一人だ。

とはいえ、いきなり冒険者と言われても戸惑うかもしれない。
だが安心してほしい。

我々冒険者ギルドは、君たちのような駆け出し冒険者のために、訓練用の迷宮を用意している！
昔に攻略され、安全が確保された迷宮を、君たちのような者たちの訓練のために改修・開放しているのだ！

君たちはこの迷宮に挑み、最奥部に隠されたお宝を、我らがギルドに持ち帰るのだ！
```

---

## セッション情報

- 想定PC数: 3〜4人
- 想定脅威度: 1〜2
- 想定プレイ時間: 1〜2時間

### 1PCあたりの報酬の目安

| 項目              |  WLT |
| :---------------- | ---: |
| 依頼報酬          |    1 |
| 戦利品 (ドロップ) | 0～1 |
| 戦利品 (探索)     | 0～1 |
| 合計              | 1〜3 |

### セッション所要時間の目安

| フェイズ       | 時間           |
| :------------- | :------------- |
| 冒険準備       | 15分           |
| 訓練迷宮の探索 | 1時間 ～ 2時間 |
| 帰還・報告     | 15分           |
| 合計           | 約1〜2時間     |

### 備考

- 駆け出し冒険者のためのシナリオです
- 一回も冒険をしたことのないPCが対象となります
- 各種判定のやり方や、戦闘のルールをレクチャーしたり、GMの練習をする際に使用してください

## 冒険準備

```
冒険者登録を済ませたばかり、あるいはこれから登録しようとしているPCたちに声がかかる

「お前ら、ちゃんと訓練は受けたか？ いるんだよ、最低限の訓練も受けずに冒険に行って、返ってこないヤツらが」

どうやらギルド職員のようだ。
顔の傷跡や大柄な体躯、ゴツゴツとした筋肉から、彼が元冒険者の職員であると察することができるだろう。

「少額だが、報酬も出るぞ。駆け出しなら、やっておいて損はない。死んでからじゃ、訓練もできんからな」

君たちは、この男の助言を受け、訓練迷宮に挑むことにした。
```

訓練の内容:
1. 都市のとある区画に、かつて小さな迷宮だった跡地がある
2. ギルドはそこを調査・改修し、訓練用の迷宮としている
3. 迷宮内にはトラップや、訓練用の魔物を用意している
4. 迷宮を踏破し、最奥にある宝を持ち帰ったら訓練は完了となる

迷宮の利用にあたっての説明:
1. ギルドから鍵をもらい、訓練迷宮のある建物に入る
2. 簡易的な迷宮だが、関係者以外が入ると死の危険性もゼロではない
3. 必ず内側からかんぬきを嵌めておくこと
4. 脱出するときも、必ず鍵をかけてから出ること
5. 帰りが遅くなる場合、ギルド職員が救援に行く

報酬:
1. 訓練の推奨金として、一人あたりWLT 1を報酬として設定している
2. 冒険で得られた戦利品は、各自の自由にして良い
    - ただし、すでに探索しつくされている小規模迷宮なので、大した戦利品は得られないだろう

装備の貸し出し:
- 訓練にあたり、冒険者ギルドから下記の装備のいずれかを借りることができる
- 借りられるのは1種類のみであり、使用後は返却が必要
- 武器の状態が悪くなったり防具の装甲が0になっても、特に賠償などは不要

貸し出されている装備品
- ショートソード
- バトルアックス
- スピア
- ショートボウ + 矢筒 + 木の矢
- レザーシールド
- レザーアーマー

旅立つ君たちへの選別だ、と言われ、下記ももらえる:
- ヒールポーション
- このアイテムは、貸し出しではなくプレゼントだと言われる
    - 後で売っても良い

## 迷宮探索

### 迷宮の構造

- 5部屋からなるシンプルな構造

```
=== 1F ===

+----+
| R1 |
+-+--+
  |
  S1  ↓ B1F
  |
=== B1F ===
  |
+-+--+      +----+      +----+
| R2 |--P1--| R3 |--H1--| R4 |
+----+      +-+--+      +----+
              |
              S2  ↓ B2F
              |
=== B2F ===
              |
            +-+--+      +----+
            | R5 |--P2--| R6 |
            +----+      +----+
```

---

### R1 入口:

- 都市の外れに、統一言語で「冒険者ギルド管理 訓練用迷宮」と書かれた看板のかかった建物がある
- ギルド職員から預かった鍵を差し込むと、扉が開く
- 屋内に入ると、照明のない、埃っぽい部屋が冒険者たちを出迎える
- 扉を閉じて鍵を締める
    - 明かりがなくなり、暗視のない者は前が見えなくなる
    - PCたちは、ここでたいまつなどの光源を用意する必要がある
    - 暗視を持たず、光源もない場合、PCはすべての判定に-2dのペナルティを受ける
- 部屋の地面は土で、建物の外と同じ地面のようだ
    - 隅を見渡すと、所々に草が生えている
    - 日照が悪いのか、背の低い植物がひっそりと生えているだけのようだ
    - また、所々にキノコも生えているようだ
- 部屋の中央には、地下へと続く階段がある
    - 階段の材質は、部屋の床とは打って変わり、石レンガでできているようだ
- 冒険者たちはこの階段を降りることで地下に進むことができる

---

### S1 階段:

- 光のない、暗い階段
- 手元の明かりだけが冒険者たちの頼りだ
- 目標値2の探索判定に成功すると、階段の壁に、何かをかけるための金具が見つかる
- 金具に対して目標値3の知識判定に成功すると、これは壁掛けたいまつの名残であることがわかる
    - 古代の迷宮には、壁掛けのたいまつがかかっていることがよくある
    - この壁掛けたいまつは一見、火の消えることのないたいまつのように見えるが、壁から外すと途端に炎を失う仕掛けだ
    - 迷宮にはよくある遺物である。便利だからと再現が試みられているが、実はまだ原理が不明とされる
    - この迷宮の壁掛けたいまつは全部外されていることがわかる。それは、おそらく意図的なものである
- 階段は、まっすぐ続いている
- 階段を降りると、R2に行き着く

> ロールプレイ用の判定:
> - GMは知識判定を求め、壁掛けたいまつについてのフレーバーを読み上げても良い
> - その際は、成功したPCが知っていたこととしてロールプレイさせると良い
>
> 迷宮の材質:
> - この迷宮は、石レンガ造りの壁や床でできている

---

### R2 部屋:

- 何の変哲もない部屋
- 定期的に誰かが訪れているのか、床にはホコリなどはたまっていない様子だ
- 階段から降りて左手側に、二人程度まで並んで歩ける程度の、狭い通路が見える

---

### P1 陥没床の通路:

- せいぜい二人程度でしか歩けない、狭い通路
- この通路の途中には、陥没床がある
- 明かりを灯している場合、目標値1の探索判定、または目標値2の感知判定で罠を検知・回避できる
    - 探索判定はPC全員で行えるが、PLが能動的に通路での探索を宣言した場合にのみ認めること
    - PLが宣言せずに通路を進む場合、先頭を歩くPCに感知判定を行わせること
- 明かりがない場合や、暗視がない場合、探索判定の目標値は3、感知判定の目標値は4となる
- 感知判定に失敗したPCはそのまま穴を踏み抜いてしまう
    - 陥没床には槍が仕込まれており、踏み抜くと、防護・回避不可で3点のダメージを受ける

---

### R3 部屋:

- 部屋に入る直前、すべてのPCは目標値3で感知判定を行う
- 成功すると、ネズミの鳴き声を聞き取ることができる。通常の遭遇となる
- MtV 2で失敗した場合、警戒しているがネズミとの不意の遭遇となる。イニシアチブ判定に-2dのペナルティを受けての遭遇となる
- MtV 1以下で失敗した場合、ネズミからの奇襲を受ける。奇襲を受けての遭遇となる

遭遇:
- ラージラット x PC数との遭遇となる
- ラージラットはすべて敵前衛に配置され、PCの前衛に対して均等に攻撃を加える

探索:
- 戦闘後、部屋を探索できる
- 目標値3の探索判定に成功すると、H1の隠し通路を発見する
- ラージラットに対する知識判定を行う場合、目標値2を要求する
    - 成功すると、ラージラットが現れたルートを推測することができる
    - ラージラットの痕跡を追って探索判定をする場合、隠し通路の発見の目標値を-1する。すでに探索判定済の場合、再度判定できる

階段:
- 下階に続く階段がある

隠し通路:
- ラージラットが現れた小穴を調べると、それは50cm程度の高さの穴であることがわかる
- 匍匐前進をすることで、この奥に行けそうだと感じる
- 匍匐状態なので、明かりを持った状態では奥が見通せない
    - 暗視を持つPCは、この通路は5m程度で奥の部屋につながっていることがわかる

---

### H1 隠し通路:

- 匍匐で移動することができる狭い通路

> 大柄なPCについて:
> - ケンタウロスなどのPCでも、なんとか通ることができたこととして良いです

### R4 小部屋:

- 小さな部屋
- 部屋の壁に、小さな鉄格子が見える
    - ラージラットは、おそらく下水などからこの鉄格子を抜けてこの迷宮に入り込んだのだろう
- ネズミの棲家となっているのか、糞便などが転がっている
- この部屋から価値のあるものを探す場合は、探索判定を行い、その達成値で戦利品を獲得する

| 戦利品     | 目標値 | 価値 | 重量 | 説明                                                                   |
| :--------- | -----: | ---: | ---: | :--------------------------------------------------------------------- |
| ガラクタ   |      2 | 0～1 |    0 | 獲得時、2M6を振る。価値はMtV-1となる                                   |
| 小さな宝石 |      4 |    1 |    0 | 何かの拍子にこの部屋に流れ着いたのだろうか？ありがたくいただいておこう |

### S2 ロープの張られた階段:

- ロープの罠が仕掛けられた階段
- 石レンガの色と同化するように着色されたロープが、階段の途中に張られている
- 目標値3の探索判定または感知判定で罠を検知・回避できる
    - この罠の検出を探索判定で能動的に行う場合、先頭を歩く者だけは目標値を-1して良い
    - これは、階段の足元という死角に仕掛けられた罠であるため、後続からは視認が難しいからである
- 先頭を歩く者が感知判定に失敗すると、ロープに足を取られて階段の底に転落する
    - 転落したPCは、自身の携行重量の点数だけダメージを受ける。このダメージは回避・防護不可

---

### R5 準備室  / P2 扉

- 扉のある部屋
- 目標値3の探索判定に成功すると、この部屋の床や壁に戦闘の痕跡が見つかる
    - かつてここが本物の迷宮だった際に、ここで戦闘が行われていたことが推測できる
- この部屋には、扉以外は特に何もない

扉:
- 扉には統一言語で「訓練の間」と書かれている
- 扉は鉄枠の木製の両扉で、1枚あたり1.5m程度の幅、合計で3mの幅がある。押せば簡単に開きそうだ
- この扉にはブービートラップが仕掛けられており、そのまま開けるとトラップが作動する
- PCは扉に対し、目標値3で知識判定、探索判定、感知判定のいずれかを行うことができる
    - 成功すると、扉をあけると同時に悪臭ガスが噴出されることがわかる
    - 悪臭ガスを吸った者は、一時的にすべての判定に-1dのペナルティを受ける、というトラップである
    - 両扉だが、右半分の扉の先にワイヤーがあり、そちらを押すとトラップが作動するという仕掛けのようだ
- 左半分の扉のみを開けて中に入ることで、トラップを無視することができる
- トラップの解除を希望する場合、目標値3の解除判定に成功する必要がある
- トラップが作動すると、R6にいるトレーニングゴーレムは侵入者を即座に検知し、イニシアチブ判定に-2dのペナルティを受けての遭遇となる
    - このペナルティは、悪臭ガスのペナルティと加算されるため、-3dのペナルティを受けてのイニシアチブ判定となる
- トラップを作動させずにR6に侵入すると、通常の遭遇となる

---

### R6 訓練の間:

- トレーニングゴーレムが守護する部屋
- 部屋に侵入するか、扉のトラップを作動させると起動し、遭遇となる

遭遇:
- トレーニングゴーレム・剣 x 3
- トレーニングゴーレム・弓 x 2
- トレーニングゴーレム・杖 x 1
- トラップを作動させている場合、悪臭ガスのペナルティが戦闘開始から戦闘終了まで継続する
    - トレーニングゴーレムたちは悪臭ガスの効果を受けない
    - 悪臭ガスのペナルティは、戦利品判定にも適用される

戦闘後:
- 戦闘が終了すると、悪臭ガスがなくなる
- トレーニングゴーレムが待機していた台座を調べることができる
    - 調べると、ガラス玉を入手できる

| 戦利品   | 目標値 | 価値 | 重量 | 説明                                   |
| :------- | -----: | ---: | ---: | :------------------------------------- |
| ガラス玉 |      - |    0 |    0 | 安っぽいガラス玉。迷宮踏破の証のようだ |

脱出:
- PCたちは、任意にこの迷宮から脱出して良い

## 結末

共通の処理:
- 借りていた装備品は、忘れずに返却すること
- キャラクターシートに書きっぱなしで借りパク状態とならないよう、GMは注意すること

### 成功

- 君たちはガラス玉を冒険者ギルドに持ち帰った
- WLT 1を訓練の報酬金として獲得する
- 冒険者たちは1点のEXPを獲得する

### 敗北

- 冒険者ギルドの監督者が駆けつけ、君たちは救助される
- LUCが0のPCは、不運にも命を落としてしまったようだ
    - 神殿にて蘇生を受けた君は、WLTを10失い、CONとMNDを恒久的に-1点して復活する
- 冒険者たちは1点のEXPを獲得する
