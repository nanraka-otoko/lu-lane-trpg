# リコレクション: 冒険者になった日

```
君たちは冒険者となるべく、馬車に乗り、共和国の首都へ向かう。
もちろん、それ以外の理由で乗り合わせている者もいるかもしれない。

馬車は合計3台。君たちの他にも、一般の旅行客なども乗り合わせている。
その馬車は、2日をかけて首都に到着するはずであった。
乗車から1日が経ち、キャンプで夜を明かして2日目、その朝に事件は起こる。

君たちが乗る馬車が魔族に襲われたのだ。
奴らに立ち向かえるのは、君たちしかいない。

それは望んだことか、望まざることなのか――
――いずれにせよ、この場所から君たちの冒険が始まるのだ。
```

## 魔族の襲撃

魔族の襲撃の描写:
```
君たちがそんなことを思っていると、馬車はやがて小さな森の入り口に差し掛かる。
そのとき、馬車が急に激しく揺れた。
どうやら馬引きが急制動をかけたようだ。

そしてその衝撃からひと呼吸おいて、大声が上げられる。

「魔族だ！魔族が出たぞ！」
「結構な数だ！冒険者は早く出てきてくれ！」

どうやら、魔族がこの馬車を襲撃してきたらしい。
馬車の護衛と思わしき冒険者たちが3名、馬車の前に躍り出る。

「オークだ！なんとしても俺たちで食い止めるぞ！」
「ああ！馬車はそこで待機しててくれ！」

焦りを感じる声でそう言うと、その冒険者たちは馬車前方の魔族へと向かっていった。
その10秒ほどたった後、今度は馬車の乗客から悲鳴が上がる。
```

PCたちが正面のオークとの戦闘に加勢しようとした場合は次のように進行する:
- 「君たちが冒険者の加勢に向かおうとした次の瞬間、今度は乗客から悲鳴が上がる」

ゴブリンの襲撃の描写:
```
「キャーッ！！ゴブリンよ！ゴブリンが後ろからァ！！」

緑色の小汚い肌の小柄な魔族たちが、馬車の後方から小隊を組んで向かってくる。

距離は馬車からおよそ50m程度。
あと20秒もすれば、この馬車は奴らに蹂躙されてしまうことだろう。
馬車正面に向かった冒険者たちは、戻る気配がない。

一瞬の逡巡の末、君たちは馬車から後方へと躍り出る。
それに気づいたゴブリンたちは、不意打ちを悟られたと思ったのか、一瞬立ち止まった。
そして馬車に立ちはだかる君たちに対し、敵意を剥き出しにして突っ込んでくるのであった。
```

ロールプレイについて:
- 非戦闘員だけど戦いたいというRPをしたいPLには、馬車備え付けの武具を借りる体で出撃させると良い
- 防具の装備時間は考慮せず、装備した状態で戦闘を開始して良い

遭遇:
- ゴブリン x 7
- 4体が前衛、3体が後衛に配置される
- 後衛のゴブリンは、後衛ターンで前衛に移動し、攻撃を行う

戦闘後:
- PCたちは戦利品判定を行うことができるが、この時点では戦利品を獲得できない
    - 危機的状況にあるため、他のことを優先する必要があるからである
    - PCは戦利品判定を行い、そのロール結果をもって馬車に乗り合わせた他の客が戦利品を回収し、後でPCたちに寄贈される

## 冒険者からの依頼

```
君たちはゴブリンの襲撃部隊との戦闘を終え、馬車に駆け戻る。
すると、馬車前方に向かっていた冒険者たちもまた、戦闘を終えたようであった。

君たちは馬車前方で戦っていた冒険者たちに駆け寄る。

冒険者は2人。1人は気絶しており、もう1人は深く傷ついている。
そして戦場だった場所には、オークの死体が数体転がっている。

それを見た君たちは、起きている冒険者に話しかける。
(話しかけない場合は、向こうから話しかけてくる)

「助かった、礼を言わせてもらうよ。ありがとう。
  我々だけではこの馬車を守り抜くことはできなかった」

  おかげでこの有様さ」

冒険者はそう言いながら、気絶した仲間を手当をしている。

「偶然乗り合わせていたのが君たちで良かったよ。
  一体どこのギルドに所属しているんだい？」

冒険者にそう聞かれ、君たちはまだ冒険者にすらなっていないことを告げる。

「驚いた。これから冒険者になる者に助けられてしまうとは
  世の中、不思議な縁もあるものなのだな」

そして冒険者の男は深く息を吸い込み、真剣な声音で君たちに話しかける。

「……君たちに頼みたいことがある。
  僕の仲間が一人、奴らに連れ去られた」

「彼女を連れ去ったのは、オークの戦士だった。両手持ちの大剣を担いだ個体だ。
  街道を離れ、森の中へと去っていった」

そう言って、冒険者の男は森の奥を睨む。
どうやらその仲間は、この森の中に連れ去られてしまったらしい。

「……虫の良い話だってことは分かっている。
  それを承知でどうか……彼女を助け出してはもらえないだろうか？」

「見ての通り、私は今動くことができない……
  奴らにやられてしまい、このザマだ」

男は腱が切れた右腕と左足を見せる。
確かに、この状態では満足に動くことは無理だろう。

「こんな状態だが、少し治療すれば馬車の護衛くらいはできる。
  彼も直に目を覚ますはず。だが、彼女の安全を考えると猶予がないんだ」

「報酬も用意する。無理を承知で頼む、どうか、力を貸してほしい」
```

オークに関する知識:
- PCは目標値2の知識判定を行う
- 成功すると、オークたちは集落を形成したり、迷宮や遺跡を根城にしているという知識を思い出す

    - 急ぎ追撃に向かうべきであることがわかる

依頼の詳細を聞く:
- PCは目標値2の感知判定を行う
- 成功すると、この男の依頼には嘘はないが、まだ言っていないことがあるように感じる
- 詳細な話を聞くと、男は真意を話してくれる

## 森の中の追跡

- 10:00 から探索をスタートする
- 15:00 がタイムリミット
    - そこまでに対象を救助できなかった場合、馬車の乗客たちの安全のためにも引き返す必要がある

森に踏み込む際の描写:
```
君たちは冒険者からの依頼を受け、意を決して森に立ち入る。

森の入口に立つと、オークが踏み荒らしたような草木の痕跡と共に、赤い血痕が見つかる。
これを追跡していけば、いずれオークにたどり着くはずだ。
```

追跡のルール:
- 追跡対象までの接近の度合いを「追跡値」として管理する
- 追跡度が30以上になると、PCたちは追跡対象に追いつくことができる
- 10分を所要し、探索判定を行って追跡を行う
    - この判定はPC全員で行い、最も高い達成値を参照する
    - 探索判定の達成値を、追跡値に加算していく
- 300分の猶予があり、合計で30回までの追跡を行うことができる
    - 追跡にのみ10分を所要する
    - 追跡の最中に、イベントで木などの痕跡に探索判定をする場合は時間を所要しない
- 300分経過時点で追いつくことができなかった場合、タイムオーバーで依頼は失敗となる

> 追跡の補助:
> - 自然環境であり、草木があるため、植物や自然から助言を得るスペルが使用できる
>     - これらの魔法を行使した場合、PT全員の探索判定に+1dのボーナスが得られる
> - 痕跡に対する判定と、匂いに対する判定にボーナスが得られる特性がある場合は、探索判定にボーナスを適用して良い
> - その他、PLがボーナスを希望した場合、GMの裁量で認めて良い
>
> 女神官の無事:
> - 女神官は出血して気絶している
> - 13:20 になると、女神官はオークに殺害される
>     - PCたちが20回追跡を行った時点で殺害が生じるというこよ
>     - 殺害の条件は追跡値25のシンボルイベント「女神官の遺体」を参照すること

シンボルイベント:

### 追跡値 10: 不自然な案内板

- 君たちが足跡を追って森を進んでいくと、不自然に文字の刻まれた木が現れる
- 木には、次の文字が共通言語の文字で刻まれている

```
「にっチ ぁジと ->>」
```

- 木の下には足跡が2方向に向かって続いている
- どちらも追跡中のオークと同じ足跡だが、何度か行ったり来たりしているようであり、どちらに向かって進んだのかはわからない

足跡の追跡について:
- 足跡は念入りに往復しており、血痕もどちらにも往復している
- 探索判定では手がかりを得ることはできない

知識判定:
- この文字に対して目標値3で知識判定を行うことができる
- 成功すると、この文字は魔族が見様見真似で描いた文字であることがわかる
    - 加えて、魔族は共通言語を基本的に用いないので、これは魔族間でのコミュニケーションのために書かれた文字ではないと推測できる
- MtV 4以上で成功した場合、以下のことを確信できる
    - 文字は明らかにオークが書いたものである
    - 足跡が往復していることから、オークは意図的に追跡者を騙そうとしている
    - 2択を迫るだけであれば足跡を偽装すれば良いだけのことだから、この文字は嘘の手がかりである
- MtV 2で失敗した場合、「こっち アジト →」と書いているのではないかと推測する
    - 右側にアジトがあることを示している看板だと察する
- MtV 1以下で失敗した場合、何も情報は得られない

分かれ道:
- 右方向の、文字がに示された方向に進むと、足跡が途中で途切れる
    - こちらに進んでしまった場合、オークとの距離が放される
    - 追跡値が-3される
- 左方向の、文字とは逆方向に進むと、足跡が続いている
    - そのまま追跡ができる

> オークの知性:
> - オークは人族を騙したり、知恵を働かせることがある
>     - 先程の馬車の襲撃でも、オークとゴブリンの部隊を2手に分けて襲撃する程度には知恵が働く
> - 共通言語を使って秩序の者を騙そうとする策略も思いつく
> - だが、秩序の者どもに対して、オークたちは己と駆け引きが成立すると想っていない
>     - そのため、裏の裏をかく、といったことは考えていない
>     - 木に彫り込んだ案内も、これを見た人族の連中は騙されて右に追跡をするはずだ、と確信している
