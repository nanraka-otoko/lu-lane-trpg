# 橋を占領する魔族の討伐

## シナリオの概要

```
街と村を結ぶ石橋は、人々の暮らしを支える交易路の要所だ。

だが今、その橋はゴブリンどもに占拠されている。
奴らは橋の向こう岸にバリケードを築き、通行人を脅かし、荷物を奪い、好き放題に振る舞っている。

ギルドに駆け込んだ商人の証言によれば、ゴブリンは数体。
橋を封鎖しているだけで、周囲の集落を襲うほどの勢力ではないらしい。

とはいえ、橋を通れなければ村との物流は止まる。
手早く片付けなければならない。

橋の上の戦いだ。退路は少なく、足場は狭い。
正面からぶつかるか、別の手を考えるか。それは君たち次第だ。
```

---

## セッション情報

- 想定PC数: 3〜4人
- 想定脅威度: 1〜2
- 想定プレイ時間: 1.5〜2時間

### 1PCあたりの報酬の目安

| 項目              |  WLT |
| :---------------- | ---: |
| 依頼報酬          |    2 |
| 戦利品 (ドロップ) | 1〜2 |
| 合計              | 3〜4 |

### セッション所要時間の目安

| フェイズ       | 時間         |
| :------------- | :----------- |
| 冒険準備       | 15分         |
| 目的地への移動 | 15分         |
| 偵察           | 15分         |
| 橋の戦闘       | 30分 ～ 45分 |
| 帰還・報告     | 10分         |
| 合計           | 約1.5〜2時間 |

### 備考

- かなり簡易的なシナリオです
- 移動を必要とするセッションの道中に、このシナリオを追加してボリュームアップするといった使い方をしても構いません
- 例えば、商人の護衛のシナリオ内に追加でこのシナリオを搭載しても良いでしょう
    - その場合、報酬は橋を渡れずに困っていた別の商人から受けとる、などといった扱いにすると良いでしょう

## 冒険準備

### 情報収集

君たちはギルド職員から依頼の説明を受ける。

1. 街の南にある石橋をゴブリンが占拠している
2. 通行人や商人が渡れなくなっており、村との交易に支障が出ている
3. ゴブリンは8体ほどが目撃されている。うち1体は他より装備が良く、号令をかけているらしい
4. 橋の向こう岸にバリケードを築いて陣取っているらしい
5. 討伐の報酬は一人あたりWLT 2

依頼人について:
- 依頼人は、被害に遭った商人のひとりだ
- 荷物を奪われたが命は取られなかった
- だが商人は怯えており、「次の犠牲者は出ないとは限らない」と訴える

> GM向け: PLがさらなる情報を求めた場合、目標値2の対人判定を要求する。
>
> 成功すると:
> 1. 橋の下には川が流れている。深さは腰くらいだが、流れはかなり速い。装備のまま落ちれば危険だ
> 2. ゴブリンの中に弓を持っている奴が複数いた。バリケードの後ろから矢を浴びせてきた
> 3. 号令をかけている個体は、ゴブリンにしては良い装備を身につけている。手強い相手だ

### 物品調達

- 特別な準備は必要ないが、ロープがあると川を渡る際に役に立つかもしれない

## 目的地への移動

- 街を出て南の街道を歩く。半日ほどの道のりだ
- 天候は穏やかで、街道の治安も悪くない

道中、すれ違う旅人や商人から橋の状況を聞くことができる:
- 「橋の手前で引き返してきた。近づくと矢を撃ってくる。それも複数本だ」
- 「小さい奴らだったが、数がいた。5匹は見えた」
- 「先月、近づきすぎた行商人が矢で射られて死んだそうだ」
- 「他のゴブリンよりも装備が良い奴がいた。あれが頭目だろう」

### 橋の手前

- 街道の先に石橋が見えてくる
- 川幅は20mほど。橋はその中央をまっすぐに渡している
- 頑丈な石造りで、欄干もある
- 橋の向こう側に、木の土台が組まれているのが遠目に見える
    - 木材や荷車の残骸を積み上げた粗雑なものだ
    - 残骸の土台の上には、ゴブリンのような影が見える

選択:
- GMはPCに行動を選択させる
- 正面から突入する、偵察する、川を迂回するなどの選択肢がある

---

## 偵察

- PCが橋の手前から敵を観察する場合、代表者が目標値2の隠密判定を行う
- 成功すると、下記の情報が得られる
    1. ゴブリンは合計10体
        - ゴブリン x 5 が土台を見上げてはしゃいでいる
        - ゴブリンアーチャー x 3 が土台の上に登って座っている
        - ゴブリンシャーマン x 1 が腹をかいて寝ている
        - 他より装備の良い個体 (ゴブリンリーダー) が土台の上で何かを喋っている。意味はわからない
    2. 土台の手前には、粗雑な作りのバリケードがある。ゴブリンたちがそこではしゃいでいる
    3. ゴブリンたちは橋の方角を常に警戒している。橋の上に踏み込んだ瞬間、戦闘が始まる
    4. 橋の近くには、ゴブリンが殺したと思わしき商人の亡骸がある。ひどく食い荒らされている
- 失敗した場合、ゴブリンたちは冒険者の襲撃を感知する
    - 気が付かれた状態でPCが戦闘を仕掛ける場合、イニシアチブロールに-2dのペナルティを受ける

---

## 川を渡る

- PCは川を泳いで対岸に渡り、敵の背後に回り込むことができる
- 橋から少し離れた場所で川に入り、川を渡る

渡河:
- 橋を渡るPCは、目標値3の汎用行動判定を行う
- ロープを持っている場合、判定に+2dのボーナスを得る
- 水棲の特性などを所持している場合は、判定不要で川を渡ることができる

成功:
- 対岸に渡れる。土台の背後に回り込める
- 戦闘開始時、川を渡ったPCは「敵後衛」のエリアに配置される

失敗:
- 流れに足を取られ、下流に流される
- 4M6点のダメージを受ける
- 流れ着いた岸から橋までは徒歩で15分。再挑戦する場合は再度判定を行う
- 再挑戦する場合は、川に慣れたので目標値が2になる

連続して失敗した場合 (2回目以降の失敗):
- 川底の岩に打ちつけられ、8M6点のダメージを受ける
- HPが0以下になった場合、目標値2で運判定を行う
    - 失敗するとそのまま溺死する
    - 成功した場合、下流で意識を失っているところを発見される。HPは1に戻る

---

## 橋の戦闘

- 橋に踏み込むと、ゴブリンたちとの遭遇となる

### 地形

- 敵後衛: 残骸の土台
    - このエリアからの遠隔攻撃の攻撃判定に+2d
    - 他のエリアからこのエリアへの近接攻撃の攻撃判定に-2d
    - このエリアに移動する場合、予備行動ではなく行動を消費する
- 敵前衛: 小さなバリケード
    - このエリアに移動する場合、予備行動ではなく行動を消費する
    - このエリアへの近接攻撃と、このエリアからの近接攻撃の攻撃判定に-1d

### バックアタック

- 橋の後ろから攻め入ったPCがいる場合、バックアタックが可能
- 奇襲となり、最初のラウンドは敵全員に-4dのペナルティが与えられる
- バックアタックしたPCは敵後衛のエリアに配置される
- バックアタックしたPCは、敵陣営のエリアでも包囲されない

### 敵の編成

遭遇:
- ゴブリンリーダー x 1、ゴブリンシャーマン x 1、ゴブリンアーチャー x 3、ゴブリン x 5 との遭遇となる
- ゴブリンリーダー、ゴブリンシャーマン、ゴブリンアーチャーは後衛に、ゴブリンは前衛に配置する
- 奇襲が成功した場合、ゴブリンシャーマンは前衛に配置される

> 難易度の調整:
> - PCの人数が多い場合や装備が良い場合は、レッドキャップを1～2体追加する
> - すでに偵察している場合は「残骸の裏で寝ていた個体が起き上がった」とすると良い

敵の行動:
- ゴブリンリーダーは、手下の数が2体以下になると、逃亡を企てる
    - PCと同様、撤退のルールに従い、1ラウンドの間すべての行動に -2d のペナルティを受ける
    - ラウンド終了時に撤退する
    - 撤退したゴブリンリーダーの戦利品は得られないが、依頼の達成状況に影響はない
- 前衛にゴブリンシャーマンがいる場合、自身のターンで敵後衛に移動を試みる
    - この移動は行動扱いとなるので、最初のターンに他の行動ができない

### バリケードの探索

探索:
- 戦闘後、バリケードを調べることができる
- 探索判定を行い、そのMtVを参照して戦利品を獲得する

| 戦利品   | 目標値 | 価値 | 重量 | 説明                                                                                                |
| :------- | -----: | ---: | ---: | :-------------------------------------------------------------------------------------------------- |
| ガラクタ |      2 |    0 |    0 | 商人から奪った雑貨を適当に解体して作ったバリケードのようだ。次のPCの探索判定に+1dのボーナスを与える |
| 金品     |      3 |    1 |    0 | 商人から奪った金品。依頼人に渡すことで、価値:1として謝礼が得られる                                  |

---

## 帰還・報告

- ゴブリンを討伐し、橋の通行が再開される
- バリケードを撤去すれば、商人や旅人が再び行き来できるようになるだろう
- ギルドに戻り、依頼の完了を報告する
- ギルド職員は討伐の報告を受け、報酬を支払う

## 結末

### 成功

- 依頼達成の報酬: 1PCあたり WLT 2
- 冒険で得た戦利品を換金できる
- 冒険者たちは1点のEXPを獲得する

### 撤退

- PCたちは橋の手前まで撤退できる。ゴブリンは橋を追って来ない
- 撤退して帰還した場合、依頼は失敗となり、報酬は支払われない
- 戦闘を1回でも行っていた場合、冒険者たちは1点のEXPを獲得する

### 敗北

- PCが全滅した場合、すべてのPCは運判定を行う
    - 死亡しているPCは達成値に-1の補正を受け、死の結末のみを受け入れる
- 達成値0
    - 君は死亡した
    - 装備を剥ぎ取られ、死体は川に投げ捨てられる
    - 死体は下流に流され、回収されることはなく、蘇生されることもなかった
- 達成値1
    - 君は死亡した
    - しばらくした後、川岸に流れ着いた死体が回収される
    - 望むのなら、蘇生の奇跡を受けることができる
    - 蘇生を受ける場合、WLT を 10 失い、CON と MND を恒久的に -1 して復活する
    - 装備中の盾と防具のみを残し、それ以外の所持品をすべて失う
- 達成値2以上
    - 君は一命を取り留め、なんとか街に撤退した
- 報酬は支払われない
- 冒険者たちは 1 点の EXP を獲得する
