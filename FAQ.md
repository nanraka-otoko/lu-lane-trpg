# よくある質問

## キャラクター / キャラクターメイキング関連

### クラスやレベルがないと、どういう方向でキャラクターを作れば良いかわかりません

主なアプローチは2つです:
1. キャラクターメイキング前に「こういうキャラクターでプレイしたい！」というイメージを固めておき、それに沿って種族や背景を決定する
2. ステータスが決まった時点で、どのような方向性でプレイするのかを決定する

採用するアプローチ:
- 両方のアプローチを同時に採用すると良いでしょう
- 「こういうことが好きだから/したいから、この種族を選び、このステータスを伸ばそう」という風にメイキングします
- 実際に狙い通りのステータスにならないことや、思いもよらない組み合わせのキャラクターとなることもあります
- 最終的には、完成したキャラクターでできることを考え、アイテムや装備を揃えると良いでしょう

メイキングの方向性の例:
- 前衛に立って戦う戦士や騎士、タンクやアタッカーをしたい場合:
    - CONとSTRを高めるようにメイキングすると良いでしょう
    - DEXもある程度あると安心です
- 前衛と後衛を行き来しながら戦う戦闘をしたい場合;
    - DEXを優先し、攻撃の選択肢を増やすと良いでしょう
    - 前衛に出る場合は、CONとSTRがある程度あると安心です
- 弓などの遠隔攻撃で戦いたい場合:
    - ダメージと回避に影響するDEXを中心に高めるとよいでしょう
    - 防具の重量を節約できるため、STRが凡庸でも戦うことが可能です
- 魔法使いや神官をしたい場合:
    - MPを高めるMNDと、魔法・奇跡の行使の条件となるINT、FTHを高めると良いでしょう
    - 才能には魔法・奇跡に影響するものも多いです
    - 杖や防具を持つために、最低限のSTRがあると安心です
- 魔法戦士、神官戦士がしたい場合:
    - 様々なステータスが要求されるため、ステータスの調整が難しいロールです
    - 使用したいスペルの要求INT、FTHを確保したら、残りはCON、STR、DEXを確保すると良いでしょう
- 魔法と奇跡を両方とも使う賢者ロールがしたい場合:
    - MND、INT、FTHのすべてが必要です
    - ステータスの確保が難しく、データの参照量が多いため、初心者には推奨しません
    - メイキングの時点でPLに賢さと戦略が求められるため、リアル賢者向けです

### どの種族にどういう特徴があるのかがわからない。ざっくり説明してほしい

- あくまでもステータスと特性による、適性の傾向です
- 実際にメイキングすると、全く違った結果が得られることもあります

| 種族             | 特徴                                                                                                         |
| :--------------- | :----------------------------------------------------------------------------------------------------------- |
| ヒューム         | 種族専用の背景を持たないため、ステータス決定のランダム性が高い。専用魔法も扱いが難しい。最もピーキーな種族   |
| エルフ           | INTとDEXを高めやすい。魔法剣士や後衛の魔法使い向け                                                           |
| ドワーフ         | CONが高く、酒でMPを回復できるので前衛の魔法戦士・神官戦士向け                                                |
| ビースト         | CON以外は平凡ですが、2個所のステータスが+1できるため、ステータスの自由度がかなり高い種族                     |
| ドラゴニュート   | 戦闘に欲しいステータスが軒並み高い種族。何でもできるキャラクターを作りたい時向け                             |
| ダークエルフ     | STRがあるので前に出やすいエルフ。禁止魔法を使用しやすいのが特徴                                              |
| リザードマン     | CONとSTRが高く、物理耐性も持っているため、前衛の適性が非常に高い                                             |
| ハーフリング     | テクニカルなヒューム。低いSTRの代わりに回避が強く、高いDEXを活かしやすい                                     |
| バードフォーク   | 鳥の爪による2回攻撃ができる近接DPS。DEXを取るかSTRを取るかが重要                                             |
| フェアリー       | STRがないピーキーな種族。高MND+高INTでスペルキャスターの適性が高い                                           |
| ケンタウロス     | STRがとにかく高いので、強い武器を装備しやすい種族。前衛に向くが、回避が貧弱                                  |
| マーフォーク     | 水中向けの種族。DEX、LUC、APPが高いため、冒険中の判定に強いオールラウンダー                                  |
| ウッドボーン     | MND、INT、FTHが高く、スペルキャスターの適性が高い。代償としてDEXが低く、様々な判定で足かせとなる             |
| エルトメンヒェン | APPが高く、生存力も高い種族。戦闘に向いた特性やステータスではないが、専用魔法をAPPで行使できる               |
| マイセリアン     | 最高のCONと状態異常耐性を持つが、DEXが低く、炎に弱い。耐久はどんな場面でも生きるので、オールラウンダーな種族 |
| アラネア         | 高いDEXとLUCを持ち、状態異常耐性もある万能な種族。FTHが低いため、低難易度の奇跡の行使すら危うい点に注意      |
| サーペンテイル   | 高MND、高FTHの種族。状態異常耐性を持つ点と合わせ、行動妨害に強いのが特徴。露骨に神官向け                     |

### 種族ごとのステータスのダイス数って合計でどれくらいなの？

- ヒューム、ビーストが36個です
    - ビーストは任意のステータスが+1されるため、実質的に42個程度のダイスを持ちます
    - 任意の箇所なので、ステータス面ではかなり有利です
- 残りの種族は概ね40～42個です
- フェアリーは40個ですが、STRが固定値ごと存在しないので、ステータスの合計値は最も低くなります
- ドラゴニュートは46個のダイスを持つため、最もステータスが高くなりやすい種族です

### お金が0や0以下になったんだけどどうすれば良い？

- それがあなたのゲーム開始時点での所持金です
- 価値:0のアイテムや装備を上手に使って最初の冒険を乗り切ってください

### 「混血」の才能を取ったとき、種族専用の背景は2種類の種族でそれぞれ獲得できるの？

- できます
- エルフの生まれでドワーフの混血となったとき、エルフの背景とドワーフの背景をそれぞれ1つずつ選べます
- どちらか一方の背景を2種類獲得することはできません

### ステータスのリロールってしていいの？

- GMと相談の上で、リロールをしても構いません
- リロールする際は、ステータスすべてを決定した後に、その結果を見てからリロールしてください。要は、部分リロールはNGです
- リロールの回数は、GMが認めた回数までとしてください。目安は3回です
- 上書きリロールでも、複数のリロール結果の代表値を選ぶ方式でも大丈夫です。好きな方を選んでください

### 壮年「任意のステータスに+1」をするタイミングに指定はあるの？

- キャラクターメイキング中の任意のタイミングで決定して良いです
- ステータスロール後に決定しても良いですし、背景の決定が終わった後でも構いません

### ビーストの特性「獣の身体」の効果は、いつ決めるの？

- キャラクターメイキング中の任意のタイミングで決定して良いです
- ステータスロール後に決定しても良いですし、老年であれば老いの代償を得た後でも構いません

### 成長要素が渋すぎない？

- 渋いですが、意図的に渋くしています
- 各ステータスの1点が相対的に大きいため、1度の成長でも大きな影響が生じます
- より成長要素を強めたい場合は、GMと相談の上で、セッションで得られる経験点を調整して構いません

### 背景とステータスの関係は？ 成長ではLUCとAPPが伸びないのに背景で伸びるのはなぜ？

- 背景で増減するステータスには複数の解釈でロールプレイに活用できます
    - そうした生業をこなすために、後から必要な能力を身に着けた
    - 生業の中で自然と能力が向上した
    - もともとそうした素養があったので、活かせる生業についた
- 背景とステータスとの関係性は、GMとPLの間で自由に解釈して構いません
- 成長で伸びないLUCとAPPが背景で伸びるのは、素養に紐づくステータスであるためです
    - LUCとAPPは「そうした生業についたから伸びた」ではなく、「そういう素養があったので適した生業で成功した」という側面が大きいです

### 独自の背景とかを設定してもいいの？

- GMとPLの合意が形成されていれば、独自の背景を設計・実装しても構いません
- キャンペーン用に独自の背景を用意することも良い選択でしょう
- ステータスやWLTの強弱については、セッションの方向性を考慮して決定すると良いでしょう

### ビーストのケモ度ってどれくらいなの？

- 特に世界観上での指定はありません
- ほとんどヒュームの見た目で、耳と尻尾だけあるという程度でもOKです
    - 明らかに爪が鋭くなさそうでも、データ上は爪で攻撃できます
- モッフモフの獣人にしても良いですし、脚を逆関節にしても構いません
- 顔つきも原典となる動物に寄せても良いですし、程よくヒトにしても良いです
- 要は、PLの自由ということです

### ドラゴニュートの顔つきは？ドラゴンの顔？

- 四肢があり、背中に龍の翼を持つという身体的特徴以外は、すべて自由です
- 前進が鱗に覆われた龍っぽいドラゴニュートにしても良いですし、ほとんどヒトで翼だけ龍でも大丈夫です
- 同様に、リザードマンもトカゲに寄せても良いし、トカゲの手や脚や尾だけあるけどほとんどヒト、という風にしても構いません

### ウッドボーンやエルトメンヒェン、マイセリアンは？

- 亜人種の見た目は、本当にPLの好みで決定して大丈夫です
- 樹人は蠢く樹木が人型をしているというような奇っ怪な見た目でも良いですし、二足歩行の美女でところどころが樹木っぽい感じでも良いです
- 花人は完全に花が二足歩行していて顔があるでも良いし、頭から花が生えているだけというのでも構いません。PLの好みで自由に選べます
- マイセリアンも同様です。キノコの怪物のような見た目にしても良いし、髪の毛が茸の傘になったような造形にしても良いです
- 重要なのは、APPが1点以上ある限りにおいて、これらの種族は秩序の文明に所属する権利があるということです
    - そしてどれだけ造形が植物っぽくなくても、地面に埋まれば栄養を吸収できたり、光合成できたりするということは、データ上では覆りません

## 戦闘

### HPと被ダメがキツくない？

- 厳しい世界です
- 無防備な状態でナイフで数回刺されたら、英雄でも死ぬのがルゥレーンです
- 低FTHのキャラクターでも、杖や供物を借りることで回復の奇跡が使用可能です
    - あまりスペル適性がないメンバーのMPを遊ばせないようにするのも、リソース管理の重要なポイントとなります

### MP管理が厳しすぎない？

- 厳しい世界です
- 賢者のRPをしたい場合は、リソース管理の面でリアルな賢者になる必要があります
- 触媒や供物を活用してください

### 魔法・奇跡のためにMPを温存していると、後衛が暇です

- 暇な手番でも「警戒指示」などの行動が可能です
- 武器がなくても、投擲で攻撃することが可能です
- 前衛でスタートし、最初の手番で近接攻撃してから後衛に移動、というプレイも可能です
- 逆に後衛でスタートし、リアクティブヒールを狙うというプレイも可能です
- あなたのPCで、今何ができるかを考えて行動すると良いでしょう

### 武器の持ち替えは本当に自由でいいの？

- はい。自身のターン内であれば、行動に関係なく武器を自由に持ち替えられます
- ロングソードとメタルシールドを装備した状態で、投げナイフを予備行動で投げ、ロングソードを2Hで攻撃し、その後に剣と盾を片手に持つことが可能です

### 投げ紐などで投擲する石って、拾える場所に制限ある？きれいな屋内だと使えないなどある？

- ありません
- 投擲の攻撃手段は、ルール上、弾を自由に補充できるものとして扱ってよいです
- ただし、ルール上で無限に石を生成できても、ロールプレイ上で無限に石を積み上げる行為はできません

### 投擲の石などは、拾い上げるために予備行動を使用しますか？

- 使用しません
- 手元で素早く石を用意できているものとして良いです

### 防護を下げる効果で、防護はマイナスの値になりますか？

- なりません
- 防護が減少によって負の値となる場合、防護は0として扱います

### 威力がマイナスの値の場合の時、ダメージはどうなりますか？

- 通常の攻撃と同様に $威力+攻撃判定MtV$ で計算します
- 例: 威力が-2で、攻撃判定のMtVが2の場合、ダメージは0となりますが、MtVが3の場合は1点のダメージを与えることができます

### ダメージがマイナスの値となった場合、逆に回復しますか？

- 回復しません。0点のダメージとして扱います
- 例: 威力が-2で、攻撃判定のMtVが1の場合、計算結果は-1となりますが、0点の攻撃として扱います
- 魔法やアイテムによるダメージについても、同様に扱います

### 防具やアクセサリで複数の装甲があるとき、どこのアイテムから装甲を減算しますか？

- 任意の箇所の装備の装甲値から増減して構いません
    - 例: レザーアーマーと帽子を装備中に2点の攻撃を受けた時、レザーアーマーから2点引いても良いし、帽子から1点引き、残りをレザーアーマーで受け持っても構いません
- 装備を捨てていく場合など、個別の数値を参照する時に、後で計算しても大丈夫です
    - 例: 上記の例で装甲が2/4のとき、新品のレザーアーマーを見つけたので装備を取り替えるとき、レザーアーマーが2点削られていたことにしても良い
- 修理判定などの装甲を回復する手段で回復量が個別の装備の装甲値を上回っている場合も、分配して大丈夫です
    - 例: レザーアーマーが2/3、帽子が0/1のとき、装甲を回復する効果で2点の回復があった場合、レザーアーマーと帽子をどちらも1点ずつ回復しても良い

### 防護が複数あるときにマイナスしたら、どこのアイテムから減算しますか？

- 装甲と同様に、任意の箇所に対して防護が減少したものとして扱ってよいです
- ゲームの進行処理を軽量化することが目的の裁定です
- 後から「実は盾じゃなくて防具の防護が減っていました」としても良いです

## その他

### キャラクターを持ち上げる場合は、どのような処理を行いますか？

- 重量:3の所持品として担ぎ上げることができます
- 才能「小柄」を持っている場合、重量を-1します
- 才能「恵体」を持っている場合、重量を+1します
- ケンタウロスは重量を+2します
- バードフォークは重量を-1します
- ハーフリングは重量を-1します
- フェアリーは、常に重量:0として扱います
- そのキャラクターの装備品や携行品ごと持ち上げる場合は、その重量を加算します
    - 所持品枠は、そのキャラクターの枠を利用することが可能です (持ち上げた側の所持品として扱わない)

### 一日にどれくらいのお金を使うの？

- 一日の生活費は、WLT 0です
    - 無料という意味ではありません
    - ゲームシステム上ではお金として計上されませんが、ルゥレーンの実世界上ではお金が動いています
- 通常、街で生活するのに必要なお金は、ひと月あたりWLT 1～2程度です
    - これは2026現在の日本でいう、6万円～15万円程度の金銭です
    - 暮らす街や地域によって異なりますが、おおよそこの程度のWLTで生活することが可能です
- WLT 12あれば、一年を生き抜くのに足ります
    - これは2026現在の日本でいう、100万円～150万円程度の金銭です
- WLT 20程度あれば、一年は標準的な暮らしが送れます
    - 250万円～300万円程度というイメージです

### メイキング直後のお金の数値が低すぎない？ WLT 10が現実で車1台という目安だとしても、老年ならもっと稼いでるじゃないの？

- 背景により獲得するWLTは「冒険者になるために捻出できた資金」という意味です
- PCたちが冒険者になる以前は、何年もの間その生業を続けてきています
- 「一般市民が個人で蓄財できる限界」がWLT 500です
    - これは、様々な生活費などを工面しつつ、倹約して貯蓄した場合の資産です
    - 大半の人々は、生涯でここまで大きな資産を蓄財しません。およそWLT 50～150の貯蓄がせいぜいです
- 冒険者になるためにすべての資産を突っ込める人は稀です
    - 多くの場合、家族や親戚、世話になった人たちのためにWLTを消費しています
    - そうした中で残ったお金が、メイキング直後のWLTです
- WLT 10はかなりの資産ということです
    - 隣県への旅行で、みんながバイクに乗る中、一人だけ車に乗ってくるようなイメージです
    - 高級車 (≒秘宝級の武具) には及びませんが、それでもかなり奮発していると言えるでしょう

### ってことは駆け出し冒険者のWLT 2って結構な金額じゃない？

- そうです。WLT 2は、およそ10万円だとお考えください
- 往復などを含め、約1週間の勤務で10万円相当の給与がもらえるのが駆け出し冒険者です
- 実際は、戦利品の売買などもあるため、もっと稼ぐ事ができます
    - 1度の冒険でWLT 5を稼いだとすると、30万円程度の収入となります
- 冒険者は命を賭して文明に貢献しているため、これくらいの収入が得られるようになっています
- 「一週間の短期バイトで、未経験でも30万稼げるよ！でも5%の確率で死者が出ているよ！」という仕事だと思ってください

### なんで異種族交流できてるの？恨みとかないの？

- 恨みの歴史はかなり根深いです
- 秩序の秘宝がもたらされる前は、普通に種族間の闘争があちこちで起こっていました
- エルフは人間狩りを普通にしていましたし、ドワーフはエルフの森を焼きまくっていました
- マーフォークは海辺の人族や家畜を攫っていましたし、バードフォークも地上の生物を襲っては食料にしていました
- ヒュームは他種族と手を組み、獣人狩りをしたり、奴隷売買も行っていました
- それでも今の平和があるのは、同じ神を信仰しており、同じ言葉を話せるようになったからです
- 種の保存のためには共生するのが最も効果的だ、というドライな理由で文明に参加している者も、もちろんいます
- 社会への適応ができない者、文明そのものに敵対する者には、統一文明は容赦をしません
    - 罪人の扱いが非常に厳しいのは、それが理由です
    - 魔族は言うに及びませんが、それと同じことをする者は、魔族扱いもやむ無し、という考えです
- 魔族たちへの排他性と、脅威に対する軍事的同盟というのも、統一文明が指示されている要因のひとつです
    - 非常に危ういバランスの上で、ルゥレーン共和国は成り立っています
- 負の側面だけではなく、異種族交流の結果として文明が大きく栄えてもいます
- 知らなかった美味しい食事、便利な道具、役割の分担など、もはや種族間対立をしていた頃には戻れない者も多いでしょう
- 近代化に伴って種族間の偏見が取り払われて行っていると捉えることも可能です
    - それどころか、魔族への人権運動といった動きも、もしかしたら行われているのかもしれません

### クラスがないのはなぜ？

- 多様な遊びを保証し、死にデータを減らすためです
- クラスごとの決まったプレイ指針があると、毎回のセッションが似た展開となります
- GM側が用意するPLへのメタ戦術が固定化されるなど、冒険の多様性を狭める要因となります
- こうした制約は、シナリオ制作へのストレスとなり、GMが減る→プレイの機会も減る、という流れにつながります
- ルゥレーンTRPGではクラスをオミットし、ステータスと装備、種族によってビルドとロールを決定します

### レベルがないのはなぜ？

- 死にデータを減らすためです
- ルゥレーンは「高レベルの冒険者になると、ゴブリンはいてもいなくても同じ」という論理が通用しない世界です
    - 相当な達人でも、徒手で10匹のゴブリンに囲まれたら死にます
- WLTが実質的なレベルの代替として機能するようにシステムを設計しています
- 装備を制限するだけで、ある程度上級の冒険者とも一緒にプレイできるようにしています
- ステータスの1点が重たいルールなので、成長の程度はPCの戦力に大きく影響します
    - なのでレベル制に近い要素もあります
    - 一般的なルールと比較して、レベル制の比率が小さいということです

### PCに使命とかはないの？

- 冒険者が義務的に冒険をする理由付けは、世界観やルール上には一切存在しません
- 「GMが全部の世界観を知っていないとPLがロールプレイできない問題」を解決するために、ルール上では振る舞いの指針を指定していません
    - NPCとの関係性などをルールやクラスで制限すると、どうしても「普通の冒険」が進行しにくくなります
    - 同様の理由から、使い魔や騎獣といったシステムも導入していません
- ルールブックには用意していないだけであり、PLやGMの判断でPCたちに冒険の理由を与えることは、推奨します
