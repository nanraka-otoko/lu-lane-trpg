# スペル / 魔法 / 奇跡

- このドキュメントでは、魔法や奇跡などのスペルの行使に関するルールとデータを扱います

---

## スペルの概要

- キャラクターが用いるスペルは、「魔法」と「奇跡」の2系統に分かれる
- どちらも MP を消費して行使する
- 行使時の判定に使う能力値は、魔法と奇跡で異なる

| スペルの系統   | 判定能力値 |
| :------------- | :--------- |
| 魔法 / Sorcery | INT (知力) |
| 奇跡 / Miracle | FTH (信仰) |

---

## 難易度と行使条件

- すべてのスペルには難易度が設定されている
- 難易度以上の能力値を持つ場合にのみ、そのスペルを行使できる
- 具体的には、下記の条件を満たすスペルをすべて行使できる

$$
能力値 >= 難易度
$$

> 例:
> - INT6のPCは、難易度6以下の魔法をすべて行使できる
>
> All or Nothing:
> - 基本的に「魔法を習得する」といった仕組みはなく、能力値以内の難易度のスペルはすべて行使可能
> - 混交魔法のように習得が必要なスペルが一部存在する
>
> 難易度の増減:
> - 難易度を増減する効果を得ている場合、行使条件が緩和・強化される
> - 難易度が0未満となる場合、0として扱う

---

## 魔法行使判定 / 奇跡行使判定

- 魔法と奇跡を行使する場合、魔法行使判定/奇跡行使判定を行う
- この判定時に振るダイス数は、以下の式で決定する

$$
ダイス数 = 能力値 - 難易度 + 各種補正
$$

- MtV 1以上で成功し、そのスペルが発動する
- MtV 0の場合 (ダイス数が0) の場合、そのスペルは行使できず、発動しない

> 能力値 = 難易度の場合:
> - 能力値と難易度が同値の場合、ボーナスダイスがないと MtV が 0 となり、そのスペルは行使できない
> - この場合は、何かしらの手段でボーナスダイスを得ることでそのスペルを行使できる
> 
> 難易度3の魔法を使う場合の例:
> - INT5 → (5-3) = 2M6 で判定
> - INT7 → (7-3) = 4M6 で判定 → 高い達成値が出やすく、効果が強力
> - INT3 → (3-3) = 0M6 → ダイスがないため行使不可
> - INT3 +1d → (4-3) = 1M6 で判定 → 能力値ギリギリならダイスを足せば行使可能
> - INT2 +2d → INT が難易度未満なので、ダイス数があっても行使不可

> 注意点:
> - スペルの行使条件は能力値のみに依存する
> - ボーナスダイスがあったとしても、自身の能力値を上回るスペルは行使できない
>
> 能力値と難易度とボーナスの例:
> - INT3 のキャラクターは、ティンダー (難易度2) とディスペル (難易度3) を行使できるが、サンダーボルト (難易度4) は行使できない
> - そのままではディスペル (難易度3) はダイス数0となり行使できないが、ボーナスダイスを得ることで行使可能となる
> - サンダーボルトは難易度が INT を上回っているため、ボーナスダイスを+2d以上得ても、そもそも行使ができない
>
> 難易度と能力値の考え方:
> - 能力値がスペルの難易度に満たない場合「そもそもそのスペルを行使する術を知らない」という状態を指す
> - どれだけ補助を得ても、知らないスペルを行使することはできない

---

## 魔法 / 奇跡への抵抗判定

- 魔法と奇跡への抵抗判定は存在しない
- 魔法と奇跡によるダメージや効果は無条件に対象に生じ、防護と回避では防ぐことはできない
- 障壁と装甲によって、HPへのダメージを軽減することは可能
- 魔法や奇跡によって生じる毒や睡眠などの状態異常には、生命力判定や精神力判定で抵抗できる

## スペルの射程

- 戦闘ルールのエリア制に準じる
- 射程についての詳細は [../07-combat/](../07-combat/) の「射程」を参照

| 射程       | 届く範囲           | 例                                      |

| 自身       | 自分自身           | 自分自身に「リコール」                  |

| 3 / 長距離 | 3つ隣のエリアまで  | 味方後衛 → 敵後衛に「アースニードル」   |
| 特殊       | スペルにより異なる | 「雨乞い」により、現在地域に雨が降る    |

## スペルの行使の手順

1. 行使するスペルと対象を宣言する
2. MP を消費する
    - 最低1点を消費する
    - どんな術でもMP1は必ず消費する
    - コスト軽減の下限: コスト低下の効果があっても消費MPは0以下にはならず、最低1のMP消費は免除されない
    - 「消費MPを0にする」と記載がある場合に限り、消費MPを0できる
3. 魔法行使判定 / 奇跡行使判定でロールする
4. スペルが発動する
    - 達成値に応じて効果量が変動する

## 予備行動で行使が可能なスペル

- 効果の説明に「予備行動可」と書かれたスペルは、予備行動としても行使できる
- 予備行動についての詳細は [../07-combat/](../07-combat/) の「予備行動」を参照

## 触媒 / 供物

- スペルの行使時に消費して、追加効果を得られるアイテム
- 行使判定にダイスを足したり、消費MPを軽減したりできる
- 1回のスペル行使につき、複数の触媒や供物を同時に使用できる
- 同時に使う触媒・供物の数が増えるほど、そのスペルの難易度が上昇する
    - 1つ使用: 難易度+0
    - 2つ使用: 難易度+1
    - 3つ使用: 難易度+2
    - N個使用: 難易度+(N-1)
- スペルの効果欄に必須の触媒 (骨、死体など) が指定されている場合、その触媒は1個目として扱われる
    - 追加で他の触媒を併用する場合、2個目から難易度が上昇する
- 触媒や供物の一覧は [../05-items/04-catalyst.md](../05-items/04-catalyst.md) を参照すること

> 解説:
> - 高価な触媒を用いることで、自身の能力値を大きく超えた魔法や奇跡の行使も可能となる
> - 良い触媒を1つ使うか、安い触媒を多く重ねて弱い魔法を強化するかは、術者の判断に委ねられる

---

## 効果時間

- 一部の魔法・奇跡には効果時間が存在する
- 戦闘中では、効果時間はラウンドの終了時にカウントが進む
    - 詳細は [../07-combat/](../07-combat/) の「魔法やアイテムの効果時間について」を参照すること
- 術者は、スペルの効果時間が経過する前に、その効果を任意に解除することができる

---

## 重複不可の原則

- 同じ魔法・奇跡の効果は、同じ対象に対しては重複しない (同時に作用しない)
    - 異なるキャラクターが同じ対象に対して同じ魔法・奇跡を使用した場合も、効果の重複は生じない
    - 同じ効果であっても、異なる魔法・奇跡によるものであれば、効果は重複する (同時に作用する)
- MtVを参照して効果量が変動する場合、高い方の効果を優先する
- ラウンド数が定められている場合、後にかけられた効果で効果時間を更新する
    - 先にかけられた効果の持続時間の方が長い場合、効果時間は更新せず、元の時間を維持する

> 障壁について:
> - 障壁を得る効果は、障壁に加算されるため、同一の魔法・奇跡でも重複する
> - 効果時間は、後にかけた障壁の効果時間で更新する
> - 異なる手段で得た障壁でも、同様に加算する。別個に管理しない

# 魔法の一覧

## [一般魔法](./01-common-sorceries.md)

- すべての種族が行使できる魔法
- 知力があれば、どの種族でもこれらの魔法を使用できる

## [混交魔法](./02-assorted-sorceries.md)

- 生活を豊かにするために生み出された、雑多な魔法
- 知力があればどの種族でもこれらの魔術を行使できるが、行使するためには習得が必要となる
- WLT 5を消費することで、混交魔法をひとつ習得できる

## [禁止魔法](./03-forbbiden-sorceries.md)

- 死霊術や呪術など、秩序文明では禁忌とされる魔法
- すべての種族が行使できる
- スペルを初めて行使すると、そのキャラクターのAPPが恒久的に1点減少する

## [悪魔の魔法](./04-evil-spells.md)

- 悪魔と、それに連なる眷属のみが行使できる魔法
- 通常のキャラクターが使うことは想定されていないスペル

# 奇跡の一覧

- PCのメイキング時に己が信仰する神を1柱選択する
    - すべてのPCは真実神を信仰している
    - 真実神に加え、他の神を1柱、追加で信仰できる
- キャラクターは、自身が信仰している神の奇跡のみを行使することができる
- ルゥレーンでは真実神に連なる神が多く信仰されるが、PCは外なる神を信仰しても良い

## [真実神の奇跡](./11-the-god-of-truth.md)

- 真実神がもたらす奇跡。神聖語を用いて行使する
- 秩序の陣営であり、信仰心があれば、誰でも行使できる

## [豊穣神の奇跡](./12-a-god-of-abundance.md)

- 真実神に連なる神
- 大地と雨の神であり、豊穣と繁栄を司る

## [鍛冶神の奇跡](./13-a-god-of-forge.md)

- 真実神に連なる神
- 炎と鉄の神であり、鍛冶と創造を司る

## [戦武神の奇跡](./14-a-god-of-war.md)

- 真実神に連なる神
- 笛と剣の神であり、戦争と武勇を司る

## [狩猟神の奇跡](./15-a-god-of-hunting.md)

- 真実神に連なる神
- 生命と風の神であり、狩猟と自律を司る

## [舞歌神の奇跡](./16-a-god-of-cultures.md)

- 真実神に連なる神
- 歌と踊りと酒の神であり、文化と娯楽を司る

## [海淵神の奇跡](./17-a-god-of-the-abyss.md)

- 真実神に連なる神
- 海と雷の神であり、豊漁と死を司る

## [外星神の奇跡](./21-the-god-from-beyond-the-stars.md)

- 外なる神のひとつ
- 空の果ての星々の神とされる

## [未踏神の奇跡](./22-the-god-of-the-frontier.md)

- 外なる神のひとつ
- 未知なる智慧や、知られざる未開の地を授ける神

## [悪魔神の奇跡](./23-the-god-of-evils.md)

- 魔族たちが信仰する神
- 秩序文明では信仰されておらず、存在は知られていても実態はほとんど解明されていない
- その奇跡の神威は真実神と似ていることから、真実神の一側面であるとも囁かれる
- PCは悪魔神を信仰できない

# 種族別の魔法 / 奇跡

- その種族固有の魔法や奇跡
- PCは自身の種族のスペルを行使できる
    - 「混血」の才能を持つPCは、その種族のスペルも行使可能
- 詳細は各種族のページを参照すること

# スペルの難易度や、信仰していない神の奇跡についての扱い

- 世界観の理解やロールプレイに役立ててください

> - 能力値がスペルの難易度を満たしていない場合、そもそもその人物はそのスペルのことを良く知らない
>     - FTHが3の人物は、「蘇生」の奇跡は「真実神の神殿の奇跡で、お金を払うと死者が蘇る」程度のことしか知らない
>     - INTが3の人物は、「ファイアボルト」や「アイシクルダート」のことを「炎や氷の矢を飛ばせるのが魔法使い」程度の認識
>     - 一般人であれば「真実神は傷を癒してくれる」「魔法は魔物を追い払う攻撃手段になる」といった知識は持っている
> - 真実神の奇跡や一般魔法は、統一文明内で広く普及しているので、学のある人はスペルの概要を知っていることもある
>     - FTHが3でも、物知りな人間は「真実神の蘇生は、肉体と精神をすり減らす」ということを知っている場合もある
>     - 具体的にどういうリスクがあるのか、どういうコストが必要なのか、どういう手続が必要なのかは、FTHが7ないと知らない、というイメージ
> - 信仰していない神の奇跡については、そもそも「何の神なのか」すら知らないことがほとんど
>     - 鍛冶神を信仰していない人にとって、鍛冶神は「鍛冶の神なんだよな」ということだけ知っていて、「鍛造の奇跡がある」ということは知らない、というイメージ
>     - 現実で例えると、クリスチャンはコーランを読んでいない人がほとんどだし、寺の住職は聖書を読んでいないことの方が多い、ということ
> - 自身が信仰している神については、能力値が足りていなくても、上位の奇跡で何ができるのかを概要レベルで知っている
>     - FTH3の豊穣神信仰者は「この地域の作物の稔りが良いのは、豊穣神の神官が奇跡を行使したからだ」ということを理解できる
>     - 豊穣神を信仰していない者は「このあたりの地域は作物が良く実っているな～」程度の認識
>     - 豊穣神信仰者から説明を受けると「へぇ～ありがたい神様なんだぁ～」程度の認識となる
> - 他の種族の魔法や奇跡についても、信仰していない神の奇跡と同じくらいの認識
>     - 「エルフは精霊の魔法を使えるらしい」「ドワーフは鎧を直したりできるらしい」ということだけが、多種族に伝わっている
> - 外なる神については、信仰していない者は何の神なのかすら知らないどころか、存在すら認知していない者が大多数
>     - 「未踏神って知ってる？」と聞いても、99%以上が「知らない」と答えるレベルの認知度
>     - 未踏神の信者ですら未踏神の奇跡について理解しているわけではない
>     - 体系化された神学すらないため、教義といったものが存在しない。ただ、その神威を認める者がいて、信仰されているという状態
> - PLはルールブックに書いてあることをすべて認識できるが、ルゥレーンの世界の人々は「自身の信じる信仰や理論に対して認知を深め、知性や信仰心が足りないと深く理解できない」という原則に基づいている
>     - PLが知っていることを、PCは知らないかもしれない
> - 実際のロールプレイの濃度については、セッション内で集まったGMとPLの好みで調整して構いません
>     - 若年のヒュームが、なぜか信仰していない外星神や未踏神に詳しくても、ロールプレイの一種として認められるものです
>     - GMが一般人のNPCのロールプレイをする場合の参考程度に参照すると良いでしょう
