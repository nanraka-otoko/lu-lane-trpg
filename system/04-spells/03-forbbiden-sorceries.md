# 禁止魔法

- 秩序文明では禁忌とされる魔法
- INT で行使判定する
- スペルを初めて行使すると、そのキャラクターのAPPが恒久的に1点減少する
    - この減少はスペルの種類ごとに累積する
    - 「アニメイトデッド」のみを何度行使しても、APPは1しか減少しない (減少は最初の1回目のみ)
    - 「アニメイトデッド」の行使経験がある状態で「カース」を行使すると、さらにAPPが減少する

> 禁忌:
> - 禁止魔法も、一般魔法と同様に体系化されているため、知識があれば使用できるが、好んで使う者は少ない
> - 使用に際し、何かしらの道具や代償を用いることがある
> - 行使を他者に見られると、その者からの信頼を失う
> - 街中で行使した場合、即座に衛兵に囲まれる程度には禁忌とされる

## APP の減少について

- 禁止魔法を行使した際にAPPが減少しますが、必ずしも容姿の悪化を意味しません
    - PLの解釈により、容姿が損なわれたものとして扱っても良いです
    - エクストラアームやライズアンデッドのように、見た目が悪化することでAPPを失う魔法もあります
- 禁止魔法を行使すると「禁止されている魔法を平然と使える人格破綻者」という属性がキャラクターに染みつきます
    - 見た目は純朴そうな少年や、信頼できそうな神官、絶世の美女だとしても、会話の節々に異常が生じます
- 禁止魔法の行使には集中力(NMD)や、それを知るための知識(INT)が必要です
    - そのため、禁止魔法で精神に異常をきたしたとしても、MNDやINTは減少しません
- 他者との会話で不穏なワードが垣間見えたり、単純に信頼を失うという形でAPPを損失します

### ロールプレイの例

APP 7でメイキングしたPCの場合:
- 禁止魔法の常用により、APPが1まで減少している
- 見た目はAPP 7の可憐な少女だが、人格が破綻しており、会話の節々で異常が見受けられる
- パン屋での会話の例:
    - 「あら、可愛いお嬢さん、お腹が空いているのかい？ これ、焼き立てのパンだよ」
    - 「あ、ありがとうございます。この小麦、ちょうど死体を早く腐らせるための菌床に最適なんですよね。効率が3倍になります」
    - 「ヒッ……！！」(怯える店主)
- 対人判定のMtVが減少する実態:
    - まともな会話が成り立たない
    - 相手の気持ちを理解できない
