# ロールシステム

- このドキュメントでは、ルゥレーンTRPGで用いるダイスロールのシステムについて解説します

---

## ロールシステムの概要

- ルゥレーンTRPGでは、判定に6面ダイス (d6) を複数個使用する
- 能力値の数だけダイスを振り、同じ出目が最も多く揃った個数が達成値となる
- 達成値と目標値を比較して成否を判定する

---

## ロールの記法

ロールの記法は $NM6$ を用いる:
- N: 振るダイスの数 (= 能力値)
- M: マッチ判定 (Match)
- 6: 6面ダイス

達成値の略称は $MtV$ (Match Value) を用いる:
- 例: 筋力 5 のキャラクターが筋力判定を行う場合 → 5M

状況修正がある場合は加減で表記する:
- 有利な状況の場合は +1d → $5+1M6 = 6M6$
- 不利な状況の場合は -2d → $5-2M6 = 3M6$

攻撃や魔法の効果で達成値を参照する場合、下記のように表記する:
- 効果量の表記例: 「1+MtVダメージ」「HPをMtV点回復」

---

## 判定の手順

1. 判定に使用する能力値を確認する
2. 能力値と同じ数の d6 を振る
3. 出目の中で、最も多く揃っている目の個数 = MtV
4. 判定に目標値が存在する場合、MtVが目標値以上なら判定成功、未満なら判定失敗
5. 判定に目標値が存在せず MtV だけを算出する場合、その MtV を適用する
    - 攻撃判定、魔法行使判定など

---

## 達成値 / MtV / Match Value

$$
MtV = 最も多く揃った目の個数
$$

|  MtV | 出目の例 (6個振りの例) |
| ---: | :--------------------- |
|    1 | $[1, 2, 3, 4, 5, 6]$   |
|    2 | $[1, 1, 3, 4, 5, 6]$   |
|    3 | $[1, 1, 1, 4, 5, 6]$   |
|    4 | $[1, 1, 1, 1, 5, 6]$   |
|    5 | $[1, 1, 1, 1, 1, 6]$   |
|    6 | $[1, 1, 1, 1, 1, 1]$   |

> 注意: 
> - 複数の揃い目がある場合 (例: $[1,1,1,3,3,3]$) 、最も多く揃っている方のみを参照する
> - 上記の例では3つ揃いが2組あるが、MtV は 3 となる

---

## 目標値の目安

| 目標値 | 難易度       | 説明                                         |
| -----: | :----------- | :------------------------------------------- |
|      1 | 自動成功     | ダイスを振れる状況なら、必ず達成できる       |
|      2 | 基本         | 通常の判定。7個以上振れる場合は必ず成功する  |
|      3 | 難しい       | 3個の場合は3%未満。7個振って五分五分         |
|      4 | 非常に難しい | 能力がなければ絶対に出ず、能力あっても運次第 |
|      5 | 至難         | 有利な状況が重なった上で、さらなる運が必要   |

---

## 判定のボーナス / ペナルティ

- 状況に応じてダイスの数を増減することで、判定の有利・不利を表現する
- ダイスを増やすことを「ボーナス」、減らすことを「ペナルティ」と呼ぶ
- ダイスが0個以下になった場合、その判定はロールできず、自動的に失敗となる

| ボーナス / ペナルティ | 説明                                               |
| --------------------: | :------------------------------------------------- |
|                    -4 | 極めて不利な状況。息のできない水中、崖の登攀中など |
|                    -3 | 大きく不利な状況。複数の不利状況が重なった状態など |
|                    -2 | 不利な状況。煙や暗闇で視界が悪化した中での行動など |
|                    -1 | 少し不利な状況。ぬかるんだ地面など                 |
|                    +1 | 少し有利な状況。辞書を読みながらの解読など         |
|                    +2 | 有利な状況。得意な行動、道具の補助など             |
|                    +3 | 大きく有利な状況。複数の有利状況が重なった状態など |
|                    +4 | 極めて有利な状況。背後からの奇襲など               |
