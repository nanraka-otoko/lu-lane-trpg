# 属性と状態異常のルール

- このドキュメントでは、攻撃や魔法・奇跡の属性と、状態異常について解説します
- 戦闘のルールとして記載していますが、冒険中に登場する属性や状態異常は、このドキュメントのルールに準じて処理されます

---

## 属性

- すべてのダメージソース (武器、矢弾、魔法、奇跡) には属性が設定されている
- エネミーや種族の特性、装備の効果により、特定の属性の効果が高まったり、その属性で受けるダメージが増加・減少したりする

| 属性   | 説明                                       |
| :----- | :----------------------------------------- |
| 物理   | 武器・矢弾による打撃・斬撃・刺突           |
| 魔力   | 魔法の力                                   |
| 火炎   | 熱、火、爆発                               |
| 雷     | 雷撃、電撃                                 |
| 冷気   | 冷気、氷雪                                 |
| 水     | 水流                                       |
| 土     | 岩石、土砂                                 |
| 風     | 暴風、空気                                 |
| 毒     | 毒物、酸、腐敗                             |
| 神聖   | 神々の奇跡、神聖な光                       |
| その他 | 暗黒、補助効果など上記に当てはまらないもの |

属性の指定がない場合:
- 攻撃手段に属性の指定がない場合は、そのダメージは物理属性として扱う

魔法・奇跡の属性:
- すべての魔法・奇跡には属性が定められている
- そのスペルによるダメージや効果は、そのスペルの属性として適用する

> 攻撃手段の属性:
> - 攻撃手段の効果に「火炎属性のダメージを与える」などと記載がある場合、その攻撃はその属性として扱われる
> - 物理属性以外の攻撃手段でも、攻撃によって生じたダメージは回避・防護できる
>     - 魔法・奇跡によるダメージは攻撃判定を行わないため、物理属性のダメージでも回避・防護はできない

---

## 状態異常

- 状態異常には強度が存在し、それに応じて効果量が変動する
- 強度は非線形的な指標である
    - 強度3の状態異常は、強度1の状態異常の3倍の効果とは限らない
    - 通常、指数関数的に効果量が増大する
- 状態異常によっては、付与時に抵抗を行うことができる

### 毒

- 身体を毒で侵された状態
- 付与時に、生命力判定で抵抗する
- ラウンド開始時、毒の強度に応じてHPを失う (軽減不可)
- 1点の毒を受けた場合、1点のHPを失う
- 非戦闘状態にある場合、10秒おきにHPを失う
- 毒はその強度x1日持続する
    - 2点の毒の場合、2日持続する

### 病気

- 身体を病気で侵された状態
- 付与時に、生命力判定で抵抗する
- CON、STR、DEXが病気の強度x1点減少する
    - この効果で一時的にCONが0以下になる場合、そのキャラクターは死亡する
    - STR、DEXが一時的に0以下になる場合、そのキャラクターは行動できなくなる
- HPが回復できなくなる
- 一日経過するごとに病気の強度x1点のHPを失う
- 病気はその強度x1週間持続する
    - 2点の病気の場合、2週間持続する

### 昏睡

- 通常の睡眠とは別に、精神操作や毒により眠らされている状態
- 付与時に、精神力判定で抵抗する
- 自身のターンで行動できなくなる
- 昏睡はHPに強度x1点のダメージを受けることで解除される
- 昏睡は強度に応じて効果の持続時間が異なる
- 強度1: 3ラウンド
- 強度2: 30ラウンド
- 強度3: 1時間 (360ラウンド)
- 強度4: 1日間
- 強度5: 能動的に起きることはできない

### 魅了

- 何者かに魅了されている状態
- 付与時に、精神力判定で抵抗する
- 自身の行動の決定権が魅了対象に移る
- どの行動を強要されるかは、魅了された対象や手段により異なる
- 強度を目標値として精神力判定を行い、成功することで解除できる
- 精神力判定はラウンド開始時に行い、魅了の強度によって判定を試みる頻度が異なる
- 強度1: 1ラウンドごと (解除を試みれる場合、自動成功する)
- 強度2: 3ラウンドごと
- 強度3: 1時間ごと (360ラウンドごと)
- 強度4: 1日間
- 強度5: 能動的に魅了を解くことはできない

### 凍結

- 氷や冷気により行動を阻害されている状態
- 付与時に抵抗することはできない
- 火炎属性のダメージを受けると、そのダメージの分だけ強度が減少する
    - 強度3の凍結を付与された状態で、2点の火炎属性ダメージを受けると、強度1の凍結に状態異常が軽減される
- 凍結を付与された対象は、強度に応じてDEXが減少し、移動や行動に制限を受ける
    - この効果でDEXが一時的に0以下になる場合、そのキャラクターは行動できなくなる
- 凍結は時間経過で強度が1点ずつ減少する
- 強度1: DEX-1、予備行動での移動が不可、3ラウンド経過で強度を-1
- 強度2: DEX-2、移動が不可、6ラウンド経過で強度を-1
- 強度3: DEX-4、移動が不可、30ラウンド経過で強度を-1
- 強度4: DEX-8、移動が不可、1日経過で強度を-1
- 強度5: DEX-16、移動が不可、時間経過では凍結は軽減されない

### 重ねがけ

- 同じ状態異常を重ねがけされた場合、強度は加算されない
- 新たに付与される強度が現在の強度より高い場合のみ上書きする

> - 例: 強度1の毒を受けている状態で、新たに強度3の毒をかけられた → 強度3に上書き
> - 例: 強度3の毒を受けている状態で、新たに強度1の毒をかけられた → 変化なし (強度3のまま)

### 状態異常の抵抗

- 状態異常を付与されたとき、その状態異常によっては抵抗判定を行える
- 状態異常に対応するステータス (CON または MND) で NM6 をロールする
- 抵抗の MtV ぶんだけ、付与される状態異常の強度を減算する
- 強度が0以下になった場合、状態異常は発生しない
- 状態異常によっては、対抗できないものが存在する
    - 例: 凍結

| 抵抗ロール       | 抵抗する状態異常                 |
| :--------------- | :------------------------------- |
| 生命力判定 (CON) | 毒、病気など肉体に作用するもの   |
| 精神力判定 (MND) | 昏睡、魅了など精神に作用するもの |

> 例: 敵がスリープ (強度3) をかけてきた
> - 対象の MND 4 で 4M6 → MtV 2 → 強度 3-2 = 1 → 強度1の昏睡を受ける
>
> 例: 敵がポイズンミスト (強度2) をかけてきた
> - 対象の CON 5 で 5M6 → MtV 3 → 強度 2-3 = -1 → 抵抗成功。毒を受けない
> - 対象の CON 2 で 2M6 → MtV 1 → 強度 2-1 = 1 → 強度1の毒を受ける
