# 攻撃準備 / 防御姿勢 / 警戒指示

- このドキュメントでは、戦闘中において、次の攻撃や防御の準備を行う際のルールについて解説します

---

## 準備行動

- キャラクターは自身の行動として、次の攻撃の準備をしたり、防御の準備をしたりすることができる
- 各種準備行動は、行動として実行する

---

### 攻撃準備 (MND / FTH)

- 対象を1体選択し、目標値2で MND または FTH で判定を行う
- 判定に成功した場合、その対象に対する次の自身の攻撃判定に+2dのボーナスを得る
- 次の自身の行動で攻撃をしない場合や、指定した対象を攻撃しない場合、効果が解除される

> 複数を対象に攻撃する場合について:
> - 攻撃準備で指定した対象が含まれている場合はボーナスが得られる
> - 対象を狙って攻撃を準備するため、その対象を攻撃しない場合は効果を失う
>
> 魔法・奇跡について:
> - 攻撃判定にボーナスを得る効果であるため、魔法・奇跡の行使判定にはボーナスは得られない
> 
> MND / FTH 判定:
> - 攻撃の機会を狙うには、強い集中力か、己の未来を信じる心が必要となる

### 防御姿勢 (STR / DEX)

- 目標値2で STR または DEX で判定を行う
- 判定に成功した場合、次の自身のターンまで、回避判定に+2dのボーナスを得る

> STR / DEX 判定:
> - 敵の攻撃に対して防御姿勢を取る場合、フィジカルか身軽さが重要となる

### 警戒指示 (INT / APP)

- 対象を1体選択し、目標値2で INT または APP で判定を行う
- 判定に成功した場合、その対象の次の攻撃判定に-1dのペナルティを与える
- 複数人で同じ対象に警戒指示を行っても、効果は重複しない
- 警戒指示の対象が次の行動で攻撃しない場合は、効果が解除される

> INT / APP 判定:
> - 敵を正しく警戒するには、その相手を洞察する知性と、仲間との信頼が必要となる
