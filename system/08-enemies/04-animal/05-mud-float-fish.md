## 行動表

| 行動     | 射程 | 威力 | 命中 | 備考                                                            |

## 特性・能力

| 特性・能力 | 効果                                                                     |
| :--------- | :----------------------------------------------------------------------- |
| 飛行       | 空を飛ぶ                                                                 |
| エラ呼吸   | 水場がないエリアで6ラウンドが経過すると、ラウンド終了時にHPが1点減少する |
| 汚泥耐性   | 病気の状態異常を無効化する                                               |
