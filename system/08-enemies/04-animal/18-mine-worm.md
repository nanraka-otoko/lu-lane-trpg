# マインワーム

- 地中に潜み、鉱物を食らいながら成長するミミズ状の生物
- 成虫の体長は5m～10mにも達し、巨大な口で獲物を噛み殺す
- 人族たちを、餌場である鉱山を荒らす外敵であると見做し、苛烈な攻撃を加える
- 肉食生物ではないため、殺傷した外敵は捕食せず、その場に捨て置く習性がある
- その生まれは不明だが、地海の龍の眷属であるとも囁かれる

## 特性・能力

| 特性・能力 | 効果                                                                                                           |
| :--------- | :------------------------------------------------------------------------------------------------------------- |
| 潜行       | 地中に姿を隠すことができる。戦闘時は行動として地中に潜る。地中に潜っている間は攻撃や魔法・奇跡の対象とならない |
| 奇襲       | この者に対するイニシアチブ判定に失敗した場合、奇襲を受けての遭遇となる                                         |
| 光に弱い   | まばゆい光を浴びると、そのラウンド中はすべての判定に-4dのペナルティを受け、潜行ができなくなる                  |
| 本能       | 昏睡・魅了の状態異常を無効化                                                                                   |
| 巨体       | 包囲において、3人分として扱う                                                                                  |
| 大漁       | このエネミーには、2回まで戦利品判定を行うことができる                                                          |
