# ボーンド・アビス・シェル

- 深海に棲まう二枚貝
- 2m～3mの大きさの貝殻を形成し、その貝殻で獲物に噛みついて捕食する
- 周囲の地形に体色を同化させ、獲物を待ち構えて奇襲する
- 圧縮した水を噴出し、貝殻を羽ばたかせて水中を泳ぐ。大柄な見た目に反し、水中では機敏に動き回ることができる
- 海底洞窟の陸地部分に生息することもあり、水中以外にも適応している

## 特性・能力

| 特性・能力     | 効果                                                                                                   |
| :------------- | :----------------------------------------------------------------------------------------------------- |
| 海底に棲まう者 | この者は水中に適応しており、深海の高圧をものともしない                                                 |
| 擬態           | 1m程度まで接近すると奇襲を受ける。目標値3の知識判定か、目標値4の感知判定に成功すると見破ることができる |
| 貝殻           | このエネミーは、自身が攻撃したラウンド中は防護が0となる                                                |
