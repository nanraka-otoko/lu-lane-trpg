# ギガントトード

- 体長3mを超えるカエル
- 長い舌で獲物を捕らえ、捕食する

## ステータス

| 項目         |   値 |
| :----------- | ---: |
| カテゴリ     | 動物 |
| 陣営         | 中立 |
| 脅威度       |    2 |
| CON          |    4 |
| MND          |    0 |
| STR          |    3 |
| DEX          |    3 |
| INT          |    0 |
| FTH          |    0 |
| LUC          |    0 |
| APP          |    0 |
| HP           |    8 |
| MP           |    0 |
| 防護         |    2 |
| 回避         |    0 |
| イニシアチブ |    2 |

## 行動表

| 行動     | 射程 | 威力 | 命中 | 備考                                                                                                                                                                                              |
| :------- | ---: | ---: | ---: | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 舌攻撃   |    2 |    2 |   +1 | 対象を自身のエリアに向けて、エリアを1つ強制的に移動する                                                                                                                                           |
| 噛みつく |    0 |    4 |    0 | この攻撃により対象のHPが2点以下となった場合、対象を飲み込む。飲み込まれたものは行動不能となり、ラウンド開始時に1点のダメージを受ける。ギガントトードが3点以上のダメージを受けるとその場で吐き出す |

## 特性・能力

| 特性・能力 | 効果                                                |
| :--------- | :-------------------------------------------------- |
| 粘液       | 火炎属性のダメージを受けると、自身の防護は0点になる |

## ドロップ表

| 戦利品         | 目標値 | 価値 | 重量 | 説明                                                                                                                     |
| :------------- | -----: | ---: | ---: | :----------------------------------------------------------------------------------------------------------------------- |
| 大カエルの肉   |      1 |    0 |    1 | 淡白な味で歯ごたえがある玄人好みの肉。食料として扱える。死後の腐敗が早く、その日の内に食す必要があるため、市場価値はない |
| 大カエルの粘液 |      2 |    1 |    0 | カエルの体表を覆う粘液。潤滑剤として利用価値があるほか、加工を経て貴族向けの理容品に用いられることもある                 |
