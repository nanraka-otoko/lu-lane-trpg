# ポイズンスネーク

- 毒を持つ蛇の総称
- 実際には様々な種類の蛇が存在するが、冒険者からは区別されずに呼称される
- 素早い動きで静かに接近し、獲物に噛みついて毒を与える

## 行動表

| 行動     | 射程 | 威力 | 命中 | 備考                                                          |
| :------- | ---: | ---: | ---: | :------------------------------------------------------------ |
| 噛みつき |    1 |    2 |    0 | この攻撃でHPにダメージを与えた場合、対象に強度3の毒を付与する |

## 特性・能力

| 特性・能力 | 効果                               |
| :--------- | :--------------------------------- |
| 小さな身体 | 回避判定の累積ペナルティを無視する |
| 寒さに弱い | 自身への冷気属性ダメージを+2       |
