# クラーケン

- 体長1kmを越える巨大な軟体生物。海の怪物 
- 12本の触手を操り、獲物を叩きのめす
- 深海に棲息しているが、海面近くに浮上してくることがある

## 行動表

| 行動 | 射程 | 威力 | 命中 | 備考                                                                            |

## 特性・能力

| 特性・能力     | 効果                                                   |
| :------------- | :----------------------------------------------------- |
| 多脚           | この者は12本の触手を持つ。触手は1か月で1本再生する     |
| 脚を切る       | 自らの脚を切断することで、HPを10点回復する             |
| 巨体           | 包囲において、20人分として扱う                         |
| 海底に棲まう者 | この者は水中に適応しており、深海の高圧をものともしない |
| スケール       | 強度3以下の状態異常を無効化する                        |
| 大漁           | このエネミーには、20回まで戦利品判定を行うことができる |
