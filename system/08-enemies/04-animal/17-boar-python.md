# ボアパイソン

- 猪のような獰猛さを持つ巨大な蛇
- 牙に強力な毒を持ち、獲物を丸呑みにする

## 行動表

| 行動     | 射程 | 威力 | 命中 | 備考                                                                                                                                                                                            |
| :------- | ---: | ---: | ---: | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 毒牙     |    1 |    2 |   +1 | この攻撃でHPにダメージを与えた場合、対象に強度3の毒を付与する                                                                                                                                   |
| 締めつけ |    1 |    2 |    0 | 命中時、攻撃対象のDEXを攻撃判定のMtV点減少する。                                                                                                                                                |
| 噛みつく |    1 |    4 |    0 | この攻撃により対象のHPが2点以下となった場合、対象を飲み込む。飲み込まれたものは行動不能となり、ラウンド開始時に1点のダメージを受ける。ボアバイソンが3点以上のダメージを受けるとその場で吐き出す |

## 特性・能力

| 特性・能力   | 効果                                                            |
| :----------- | :-------------------------------------------------------------- |
| 湿地帯の住人 | ぬかるんだ地面や浅瀬で足を取られず、そのペナルティを無視できる  |
| 蛇の身体     | 自身への毒の状態異常の強度を-2。木を登る場合の汎用行動判定に+4d |
| 寒さに弱い   | 自身への冷気属性のダメージを+1                                  |
