# フロストベア

- 寒冷地に棲まうクマ
- とても凶暴であり、冬眠に備えて栄養を蓄えた秋季の個体は特に危険
- 生物としての身体の造りは凡庸であるが、その凶暴性から振るわれる攻撃は非常に強力

## 特性・能力

| 特性・能力 | 効果                                                                                                                                        |
| :--------- | :------------------------------------------------------------------------------------------------------------------------------------------ |
| 寒冷耐性   | 自身への冷気属性ダメージを-2。凍結の状態異常の強度を-1                                                                                      |
| リベンジ   | このエネミーは、最後に自分を攻撃した者を優先的に攻撃する                                                                                    |
| 威圧       | 戦闘開始時、威圧感を放つ。対象は目標値2で精神力判定を行い、失敗すると3ラウンドの間、すべての判定に-1dのペナルティを受け、移動ができなくなる |
| 巨体       | 包囲において、2人分として扱う                                                                                                               |
