# ゾンビ犬

- アンデッド化した野犬
- 獣のとしての本能だけが残っており、生きた肉に近寄り、肉が失くなるまで食らいつく

## 行動表

| 行動     | 射程 | 威力 | 命中 | 備考                        |
| :------- | ---: | ---: | ---: | :-------------------------- |
| 噛みつく |    0 |    3 |    0 | 対象に強度3の病気を付与する |

## 特性・能力

| 特性・能力 | 効果                                                                                                                               |
| :--------- | :--------------------------------------------------------------------------------------------------------------------------------- |
| アンデッド | 毒・病気・睡眠・魅了無効                                                                                                           |
| 復活       | HPが0以下の場合、ラウンド開始時にHP1で復活する。HPが-CON以下の場合は復活しない。神聖属性か火炎属性のダメージを受けると、効果を失う |
| 疾駆       | 包囲エリアへの移動を予備行動で行える                                                                                               |

## ドロップ表

| 戦利品     | 目標値 | 価値 | 重量 | 説明                                                                                                                              |

| 腐った犬肉 |      1 |    0 |    0 | 腐った犬の肉。食料になるが、そのまま食べると強度3の病気を受ける。清めることで食べることができる。これを愛食する好事家がいるらしい |
