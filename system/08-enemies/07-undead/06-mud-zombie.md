# 汚泥のゾンビ

- 汚泥を身にまとったゾンビ
- 振る舞いは通常のゾンビと同じだが、その不浄は強い瘴気を纏っている
- この不浄に触れた防具は、その価値を汚損されてしまうだろう

## 行動表

| 行動 | 射程 | 威力 | 命中 | 備考                                                                                                                                                      |
| :--- | ---: | ---: | ---: | :-------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 殴る |    1 |    3 |    0 | この攻撃でHPにダメージを与えた場合、対象に強度3の病気を付与する。この攻撃により装甲が減少した防具は、価値を1点失う。1時間以内に洗浄すれば価値を取り戻せる |

## 特性・能力

| 特性・能力 | 効果                                                                                                                               |
| :--------- | :--------------------------------------------------------------------------------------------------------------------------------- |
| アンデッド | 毒・病気・睡眠・魅了無効                                                                                                           |
| 復活       | HPが0以下の場合、ラウンド開始時にHP1で復活する。HPが-CON以下の場合は復活しない。神聖属性か火炎属性のダメージを受けると、効果を失う |
