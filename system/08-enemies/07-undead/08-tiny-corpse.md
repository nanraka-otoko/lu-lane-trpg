# タイニーコープス

- 埋葬された子どもの遺体がアンデッドとなったもの
- 病により倒れた子どもたちの遺体は、状態の良いものが多く、アンデッドとなった際に強力な個体と化すことがある

## 特性・能力

| 特性・能力 | 効果                                                                                                                               |
| :--------- | :--------------------------------------------------------------------------------------------------------------------------------- |
| アンデッド | 毒・病気・睡眠・魅了無効                                                                                                           |
| 復活       | HPが0以下の場合、ラウンド開始時にHP1で復活する。HPが-CON以下の場合は復活しない。神聖属性か火炎属性のダメージを受けると、効果を失う |
