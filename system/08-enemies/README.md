# エネミー

- このドキュメントでは、セッション中に登場するエネミーの操作やステータスについて解説します

## カテゴリ一覧とリンク

- 具体的なエネミーのデータについては、各カテゴリのディレクトリ内のファイルそれぞれ参照すること

### [人間](./01-human/)

- 野盗、傭兵など。秩序文明に属していながらも敵対する者
- 異端審問官や衛兵などの友好的なNPCもこのカテゴリに属する

### [悪鬼](./02-daemon/)

- ゴブリン、トロール、オーガなどの秩序文明に敵対する魔族
- 悪鬼族はオーガのことを指すが、それ以外も含めて「悪鬼」として扱う

### [悪魔](./03-evil/)

- ペイシャー、ミィドゥルなどの悪魔
- 魔族の一派であるが、悪鬼とは種族単位で区別される
- 秩序文明とは明確に敵対する魔族であり、その頂点が魔王である

### [動物](./04-animal/)

- 野生動物や虫など
- 自然の生態系に属する生物であり、基本的に下等生物であるが、強力な種も存在する

### [植物](./05-plant/)

- 植物型の生物。
- アルラウネのような秩序の種族に属する知的な個体や、タイニープラントのような野生の個体が存在する

### [魔法生物](./06-magical-creature/)

- スライム、ミミック、ゴーレムなど、魔法の力によって生まれた生物

### [アンデッド](./07-undead/)

- スケルトン、ゾンビ等のアンデッド

### [竜](./08-drake/)

- ワイバーン、地竜などの竜族
- 翼と前腕が一体化しているのが特徴
- 龍の血を引く卑小な末裔とされるが、一般的な動物と比して強力な種が多い

### [龍](./09-dragon/)

- ドラゴン
- 四肢とは別に翼を生やしており、竜と明確に区別される
- 始祖の龍に連なる存在であり、非常に強力な力を秘めている

### [異形](./10-another/)

- 異界からの来訪者
- ルゥレーンの理を外れた存在
- 別の次元や世界からの来訪者や、宇宙から飛来した者が該当する

### [重要人物](./99-legends.md)

- ルゥレーン上に存在する、あるいは存在した人物

## エネミーのステータスとパラメータ

- エネミーのステータスとパラメータは、PCに準じる
    - 詳細は [../02-player-character/01-status.md](../02-player-character/01-status.md) を参照すること
- エネミーは装甲の数値を持たない
    - PCの装甲に相当する装備を持つエネミーの場合、HPにその点数を加算する
    - 例: 「野盗」はレザーアーマーを装備しているので、装甲3点分を加算した数値でHPを設定する
- 回避:
    - 回避のパラメータは、戦闘処理の簡略化のために用いるデータである
    - 回避の数値を用いて回避判定をする場合、エネミーは回避判定でロールをせずに、回避の数値をそのままMtVとして用いる
    - 回避の数値はDEX/3を目安としている
    - GMの判断で、敵もPCと同様の回避判定を実行しても良い
        - 例えば強敵との戦闘の場合は、この判定を簡略化せずに処理するなど、使い分けると良い
    - 詳細は [../07-combat/](../07-combat/) を参照すること
- エネミーのデータをベースとして、個体差を設けても良い
    - 例えば、同じゴブリンでもHPが少しだけ高い個体、DEXが高い個体、攻撃手段が強力な個体などを用意しても良い

### 回避を簡略化した場合の累積回避ペナルティについて

- 同一ラウンドにおいて、連続で回避するごとに、回避の数値を-1して良い
- 通常の累積ペナルティと比較して重みが大きくなるが、簡略化のためとして良い
- 鞭などの累積回避ペナルティに追加のペナルティを与える効果についても、減少ダイス数を直接、回避の数値に反映して良い

## 陣営

- エネミーやNPCには陣営が定められている
- 陣営はあくまでデフォルトの態度であり、状況やGMの判断でその振る舞いは変化する

| 陣営 | 説明                                                                       |
| :--- | :------------------------------------------------------------------------- |
| 敵対 | こちらを捕捉すると敵として判断し、攻撃などの敵対行動をとる                 |
| 中立 | 理由があればこちらを襲うが、そうでなければ無視する、逃げるなどの反応をとる |
| 友好 | 基本的にはエネミー側からの敵対行動を能動的にとらない                       |

> 例: 
> - 中立の獣でも、巣に近づけば敵対になる
> - 友好のNPCでも、裏切りのシナリオでは敵対する
> - 「真実神の審問官」は基本的にPCに対して友好だが、異端認定した相手とは敵対する

## 脅威度

- エネミーの秩序文明に対する脅威度を 0～10 で表現する

| 脅威度 | 目安                                                                       |
| -----: | :------------------------------------------------------------------------- |
|      0 | 一般人でも対処できる、エネミーリストには載らない存在。野犬、害虫など       |
|      1 | 駆け出し冒険者でも十分に対処できる脅威。ゴブリン、ラージラットなど         |
|      2 | 一人前の冒険者が対処する脅威。オールドマンティス、ギガントトード、野盗など |
|      3 | 一人前の冒険者がなんとか対処する脅威。オーク、レッサードレイクなど         |
|      4 | 熟練の冒険者が対処する脅威。トロール、ラミアなど                           |
|      5 | 地域を脅かす脅威。オーガ、リッチ、地竜、ワイバーンなど                     |
|      6 | 都市規模の脅威。グリフォン、精霊種、ヴァンパイアなど                       |
|      7 | 国家規模の脅威。成竜、オーガジェネラル、上位悪魔など                       |
|      8 | 複数の国家にまたがる脅威。魔王の幹部など                                   |
|      9 | 世界を脅かす脅威。古竜、魔王の側近、クラーケン、異界の存在など             |
|     10 | 災厄そのもの。魔王、始祖の龍、ウォローメイル、エルダーウッド、虫など       |

## エネミーの操作

- GMがエネミーの操作権を持つ
- エネミーもPCと同様のルールで戦闘を行う
- エネミーは「使用スペル」に記載したスペルを優先して使用する
    - ルール上は能力値以内の難易度のスペルをすべて行使可能だが、基本的には使用しない
    - GMは判断で、記載のないスペルを行使させても良い
- PCがスペルやアイテムの効果によってエネミーを使役している場合、その操作はそのPLが行う
    - 「クリエイトゴーレム」「アニメイトデッド」などのスペルが該当する

### エネミーが取り得る戦術について

- GMは「エネミーが勝利すること」を目指すのではなく「冒険としてふさわしい戦闘を行うこと」を目指してプレイすること
- 例えば、ゲームのルール上では、1人のPCを集中攻撃する方がエネミーも有利なケースが多い
- 賢いエネミーにはそうした戦法を取らせたり、そうでないエネミーは場当たり的な行動を採用するなど、工夫すると良い

### 野生動物や本能ベースで動くエネミーの操作の例

- 目の前にいる敵を場当たり的に攻撃する
- 例えば、前衛にウルフパックが6匹いて、前衛にPCが3人いる場合、それぞれに2回攻撃する、という操作をすると良い
- 攻撃を受けた場合、怒り狂ってその攻撃をしてきた対象に優先して反撃する
- 一度防護によって攻撃を阻まれた場合、次の行動ではその相手を避けて別の標的に攻撃する
- 臆病なエネミーについては、手痛い攻撃を受けた相手を避けるように標的を選ぶ、という操作をしても良い

### 知性を持つエネミーの操作の例

- 弱っていそうな相手や、厄介そうな相手を意図的に狙う
- 例えば、ゴブリンシャーマンは、防護の高そうなPCに「エナジーショット」を行使する
- 強者との戦いを望むエネミーなら、逆に防護が高そうだったり、まだ元気そうな相手を狙わせても良い
- 知性の高いエネミー単体の操作の例である。指揮能力がなければ他のエネミーは無秩序に操作することで、差をつけると良い

### 指揮能力を持つエネミーの操作の例

- 配下を利用してPLと同じように集中攻撃をしたり、後衛を狙ったりする
- セッションのクライマックスでの戦闘や、人間の組織相手は、こうした操作をすると良い

### スペルの行使について

- エネミーもPCと同様に能力値以内のスペルをすべて行使できるが、GMは無闇に行使させてはならない
- 「ルール上で行使可能」なことと「実際のプレイで行使する」ことは別である
- 例えばゴブリンシャーマンは「スリープで野生動物を眠らせ、エナジーショットで狩る」という成功体験を得ている
    - エネミーが特定のスペルを行使するのは、エネミー自体にも背景があるからである
- GMはルールに従ってエネミーを操作するのではなく、そのエネミーをロールプレイするように操作すると良い
    - どのようにロールプレイするかについては、GMの裁量で決定して良い
