# 黒い泥

- 焦げた樹脂やすえたドブのようなひどい匂いを発する、黒くねばついた泥
- 触れたものを汚染し、それが生物であれば死に至らしめる。それは、汎用金属すらも腐食させる
- 周囲の生物の意識に反応する性質があり、近くを通った動物や冒険者に引き寄せられ、やがてそれを殺しつくす

## 行動表

| 行動   | 射程 | 威力 | 命中 | 備考                                                                                                                                        |
| :----- | ---: | ---: | ---: | :------------------------------------------------------------------------------------------------------------------------------------------ |
| 触れる |    0 |    5 |    0 | 対象の武器・盾・防具の防護を-1。この効果は累積する。この効果は修理判定では解除できず、街や都市でWLTを1支払うことで、1点分の減少を修理できる |

## 特性・能力

| 特性・能力 | 効果                                                                                                                                      |
| :--------- | :---------------------------------------------------------------------------------------------------------------------------------------- |
| 意思なき者 | 毒・病気・昏睡・魅了を無効化する                                                                                                          |
| 不定形     | 自身が受ける物理属性のダメージを-5                                                                                                        |
| 汚染物質   | この者に触れた武器の威力を-1。この効果は累積する。この効果は修理判定では解除できず、街や都市でWLTを1支払うことで、1点分の減少を修理できる |
| 悪臭       | 距離:3以内のすべての対象は、常にすべての判定に-1dのペナルティを受ける。この効果は嗅覚を持つ対象にのみ有効。この効果は重複しない           |
