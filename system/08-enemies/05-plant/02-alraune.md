# アルラウネ

- アルラウネは秩序の種族に属するエルトメンヒェンの一種であり、本来は文明社会の一員である
- ただし、文明圏から離れた森林や遺跡に自生する野生個体は、真なる言葉を持たず本能で行動する
- 甘い香りで獲物を誘い寄せ、養分として吸収する
- 秩序に属するアルラウネと遭遇した場合、通常は友好的な対話が可能

> 注: 
> - 秩序に属する知的なアルラウネを殺害した場合、それは秩序の種族に対する犯罪行為となる
> - 野生個体との区別には INT 判定が必要な場合がある
> - GMの判断で、PLに友好的接触を試みさせても良い
> - その性質こそが、捕食者たるアルラウネの罠でもあるのだから

## 特性・能力

| 特性・能力 | 効果                                                                                                  |
| :--------- | :---------------------------------------------------------------------------------------------------- |
| 甘い香り   | ラウンド開始時、近距離の全対象に強度2の魅了を付与する。魅了された対象は、アルラウネに向かって移動する |
| 火に弱い   | 自身が受ける火炎属性ダメージを+2点                                                                    |
| 不動       | 移動不可。この者は地に根を張っている                                                                  |
