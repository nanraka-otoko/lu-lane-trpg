# オーガ

- 知恵を持つ悪鬼の上位種
- 巨大な体躯と膂力に加え、魔法を操る知性を備える
- 単体でも危険だが、子鬼族やオークなどの下位魔族を使役して群れを率いることが多い
- オーガが指揮する魔族の集団は組織的に行動し、村落や隊商を襲撃する

## 特性・能力

| 特性・能力     | 効果                                                                          |
| :------------- | :---------------------------------------------------------------------------- |
| 統率           | 配下の魔族が戦闘に参加している場合、オーガのターンで配下1体へ追加行動を命じる |
| 魔法封じの呪具 | 1度だけ自身への魔法の効果とダメージのMtVを-2する                              |
| 暗視           | 暗闇を見通すことができる                                                      |
