# アラクネ

- 蜘蛛の下半身に、悪鬼の上半身を持つ魔族
- 罠の構築に長ける種族であり、地下やダンジョンの暗がりに巣を張り、獲物を待ち伏せる
- 蜘蛛の脚は6本あり、毛に覆われた身体はアラクネの防具でもある
- 2mを超える大きな体躯を持ち、口に鋏角を備え、複数の目を持つ。足を広げた開長は4mを超える
- その動きは静かだが素早く、獲物に近づいて捕食を行う
- 危険な魔族の一種とされ、ドワーフやダークエルフなどの地下に住まう種族にとっての大きな脅威となっている

> アラクネの生態:
> - 卵生であり、大量の卵を屍肉に植え付けて繁殖を行う
> - 人族は極上の卵鞘とされ、土蜘蛛族のトロフィーとして扱われる
> - 孵化した幼体は屍肉を食らい、トラップスパイダーと呼ばれるサイズにまで成長していく
> - トラップスパイダーは迷宮や遺跡に好んで棲息し、糸を用いた罠を仕掛けて獲物を捕らえ、捕食する
> - 成長と脱皮を繰り返した個体は、上体が人族のように変化していく
>     - この状態の個体が、アラクネと呼称されている

## 特性・能力

| 特性・能力 | 効果                                                                                                                        |
| :--------- | :-------------------------------------------------------------------------------------------------------------------------- |
| 多脚       | ぬかるんだ地面などでも足を取られず、そのペナルティを無視できる                                                              |
| 這う者     | 天井や壁を這って移動することができ、バリケードなどの移動制限を無視できる。滑らかな壁面や天井では無効 (GMの裁量によって判断) |
| 蜘蛛の身体 | 自身への病気・毒の状態異常の強度を-1                                                                                        |
| 暗視       | 暗闇を見通すことができる                                                                                                    |
| 複眼       | 感知判定のMtVを+1                                                                                                           |
| クモの巣   | アラクネの巣や領域で遭遇した場合、イニシアチブを+1                                                                          |
