# トラップスパイダー / タイニーアラクネ

- アラクネの幼体。上体が人型になっておらず、全身が蜘蛛の見た目をしている
- 体長は50cm～1m程度、開長さは2m程度
- 親個体のアラクネの指揮のもと、成長前の小さい個体が狭い隙間に入り込んで迷宮にトラップを張り巡らせる

## 行動表

| 行動 | 射程 | 威力 | 命中 | 備考                                                          |
| :--- | ---: | ---: | ---: | :------------------------------------------------------------ |
| 鋏角 |    1 |    3 |    0 | この攻撃でHPにダメージを与えた場合、対象に強度3の毒を付与する |

## 特性・能力

| 特性・能力 | 効果                                                                                                                        |
| :--------- | :-------------------------------------------------------------------------------------------------------------------------- |
| 多脚       | ぬかるんだ地面などでも足を取られず、そのペナルティを無視できる                                                              |
| 這う者     | 天井や壁を這って移動することができ、バリケードなどの移動制限を無視できる。滑らかな壁面や天井では無効 (GMの裁量によって判断) |
| 蜘蛛の身体 | 自身への病気・毒の状態異常の強度を-1                                                                                        |
| 暗視       | 暗闇を見通すことができる                                                                                                    |
| 複眼       | 感知判定のMtVを+1                                                                                                           |
