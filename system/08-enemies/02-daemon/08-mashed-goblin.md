# マッシュド・ゴブリン

- 菌糸に寄生されたゴブリン。身体の一部にはキノコの子実体が生えている
- 思考能力が著しく低下しているが、生物としての自我は残っている
- ある種のゾンビ化状態にあり、菌糸に栄養を提供するために所構わず近くにいる生物を襲い、その肉を喰らう
- 菌糸との共生により、生命力が向上している
- 肉を好む菌類が寄生しているとされる
