## 行動表

| 行動   | 射程 | 威力 | 命中 | 備考                                                                                                                 |
| :----- | ---: | ---: | ---: | :------------------------------------------------------------------------------------------------------------------- |
| 溶解液 |    2 |    2 |   -1 | 対象の回避MtVが1以下の場合、対象の防護-1。減少した防護は、修理判定で修復可能                                         |
| 捕食   |    0 |    3 |    0 | 対象はHPを直接失う。この攻撃で対象を絶命させたとき、スライムのHPが3点回復し、成長ロールを1回行いステータスが上昇する |

## 特性・能力

| 特性・能力 | 効果                                                                                                     |
| :--------- | :------------------------------------------------------------------------------------------------------- |
| 不定形     | 自身が受ける物理属性のダメージを-5                                                                       |
| 炎に弱い   | 自身が受ける火炎属性のダメージを+2                                                                       |
| 本能       | 昏睡と魅了を無効化する                                                                                   |
| 変性       | 自身に毒を付与されたとき、生命力抵抗を行わない。即座に毒を解除し、自身の攻撃にその毒を付与する効果を得る |
