# ポイズンスライム

- 毒性を取り込んで成長したスライム
- 溶解液には毒が含まれており、非常に危険

## 行動表

| 行動   | 射程 | 威力 | 命中 | 備考                                                                                                                         |
| :----- | ---: | ---: | ---: | :--------------------------------------------------------------------------------------------------------------------------- |
| 腐食液 |    2 |    3 |    0 | 対象の回避MtVが2以下の場合、対象の防護を-1し、強度3の毒を付与する。減少した防護は、修理判定で修復可能                        |
| 捕食   |    0 |    5 |    0 | 対象はHPを直接失う。この攻撃で対象を絶命させたとき、ポイズンスライムのHPが3点回復し、成長ロールを1回行いステータスが上昇する |

## 特性・能力

| 特性・能力 | 効果                                                                       |
| :--------- | :------------------------------------------------------------------------- |
| 不定形     | 自身が受ける物理属性のダメージを-5                                         |
| 炎に弱い   | 自身が受ける火炎属性のダメージを+2                                         |
| 本能       | 昏睡と魅了を無効化する                                                     |
| 変性体     | 自身に毒を付与されたとき、即座に毒を解除し、その強度の点数だけHPを回復する |
