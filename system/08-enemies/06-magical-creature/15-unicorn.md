# ユニコーン

- 額に鋭い一本角を生やした馬
- 清らかな水辺に好んで棲む
- 太古の時代からエルフと関係があるとされ、盟約によってその力を森人に貸し出す

> 召喚:
> - エルフの魔法「一角獣の招来」で召喚・使役が可能
> - 召喚した個体に対しては戦利品判定はできない

## 特性・能力

| 特性・能力   | 効果                                                                                           |
| :----------- | :--------------------------------------------------------------------------------------------- |
| 暗視         | 暗闇を見通すことができる                                                                       |
| 水棲         | 水中でも呼吸ができ、地上と変わらずに先を見通すことができる。水中での活動にペナルティを受けない |
| 幻獣の血     | 魔法による自身へのダメージを-1。自身へのすべての状態異常の強度を-1                             |
| 清流の守護者 | 森林・清らかな水辺では、自身のすべての判定に+1d                                                |
