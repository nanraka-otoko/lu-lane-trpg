# ミミック

- 宝箱に擬態する大型のミミック
- 冒険者の貪欲さを利用して襲いかかる

## 行動表

| 行動     | 射程 | 威力 | 命中 | 備考                                                                                                             |
| :------- | ---: | ---: | ---: | :--------------------------------------------------------------------------------------------------------------- |
| 噛みつき |    1 |    5 |    0 | 命中時、対象を拘束し、行動不能とする。対象は自身の行動を消費し、目標値3のSTR判定に成功することで拘束を解除できる |
| 咀嚼     |    0 |    5 |    0 | 拘束中の対象を咀嚼する。この攻撃は回避できない                                                                   |

## 特性・能力

| 特性・能力 | 効果                                                                                              |
| :--------- | :------------------------------------------------------------------------------------------------ |
| 擬態       | 宝箱を開けようとした者に対して奇襲する。目標値3の知識判定か感知判定に成功すると見破ることができる |
