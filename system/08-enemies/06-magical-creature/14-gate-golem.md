# ゲートゴーレム

- 要塞など門に据え付けられたゴーレム
- 炎を噴出する魔法が付与されており、門に攻め入る相手を丸焦げにする
- 非常に固く、正面から消耗戦を挑むと苦戦は免れない

## 行動表

| 行動     | 射程 | 威力 | 命中 | 備考                                                                                                                                       |
| :------- | ---: | ---: | ---: | :----------------------------------------------------------------------------------------------------------------------------------------- |
| 火炎放射 |    2 |    4 |    0 | エリアを対象に攻撃する。回避・防護不可。火炎属性のダメージを与える。使用後、2ラウンド使用不可となる                                        |
| 火炎砲弾 |    3 |    5 |    0 | 3体までを対象に同時攻撃可。防護不可。火炎属性のダメージを与える。この攻撃に対する回避判定のMtVは3倍となる。使用後、2ラウンド使用不可となる |
| 熱波放出 |    1 |    0 |    0 | エリアを対象に攻撃する。防護不可。火炎属性のダメージを与える。使用後、2ラウンド使用不可となる                                              |

## 特性・能力

| 特性・能力   | 効果                                                                                 |
| :----------- | :----------------------------------------------------------------------------------- |
| 石の身体     | 自身への雷属性のダメージを+3。毒・病気・昏睡・魅了の状態異常を無効化                 |
| 熱量         | 自身への凍結の状態異常の強度を-2                                                     |
| 自己修復     | このエネミーのHPは1日が経過すると3点回復する。他の手段ではHPを回復することができない |
| 巨大な建造物 | このエネミーを包囲することはできず、このエネミーは包囲に参加しない                   |
| 不動         | 移動不可。このエネミーに対するイニシアチブ判定は自動成功する                         |
