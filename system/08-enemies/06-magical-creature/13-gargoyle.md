# ガーゴイル

- 動く石像。背に翼を持ち、短時間の飛行を行う
- ゴーレムの一種であり自発的に敵対はしないが、侵入者の排除が命令されていることが多く、冒険者の敵となることが多い

## 特性・能力

| 特性・能力 | 効果                                                                 |
| :--------- | :------------------------------------------------------------------- |
| 飛行       | 石の翼で空を飛ぶ。連続して10分までの飛行が可能                       |
| 石の身体   | 自身への雷属性のダメージを+3。毒・病気・昏睡・魅了の状態異常を無効化 |
