# カチャ・ル・ガトル

- 6本の脚を持つ爬虫類のような悪魔
- 体長は50cmほど。機敏に動き周り、壁や天井に張り付いて移動する
- 強力な胃酸を持ち、獲物の皮膚や武具を溶かす

## 行動表

| 行動 | 射程 | 威力 | 命中 | 備考                                          |
| :--- | ---: | ---: | ---: | :-------------------------------------------- |
| 舌   |    1 |    3 |    0 | 対象の回避判定の累積ペナルティに-1dを追加する |
| 酸液 |    2 |    1 |    0 | 対象の防護を-1。効果は4回まで重複する         |

## 特性・能力

| 特性・能力 | 効果                                                                     |
| :--------- | :----------------------------------------------------------------------- |
| 潜兵       | 敵対陣営の包囲の影響を受けず、自陣営の包囲に参加しない                   |
| 暗視       | 暗闇を見通すことができる                                                 |
| 多脚       | ぬかるんだ地面などでも足を取られず、そのペナルティを無視できる           |
| 這う者     | 天井や壁を這って移動することができ、バリケードなどの移動制限を無視できる |
