# イビル・ポール

- 悪魔神の意匠が刻まれた積石の柱
- 1～3m程度の様々なサイズのものが造られている
- 秩序に属するものを感知し、排除を行う
- 下級の魔法に加え、悪魔神の奇跡を使用する
- 他の悪魔たちはイビル・ポールを要塞の守りとして設置し、戦いを有利に進めようと画策する

## 特性・能力

| 特性・能力 | 効果                                                                                                                       |
| :--------- | :------------------------------------------------------------------------------------------------------------------------- |
| 不動       | 移動不可。このエネミーに対するイニシアチブは自動勝利する                                                                   |
| 構造体     | すべての状態異常を無効化する                                                                                               |
| 侵入検知   | このエネミーを奇襲することはできない                                                                                       |
| 魔力の補充 | 4M6をロールし、自身のMPをMtV点回復する。1日の間、発動するごとにダイスの個数が1個減少する。この効果の発動には行動を消費する |
| 自己修復   | このエネミーのHPは1日が経過すると1点回復する。他の手段ではHPを回復することができない                                       |
