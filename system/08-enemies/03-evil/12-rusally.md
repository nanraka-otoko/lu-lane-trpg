# ルサリィ

- ヒュームやエルフに似た外見を持つ、異色の肌を持つ悪魔
- ルサリィに性別の概念はないが、人間の女性や男性の姿をしていることが多い
- その容姿は異性にとって魅力的であり、他者を魅了する
- 変化の能力を持ち、姿を様々に変え、秩序文明に混乱をもたらす
- 変化は幻影ではなく、物理的な変形を伴う。ディスペルなどで見破ることはできない
- 身体の一部分を変形して武器にするといったことも可能であり、高い戦闘能力を持つ

## 特性・能力

| 特性・能力 | 効果                                                                                                                         |
| :--------- | :--------------------------------------------------------------------------------------------------------------------------- |
| 魅了       | 遭遇時、全対象に強度3の魅了の状態異常を付与する。異性の対象には強度4の魅了となる。魅了された対象の行動はルサリィに支配される |
| 変化       | 身体を自在に変化する。目標値4の知識判定か対人判定に成功すると見破ることができる                                              |
| 暗視       | 暗闇を見通すことができる                                                                                                     |
