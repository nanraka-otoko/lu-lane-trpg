# マッドドレイク

- 沼地に住まう地竜
- 幼体の時点で生命力が強く、下水などを通じて全く異なる地域に移動することがある

## 行動表

| 行動         | 射程 | 威力 | 命中 | 備考                                                                                                                                                 |
| :----------- | ---: | ---: | ---: | :--------------------------------------------------------------------------------------------------------------------------------------------------- |
| 腕払い       |    1 |    2 |    0 | 予備行動可                                                                                                                                           |
| 突進         |    1 |    3 |    0 | 移動したターン、威力を+2                                                                                                                             |
| 尾           |    1 |    2 |    0 | 3体までを対象に同時攻撃可                                                                                                                            |
| マッドブレス |    2 |    3 |   -1 | 3体までを対象に同時攻撃可。この攻撃により装甲が減少した防具は、価値を1点失う。1時間以内に洗浄すれば価値を取り戻せる。使用後、2ラウンド使用不可となる |

## 特性・能力

| 特性・能力     | 効果                                           |
| :------------- | :--------------------------------------------- |
| 沼地に住まう者 | ぬかるんだ地面の影響を受けない                 |
| 不浄耐性       | 自身への病気・毒の状態異常の強度を-2           |
| 有利地形       | ぬかるんだ地面で遭遇した場合、イニシアチブを+1 |
| 炎に強い       | 自身への炎属性ダメージを-1                     |
| 雷に強い       | 自身への雷属性ダメージを-2                     |
