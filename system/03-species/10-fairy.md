# フェアリー / 蝶人 / 虫人 / 妖精

## ステータス

| ステータス | ロール式 |
| :--------- | :------- |
| CON        | 1 + 2M6  |
| MND        | 1 + 8M6  |
| STR        | -        |
| DEX        | 1 + 8M6  |
| INT        | 1 + 8M6  |
| FTH        | 1 + 4M6  |
| LUC        | 1 + 4M6  |
| APP        | 1 + 6M6  |

## 特性

| 特性           | 効果                                                                                  |
| :------------- | :------------------------------------------------------------------------------------ |
| 蝶の羽         | 飛行することができる。狭い空間での飛行おける汎用行動判定にペナルティを受けない        |
| 小さな身体     | 自身の回避判定に+4d                                                                   |
| ふしぎなどうぐ | 装備を自身のサイズにフィッティングする。重量1以下の装備品を、重量を無視して装備できる |
| 非力           | STRが増減しない。STRを用いる判定をする場合、STRを0として扱う                          |
| 自然の知識     | 植物、虫、菌類への知識判定に+2d                                                       |
| 魔法の鱗粉     | 1セッションに1回、任意のタイミングで触媒「魔法の触媒」を獲得できる                    |

## フェアリーの魔法

- フェアリーの血を持つ者のみが行使できる魔法
- INT で行使判定する

| 魔法名                     | 難易度 | コスト | 射程   | 属性 | 効果                                                                                                                                     |
| :------------------------- | -----: | -----: | :----- | :--- | :--------------------------------------------------------------------------------------------------------------------------------------- |
| フェアリー・テイル         |      1 |      1 | 自身   | 魔力 | 妖精郷への入口を把握する                                                                                                                 |
| ウィル・オー・ザ・ウィスプ |      2 |      1 | 特殊   | 炎   | 小さな炎を従える。炎はたいまつと同じ光源となるが、攻撃手段としては利用できない。効果は1日間持続する。屋外で行使する場合、消費MPを0にする |
| タイニー・シンポジウム     |      3 |      1 | 特殊   | 魔力 | 小さな動物や植物と会話できる。GMは状況に応じた情報をPLに与える                                                                           |
| パワー・オブ・フェローズ   |      4 |      1 | 接触   | 魔力 | 予備行動可。対象は自身と同じエリアにいる味方の数だけ、自身の攻撃判定と回避判定に+1dのボーナスを得る。効果は3ラウンド持続                 |
| バッド・トリップ           |      5 |      1 | 接触   | 魔力 | 対象はMPをMtV点回復し、すべての行動判定に-1dのペナルティを受ける。ペナルティは1時間持続する。1日の間、行使するごとに消費MPが+1される     |
| フェアリー・リング         |      6 |      2 | 長距離 | 魔力 | すべてのフェアリーは、自身と同じエリアにいるフェアリーの数だけ、自身の攻撃判定と回避判定に+1dのボーナスを得る。効果は3ラウンド持続       |

## 種族: フェアリー

- 蝶の羽を持つ小人。蝶人、妖精とも呼ばれる
- 30cm～50cm程度の小さな身体と飛行能力を持ち、魔法の扱いに長ける
- 蝶などの虫に近い存在とも、自然の精霊に近い存在ともされ、その起源は曖昧
- 気まぐれで悪戯好きな気性を持ち、文明社会に定住することは稀であるが、秩序文明には友好的な種族である
- その身体は極めて脆弱であり、まともな筋力を持たない
