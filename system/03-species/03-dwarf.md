# ドワーフ / 鉱人

- 頑健な寸胴体型の身体を持つ、地下や山岳に住まう種族
- 秩序の秘宝を発見した五種族のひとつ
- 頑固で職人気質な気性で知られる
- 鍛冶と採掘の技術に秀で、優れた武具や建造物を生み出す。鍛冶神を信仰する者が多い
- 酒と歌が好きな種族でもあり、舞歌神も強く信仰されている
- 生命力と筋力、技量、知力に長ける

## 特性

| 特性       | 効果                                                    |
| :--------- | :------------------------------------------------------ |
| 暗視       | 暗闇を見通すことができる                                |
| 火炎耐性   | 自身への火炎属性のダメージを-1                          |
| 酒と歌     | 1日に1回まで、酒を飲むことでMPを1点回復することができる |
| 鉱人の知識 | 石や鉱石、鉱山や坑道に対する知識判定に+2d               |

| トンネル         |      5 |      1 | 接触   | 土      | 土・石・レンガの壁に穴を開ける。木材・鉄・魔法・その他の壁には無効。MtVメートル四方までのサイズの穴を開けられる                            |
