# マーフォーク / 人魚 / 魚人

- 魚の身体を持つ水棲種。海や深い湖に暮らし、水中での生活に適応している
- 下半身を人の脚に擬態することで陸上での活動も可能だが、得意とはしない
- 交易港などでは水棲種の代表として他種族と関わる
- 容姿に優れ、身のこなしにも長ける

> マーフォークの生態:
> - 水深～300m程度までの、大陸棚とその端部に集落を築いている
> - 基本的には沿岸部からほど近い水深100m程度の深さに生息する
> - 大陸棚は沿岸から20～80km程度の距離であり、近海や沖合に少し出た程度の距離である
> - マーフォークは、遠洋でも活動可能だが、住居を必要とするため、縄張りとはしていない
> - 水深500m程度までは適応可能だが、それ以上の深海には種族としては適応していない
> - リーフエッジからさらに外洋側の沖合に向けて集落を形成することが多く、水深20～80m程度の集落が多い
> - 近年では沿岸部の他種族の街と隣接するように、海底都市を形成する例もある
>
> 秩序文明に属する貝人や軟魚人について:
> - モラスカンと呼ばれる種族。貝人や軟魚人とも呼称される
> - 脊椎を持たないこの種族は、陸上には適応せず、水中にのみ生息している

## 特性

| 特性     | 効果                                                                  |
| :------- | :-------------------------------------------------------------------- |
| 水棲     | 水中で呼吸ができる。水中での活動にペナルティを受けない                |
| 陸上擬態 | 下半身を人の脚に擬態し、陸上を歩行する。陸上での活動時、一時的にDEX-1 |
| 水に強い | 自身への水属性のダメージを-2                                          |
| 雷に弱い | 自身への雷属性のダメージを+1                                          |
