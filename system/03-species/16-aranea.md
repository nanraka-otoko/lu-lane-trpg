## 特性

| 特性       | 効果                                                                                                                        |
| :--------- | :-------------------------------------------------------------------------------------------------------------------------- |
| 多脚       | ぬかるんだ地面などでも足を取られず、そのペナルティを無視できる                                                              |
| 這う者     | 天井や壁を這って移動することができ、バリケードなどの移動制限を無視できる。滑らかな壁面や天井では無効 (GMの裁量によって判断) |
| 跳躍力     | 汎用行動判定に+2d                                                                                                           |
| 蜘蛛の身体 | 自身への毒・病気の状態異常の強度を-1                                                                                        |
| 蜘蛛の糸   | HPを1点消費することで、10mまでの蜘蛛の糸を生成できる。蜘蛛の糸は、ロープと同様の道具として使用可能                          |

## 種族: アラネア

- 上半身がヒト、下半身が蜘蛛の亜人種。蜘蛛の脚を8本、人間の上肢を2本持つ
- 古くはアラクネと混同され、他種族から忌避されており、神への信仰心が薄い
- 壁や天井を這って移動することが可能であり、その身のこなしは素早く、運にも恵まれている

### 文明との関係

- 魔族のアラクネと違い、秩序文明の一員として扱われている
- アラクネよりも一回り小さく、脚が8本であり、角や牙を持たないなど、外見からの区別が可能
- 自身の糸を利用した織物の製造などを行っており、文明社会において重宝される
- 蜘蛛であるが鋏角は持たず、食性は雑食でありヒュームに似る
