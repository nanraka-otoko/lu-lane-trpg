# バードフォーク / 有翼人 / 鳥人

- 鳥の形質を持った亜人。鳥人とも呼ばれる、鳥の翼と脚を持つ飛行種族
- 山岳や断崖の上に集落を築き、空を自在に飛び回る
- 猛禽の如き獰猛さを持つ個体もおり、獣人と同様に氏族単位で特徴が大きく異なる種族でもある
- 短い寿命、脆弱な生命と引き換えに、高い技量と知性、飛行能力を持ち、膂力にも長ける

## 特性

| 特性   | 効果                                              |
| :----- | :------------------------------------------------ |
| 鳥の翼 | 前腕に翼を持ち、飛行することができる              |
| 鳥の耳 | 音に対する判定のMtVに+1                           |
| 鳥の力 | キャラクターメイク時に、鳥の力を2つ選び、獲得する |

鳥の力の一覧

| 特性       | 効果                                                                                                            |
| :--------- | :-------------------------------------------------------------------------------------------------------------- |
| 猛禽の膂力 | STR +1                                                                                                          |
| 猛禽の爪   | 鳥の爪の威力を+1                                                                                                |
| 猛禽の本能 | DEXを用いてイニシアチブ判定を行うことができる                                                                   |
| 小鳥の愛嬌 | APP +1                                                                                                          |
| 鳴禽の声   | 声を用いた対人判定のMtVを+1                                                                                     |
| 走禽の脚   | DEX +1                                                                                                          |
| 企鵝の身体 | 水中でも1時間まで無呼吸で活動ができ、地上と変わらずに先を見通すことができる。水中での活動にペナルティを受けない |

> 鳥の力:
> - タカやワシなどの猛禽類や、小鳥、ガチョウなどの走禽、ペンギンなどの企鵝の力を得られる
> - ダチョウ型の鳥人や企鵝 (ペンギン型) の鳥人であっても、飛行能力は失われない

## バードフォークの魔法

- バードフォークの血を持つ者のみが行使できる魔法
- INT で行使判定する

| 魔法名           | 難易度 | コスト | 射程   | 属性 | 基礎効果                                                                                                          |
| :--------------- | -----: | -----: | :----- | :--- | :---------------------------------------------------------------------------------------------------------------- |
| ネイルエッジ     |      2 |      1 | 自身   | 魔力 | 予備行動可。「鳥の爪」の命中を+2。効果は3ラウンド持続                                                             |
| バードストライク |      3 |      1 | 自身   | 魔力 | 予備行動可。「くちばし」の威力を+4、移動したターンでの射程を+1。効果は1ラウンド持続                               |

| フェザーショット |      4 |      1 | 遠距離 | 物理 | 対象を3体まで選び1+MtVダメージ。このダメージは防護可能・回避不可。MPの代わりにHPをコストとして行使できる          |
| フェザーソード   |      4 |      1 | 自身   | 魔力 | 予備行動可。自身の片腕が武器「フェザーソード」に変化する。フェザーソードは3ラウンド持続する。飛行能力は失わない   |
| フェザーシールド |      4 |      1 | 自身   | 魔力 | 予備行動可。自身の片腕が盾「フェザーシールド」に変化する。フェザーシールドは3ラウンド持続する。飛行能力は失わない |
| ホバーフィールド |      6 |      1 | 近距離 | 風   | 距離:1以内の任意の味方を浮遊させる。効果は3ラウンド持続                                                           |
