# ビースト / 獣人

- 獣人とも呼ばれる、動物の特徴を持つ種族たちの総称
- 秩序の秘宝を発見した五種族のひとつ
- 混沌の時代に多数の種族が部族として統合・混合された経緯を持つ
- 狼、猫、熊、鹿、猪など、様々な動物の特徴を持つ亜種が存在する 
    - 交雑種が多く、個別の名称を持つ種族は稀
    - 同一の血統で氏族を形成することが多い
    - 哺乳類の形質を持つが、鳥類や魚類の形質は持っていない
- かつては広大な平原を支配していたとされ、狩猟神を信仰する氏族が多い
- 総じて生命力と筋力に長け、獣の性質に応じた能力を得る

> ウルフビースト:
> - 獣人種の一派で、狼の特徴を持つ種族。秩序の秘宝を発見した五種族のひとつ
> - 優れた嗅覚と俊敏さを持ち、群れでの行動に長けている
> - 獣人族の中では特に集団行動が得意であり、獣人として最初に統一文明に馴染んだ要因ともなっている
> - 文明の中では特別な獣人種であり、獣人の代議員はウルフビーストの者が多い
>
> 獣人の歴史:
> - 古の時代の獣人は、獣の亜人として多様に存在していたが、他種族からは獣人として一括りにされ、敵対していた
> - 獣人たちは、哺乳類の亜人として集落を築くようになり、血の交雑が進んだ
> - ウルフビーストのように血統を重視する氏族もいるが、それは統一の時代においてその形質を持っていただけであり、彼らもまた交雑の末裔である
> - これらの歴史はルゥレーンでは失われており、一般常識ではない

## 特性

| 特性     | 効果                                                                                                  |
| :------- | :---------------------------------------------------------------------------------------------------- |
| 獣の本能 | DEXを用いてイニシアチブ判定を行うことができる                                                         |
| 獣の身体 | キャラクターメイク時に、形質を遺伝した獣を指定する。その獣の形質に応じ、好きな能力値を2つ選んで+1する |
| 獣の力   | キャラクターメイク時に、獣の力を2つ選び、獲得する                                                     |

獣の力の一覧

| 特性     | 効果                                    |
| :------- | :-------------------------------------- |
| 獣の瞳   | 暗闇を見通すことができる                |
| 獣の鼻   | 匂いに対する判定のMtVを+1               |
| 獣の耳   | 音に対する判定のMtVを+1                 |
| 獣の触覚 | 触覚による罠や異常の感知の判定のMtVを+1 |
| 鋭い爪   | 獣の爪の威力を+2                        |
| 鋭い牙   | 獣の牙の威力を+2                        |
| 第六感   | LUC +1                                  |
| 愛嬌     | APP +1                                  |

> 獣の身体:
> - 指定できる種族は哺乳類に限る
> - 虎や猫、熊や犬、狼などは良いが、猛禽などの鳥類や、魚類などはNG
> - その獣の耳、腕、脚、尻尾などの外見的形質を1つ以上持って生まれる
