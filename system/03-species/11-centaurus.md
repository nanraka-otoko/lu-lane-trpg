# ケンタウロス / 人馬 / 馬人

## ステータス

| ステータス | ロール式 |
| :--------- | :------- |
| CON        | 1 + 6M6  |
| MND        | 1 + 4M6  |
| STR        | 1 + 9M6  |
| DEX        | 1 + 6M6  |
| INT        | 1 + 4M6  |
| FTH        | 1 + 5M6  |
| LUC        | 1 + 3M6  |
| APP        | 1 + 4M6  |

## 特性

| 特性       | 効果                                                                                             |
| :--------- | :----------------------------------------------------------------------------------------------- |
| 大きな身体 | 自身の回避判定に-2d。包囲において、2人分として扱う                                               |
| 突撃       | 移動したターンでの自身の近接攻撃の威力を+2                                                       |
| 力持ち     | 所持品を5個、追加で携行できる。追加の携行品は装備できない                                        |
| 騎乗可能   | 重量:3まで、自身の背に他者を背負うことができる。重量が超過する場合、自身の装備重量として加算する |
| 靴装備不可 | カテゴリ:靴のアクセサリを装備できない                                                            |

> 大きな身体:
> - 垂直方向の身長だけでも2m以上ある個体も多い
> - 狭い空間での移動や行動を苦手とするが、ゲームの進行上ではそのサイズを過度に考慮しなくても良い
> - 例えば、ダンジョン内はケンタウロスでも十分に活動できるほどの空間があるとしても良い

## 種族: ケンタウロス

- 半人半馬の姿を持つ種族。草原や平原を駆ける遊牧の民
- 獣人種の近縁とされることもあるが、本人たちはその分類を嫌う
- 馬車の動力として労働している者も多い。ケンタウロスの力車は馬車と比較して乗り心地が良く、操者も不要のため、多種族からの人気が高い
- 生命力や筋力に長け、身軽さも併せ持つが、身体が大きい者がほとんど
