# サーペンテイル / 蛇人

- 上半身がヒトの姿、下半身が蛇の姿を持つ亜人種。長命種
- 湿地帯や地下、樹上など幅広い地域で暮らしているが、寒冷地を避ける傾向にある
- リザードマンと同様、竜を祖先に持つとされる種族であり、竜の奇跡を行使できる
- 強い信仰心を持ち、自然との調和を重んじる
- 姿の似た魔族のラミアとは異なり、サーペンテイルはその牙に強い毒を持つ

> ラミアとの関係性:
> - 鬼蛇と違い、鱗が地味であったり、瞳も普通であったりと、過度な美貌を持たない
> - ラミアは神を信仰していないが、サーペンテイルは古代より神を強く信仰している
> - その容姿から、かつては鬼蛇の一種とされ他種族から忌み嫌われていたが、秩序文明の興りと共に文明に受け入れられていった
> - サーペンテイルはラミアを蛇の薄汚い末裔として憎悪しており、その討滅は種族としての誇りでもある
>
> 暮らし:
> - 雑食であるが、鳥の卵を好んで食べる
> - 野生の蛇と違い、鳥類を捕食せず、家畜として飼うことでその卵を得ている

| 特性         | 効果                                                            |
| :----------- | :-------------------------------------------------------------- |
| 湿地帯の住人 | ぬかるんだ地面や浅瀬で足を取られず、そのペナルティを無視できる  |
| 蛇の身体     | 自身への毒の状態異常の強度を-2。木を登る場合の汎用行動判定に+4d |
| 蛇の誇り     | 自身への魅了の状態異常の強度を-1。ラミアからの魅了を無効化する  |
| 暗視         | 暗闇を見通すことができる                                        |
| 寒さに弱い   | 自身への冷気属性のダメージを+1                                  |
| 靴装備不可   | カテゴリ:靴のアクセサリを装備できない                           |
