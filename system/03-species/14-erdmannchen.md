# エルトメンヒェン / 花人 / 草人

## ステータス

| ステータス | ロール式 |
| :--------- | :------- |
| CON        | 1 + 6M6  |
| MND        | 1 + 4M6  |
| STR        | 1 + 3M6  |
| DEX        | 1 + 3M6  |
| INT        | 1 + 6M6  |
| FTH        | 1 + 5M6  |
| LUC        | 1 + 6M6  |
| APP        | 1 + 9M6  |

## 特性

| 特性         | 効果                                                                                                                           |
| :----------- | :----------------------------------------------------------------------------------------------------------------------------- |
| 巡りゆく生命 | 死亡したとき、LUCを恒久的に1失い、HP1で復活する。HPが0点以下の時、任意にこの効果を使用できる。この効果でLUCが0となると死亡する |
| 擬態         | 背の高い植物に擬態することができる。DEX/INT/LUC/APPで隠密判定をし、そのMtVを+1。看破には感知判定での対抗が必要                 |
| 植物との対話 | 植物と対話できる。一般の植物は大した知性を持たない                                                                             |

| 植物の世話人 | MND +1, DEX +1, INT +1, WLT +3 | 植物を管理する仕事。農村で活躍する者もいる                               |

## エルトメンヒェンの魔法

- エルトメンヒェンの血を持つ者のみが行使できる魔法
- INT または APP で行使判定する

| 魔法名           | 難易度 | コスト | 射程   | 属性 | 効果                                                                                                                                                                                                                      |
| :--------------- | -----: | -----: | :----- | :--- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| リポット         |      1 |      0 | 自身   | 土   | 予備行動可。HPをMtV点回復する。1日の間、行使するごとに消費MPが+1される                                                                                                                                                    |
| フレグランス     |      2 |      1 | 遠距離 | 風   | 対象に強度1+MtV点の魅了の状態異常を付与する。嗅覚のある対象にのみ有効。魅了された者は、術者に敵対行動を取れない                                                                                                           |
| リラクゼーション |      3 |      1 | 遠距離 | 魔法 | 対象に強度1+MtV点の昏睡の状態異常を付与する。嗅覚のある対象にのみ有効                                                                                                                                                     |
| アンサーダスト   |      4 |      1 | 遠距離 | 風   | エリアを対象に行使する。対象のエリアを花粉で満たす。エリアにいる者のすべての判定に-1dのペナルティを与える。効果は3ラウンド持続                                                                                            |
| インバッズ       |      5 |      1 | 自身   | 魔法 | 予備行動可。蕾の盾に籠り、3+MtV点の防護を得る。次の自身のターンで効果は解除される                                                                                                                                         |
| アイビーウォール |      5 |      2 | 長距離 | 土   | エリアを対象に行使する。エリアに「蔦のバリケード」(expantion/field.md) を生成する。3ラウンド持続                                                                                                                          |
| コンフェット     |      6 |      1 | 長距離 | 風   | 周囲に花吹雪を起こす。友好的な対人判定は+4+MtVdのボーナスを得る                                                                                                                                                           |
| ソーンメイデン   |      7 |      3 | 長距離 | 土   | 対象を拘束し、行動不能とする。対象は自身の手番開始時に目標値3+MtVの筋力判定を行い、成功すると拘束を解除できる。筋力判定に失敗した場合、回避・防護不可で3点のダメージを受け、次の筋力判定の目標値を-1。効果は3ラウンド持続 |

> APPでの魔法行使判定が可能:
> - エルトメンヒェンの魔法は、INTとAPPの好きな能力値を使用して行使できる
> - INT3、APP5 の場合、APPの方が高いので、APPで魔法行使判定をしても良い
> - 難易度についても、高い方の能力値で参照して良い
> 
> 花人の言葉:
> - 花人が持つ混沌の言葉とは、音だけでなくその香りや容姿を含む、本人の魅力そのものを指す

## 種族: エルトメンヒェン

- 草花の形質を持った亜人。花人、草人とも呼ばれる
- 樹人と同様、自らを生んだとされる豊穣神を信仰することが多い
- 生命力と運に優れ、他種族を魅了する外見や匂いを有する
- 花人の魔法は、その魅力によっても効果を高める
