# ダークエルフ / 黒森人 / 洞森人

- エルフの同族でありながら、地下世界を生活圏とする種族
- 外見は森人と同様に美しく、黒色の肌を持つ
- 戦と共にあった種族であり、戦武神を信仰する者が多い
- 地上のエルフと比して膂力に優れるが、水への適応力が弱まっており、精霊の魔法を失っている
- 闇の魔法に通じ、夜目が利くが、闇に近づいたダークエルフたちは天の運に恵まれない

> ダークエルフの文化:
> - 統一文明には属しているが、独立心が強く、独自の社会を維持している
> - ドワーフと同様に金属の扱いに長けるが、ダークエルフは魔力を宿した鉱物の扱いを得意とする
> - 地上のエルフとは混沌の時代に袂を分かったとされ、両者の関係は複雑

## 特性

| 特性       | 効果                                                                                                            |
| :--------- | :-------------------------------------------------------------------------------------------------------------- |
| 暗視       | 暗闇を見通すことができる                                                                                        |
| 水中適応   | 水中でも1時間まで無呼吸で活動ができ、地上と変わらずに先を見通すことができる。水中での活動にペナルティを受けない |
| 雷耐性     | 自身への雷属性のダメージを-1                                                                                    |
| 地底の知識 | 魔石や魔鉄鉱、鉱山や坑道に対する知識判定に+2d                                                                   |
| 悪魔の知識 | 悪魔に対する知識判定に+2d、悪魔のエネミーに対する警戒指示の判定に+4d                                            |
| 闇の者     | 禁止魔法の行使によるAPP減少を、3種類のスペルまで無効にできる。「古代種」の才能を持つ場合、禁止魔法の難易度を-4  |
