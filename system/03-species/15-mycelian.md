## 特性

| 特性     | 効果                                                                                                            |
| :------- | :-------------------------------------------------------------------------------------------------------------- |
| 群体     | 通常の手段では死亡しない。敗北時の運判定のMtVに+2。火炎属性のダメージによりHPが-(CON+LUC)点以下になると死亡する |
| 炎に弱い | 自身への火炎属性のダメージを+1                                                                                  |
| 菌の身体 | 自身への毒・病気の状態異常の強度を-3                                                                            |
| 不眠     | 睡眠を必要とせずに活動できる。自身への昏睡の状態異常の強度を-3                                                  |
| 分解者   | 土や木、生物死骸から栄養を得る。食事を必要とせずに活動できる                                                    |

## 種族: マイセリアン

- キノコの形質を持った亜人。茸人とも呼ばれる、知性を持つキノコの種族
- 普段は土や樹木に埋まった状態で生活するが、二足歩行も可能である
- 生物としてはほとんどキノコであり、食事などもしない
- 生命力に秀でた種族であり、状態異常に対する耐性が非常に高いが、火に脆弱
- 通常の手段では死亡することがないが、炎に焼かれることで死に至る

### マイセリアンの生態

- マイセリアンは自己を複製する形で繁殖を行うため、一つの種から生まれる個体はすべてクローンである
- 自我は個体ごとに異なっているため、他者との交わりで異なる性格に育つことがある

### 文明との関わり

- ゴミや下水の処理と関わりがある種族であり、あまり表には出てこないが、文明圏の多くの場所で生活している
- 種族固有の国家を持たないが、世界の各地にマイセリアンが多く棲まう森が点在する
- 文明と関わりの薄いマイセリアンは、旅人を見守る程度にしか他種族に干渉せず、土や木に身体を埋めて静かに暮らしている
