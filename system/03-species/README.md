
# 種族

- このドキュメントでは、ルゥレーンに生息する様々な種族のデータを扱います
- 種族ごとのステータス、特性、背景、魔法・奇跡を参照する場合は、それぞれの種族のドキュメントを参照してください

---

## 秩序文明の種族 / 人族

- PCとして選択可能な種族
- 秩序の統一文明に属している

### [ヒューム / 人間 / 只人](./01-hume.md)

- 最も数が多く、秩序文明の中核を担う種族
- 秩序の秘宝を発見した五種族のひとつ
- 能力的には特徴がないが、人の本質的な力、運に長ける

### [エルフ / 森人 / 耳長人](./02-elf.md)

- 長い耳と美貌を持ち、長命で、魔法の扱いに秀でた種族
- 秩序の秘宝を発見した五種族のひとつ
- 水の中にも適応でき、知力と魅力、技量に秀でる

### [ドワーフ / 鉱人](./03-dwarf.md)

- 頑健な寸胴体型の身体を持つ、地下や山岳に住まう種族
- 秩序の秘宝を発見した五種族のひとつ
- 生命力と筋力、技量、知力に長ける

### [ビースト / 獣人](./04-beast.md)

- 獣人とも呼ばれる、動物の特徴を持つ種族たちの総称
- 秩序の秘宝を発見した五種族のひとつ
- 狼、猫、熊、鹿、猪など、様々な動物の特徴を持つ亜種が存在する 
- 総じて生命力と筋力に長け、獣の性質に応じた能力を得る

### [ドラゴニュート / 龍人](./05-dragonewt.md)

- 龍の血を引くとされる種族。龍の形質を持った亜人
- 秩序の秘宝を発見した五種族のひとつ
- 強靭な肉体に加え、高い知性と龍への信仰心を併せ持ち、戦闘に秀でる

### [ダークエルフ / 黒森人 / 洞森人](./06-dark-elf.md)

- エルフの同族でありながら、地下世界を生活圏とする種族
- 外見は森人と同様に美しく、黒色の肌を持つ
- 地上のエルフと比して膂力に優れるが、水への適応力が弱まっており、精霊の魔法を失っている
- 闇の魔法に通じ、夜目が利く

### [リザードマン / 蜥蜴人](./07-lizardman.md)

- 爬虫類の特徴を持つ種族
- 生命力と筋力に優れ、竜への信仰心と硬い鱗を持ち、爪と尾は武器となる

### [ハーフリング / 小人](./08-halfling.md)

- ヒュームの半分ほどの、小柄な体躯を持つ種族
- 非力だが身軽であり、器用さに長ける

### [バードフォーク / 有翼人 / 鳥人](./09-birdfolk.md)

- 鳥の形質を持った亜人。鳥人とも呼ばれる、鳥の翼と脚を持つ飛行種族
- 短い寿命、脆弱な生命と引き換えに、高い技量と知性、飛行能力を持ち、膂力にも長ける

### [フェアリー / 蝶人 / 妖精](./10-fairy.md)

- 蝶の羽を持つ小人。蝶人、妖精とも呼ばれる
- 30cm～50cm程度の小さな身体と飛行能力を持ち、魔法の扱いに長ける
- その身体は極めて脆弱であり、まともな筋力を持たない

### [ケンタウロス / 人馬 / 馬人](./11-centaurus.md)

- 半人半馬の姿を持つ種族。草原や平原を駆ける遊牧の民
- 生命力や筋力に長け、身軽さも併せ持つが、身体が大きい

### [マーフォーク / 人魚 / 魚人](./12-merfolk.md)

- 魚の身体を持つ水棲種。海や深い湖に暮らし、水中での生活に適応している
- 下半身を人の脚に擬態することで陸上での活動も可能だが、得意とはしない
- 容姿に優れ、身のこなしにも長ける

### [ウッドボーン / 樹人](./13-woodborn.md)

- 樹木の形質を持った亜人。樹人とも呼ばれる
- 生命力、精神力に秀でているが、その動きは鈍重である
- ウッドボーンの奇跡は、知識によっても行使することができる

### [エルトメンヒェン / 花人 / 草人](./14-erdmannchen.md)

- 草花の形質を持った亜人。花人、草人とも呼ばれる
- 生命力と運に優れ、他種族を魅了する外見や匂いを有する
- 花人の魔法は、その魅力によっても効果を高める

### [マイセリアン / 茸人](./15-mycelian.md)

- キノコの形質を持った亜人。茸人とも呼ばれる、知性を持つキノコの種族
- 生命力に秀でた種族であり、状態異常に対する耐性が非常に高いが、火に脆弱

### [アラネア / 蜘蛛人](./16-aranea.md)

- 上半身がヒト、下半身が蜘蛛の亜人種。蜘蛛の脚を8本、人間の上肢を2本持つ
- 壁や天井を這って移動することが可能であり、その身のこなしは素早く、運にも恵まれている

### [サーペンテイル / 蛇人](./17-serpentail.md)

- 上半身がヒトの姿、下半身が蛇の姿を持つ亜人種
- 竜への強い信仰心を持ち、さらに身のこなしにも長ける
- サーペンテイルはその牙に強い毒を持つ
