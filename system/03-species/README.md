
# 種族

- このドキュメントでは、ルゥレーンに生息する様々な種族のデータを扱います
- 種族ごとのステータス、特性、背景、魔法・奇跡を参照する場合は、それぞれの種族のドキュメントを参照してください

---

## 秩序文明の種族 / 人族

- PCとして選択可能な種族
- 秩序の統一文明に属している

### [ヒューム / 人間 / 只人](./01-hume.md)

- 最も数が多く、秩序文明の中核を担う種族
- 秩序の秘宝を発見した五種族のひとつ
- 能力的には特徴がないが、人の本質的な力、運に長ける

### [エルフ / 森人 / 耳長人](./02-elf.md)

- 長い耳と美貌を持ち、長命で、魔法の扱いに秀でた種族
- 秩序の秘宝を発見した五種族のひとつ
- 水の中にも適応でき、知力と魅力、技量に秀でる

### [ドワーフ / 鉱人](./03-dwarf.md)

- 頑健な寸胴体型の身体を持つ、地下や山岳に住まう種族
- 秩序の秘宝を発見した五種族のひとつ
- 生命力と筋力、技量、知力に長ける

### [ビースト / 獣人](./04-beast.md)

- 獣人とも呼ばれる、動物の特徴を持つ種族たちの総称
- 秩序の秘宝を発見した五種族のひとつ
- 狼、猫、熊、鹿、猪など、様々な動物の特徴を持つ亜種が存在する 
- 総じて生命力と筋力に長け、獣の性質に応じた能力を得る

### [ドラゴニュート / 龍人](./05-dragonewt.md)

- 龍の血を引くとされる種族。龍の形質を持った亜人
- 秩序の秘宝を発見した五種族のひとつ
- 強靭な肉体に加え、高い知性と龍への信仰心を併せ持ち、戦闘に秀でる

### [ダークエルフ / 黒森人 / 洞森人](./06-dark-elf.md)

- エルフの同族でありながら、地下世界を生活圏とする種族
- 外見は森人と同様に美しく、黒色の肌を持つ
- 地上のエルフと比して膂力に優れるが、水への適応力が弱まっており、精霊の魔法を失っている
- 闇の魔法に通じ、夜目が利く

### [リザードマン / レプティリアン / 蜥蜴人](./07-lizardman.md)

- 爬虫類の特徴を持つ種族
- 生命力と筋力に優れ、竜への信仰心と硬い鱗を持ち、爪と尾は武器となる

### [ハーフリング / 小人](./08-halfling.md)

- ヒュームの半分ほどの、小柄な体躯を持つ種族
- 非力だが身軽であり、器用さに長ける

### [バードフォーク / 有翼人 / 鳥人](./09-birdfolk.md)

- 鳥の形質を持った亜人。鳥人とも呼ばれる、鳥の翼と脚を持つ飛行種族
- 短い寿命、脆弱な生命と引き換えに、高い技量と知性、飛行能力を持ち、膂力にも長ける

### [フェアリー / 蝶人 / 虫人 / 妖精](./10-fairy.md)

- 蝶の羽を持つ小人。蝶人、虫人、妖精とも呼ばれる
- 30cm～50cm程度の小さな身体と飛行能力を持ち、魔法の扱いに長ける
- その身体は極めて脆弱であり、まともな筋力を持たない

### [ケンタウロス / 人馬 / 馬人](./11-centaurus.md)

- 半人半馬の姿を持つ種族。草原や平原を駆ける遊牧の民
- 生命力や筋力に長け、身軽さも併せ持つが、身体が大きい

### [マーフォーク / 人魚 / 魚人](./12-merfolk.md)

- 魚の身体を持つ水棲種。海や深い湖に暮らし、水中での生活に適応している
- 下半身を人の脚に擬態することで陸上での活動も可能だが、得意とはしない
- 容姿に優れ、身のこなしにも長ける

### [ウッドボーン / 樹人](./13-woodborn.md)

- 樹木の形質を持った亜人。樹人とも呼ばれる
- 生命力、精神力に秀でているが、その動きは鈍重である
- ウッドボーンの奇跡は、知識によっても行使することができる

### [エルトメンヒェン / 花人 / 草人](./14-erdmannchen.md)

- 草花の形質を持った亜人。花人、草人とも呼ばれる
- 生命力と運に優れ、他種族を魅了する外見や匂いを有する
- 花人の魔法は、その魅力によっても効果を高める

### [マイセリアン / 茸人](./15-mycelian.md)

- キノコの形質を持った亜人。茸人とも呼ばれる、知性を持つキノコの種族
- 生命力に秀でた種族であり、状態異常に対する耐性が非常に高いが、火に脆弱

### [アラネア / 蜘蛛人](./16-aranea.md)

- 上半身がヒト、下半身が蜘蛛の亜人種。蜘蛛の脚を8本、人間の上肢を2本持つ
- 壁や天井を這って移動することが可能であり、その身のこなしは素早く、運にも恵まれている

### [サーペンテイル / 蛇人](./17-serpentail.md)

- 上半身がヒトの姿、下半身が蛇の姿を持つ亜人種
- 竜への強い信仰心を持ち、さらに身のこなしにも長ける
- サーペンテイルはその牙に強い毒を持つ

## 魔族

- 魔族として扱われる種族について解説します

> ステータスについて:
> - 一部の種族には、ロール式を用意しています
> - エネミーデータを生成する場合や、その種族の生まれの限界を把握するときの目安として参照してください

### ゴブリン / 子鬼族

- 緑色の肌を持つ、小型の魔族。数が多く、群れで行動する
- 知性は低いが狡猾で、ダンジョンや辺境に巣食い、冒険者や旅人を襲う。最も遭遇頻度の高い魔族のひとつ
- 集団での戦術行動が可能で、簡素な道具や武器を扱う
- 悪鬼族に近いホブゴブリンやゴブリンチャンピオンなども存在する
- 知能の高い個体はゴブリンシャーマンやゴブリンウィザードとなるなど、種族内での個体の多様性に富む
- 集団行動は可能な一方で、文明を築くことができない略奪民族である
- 自給自足ができないため、必ず他者から糧を奪おうとする

> 小鬼ではなく子鬼:
> - ゴブリンはより強き者に従属する魔族であり、悪鬼の子分という意味で、子鬼と呼ばれる
> - 小さな悪鬼ではない。ホブゴブリンなどの亜種は、成人のヒュームを超える体躯を持つ

### コボルド / 犬小鬼族

- 犬と爬虫類の形質を混ぜたような、醜悪な外見を持つ小型の魔族
- 子鬼族と同様に群れで行動し、ダンジョンや廃坑に巣食う
- 罠の設置に長けており、その狡猾さは体格の小ささを補って余りある
- 他の強力な種族に従属しており、共生関係を結ぶことで種族として生きながらえてきている
- 秩序の文明内でも奴隷として飼われているケースが存在する。人権がなく、その扱いは凄惨である

### オーク / 鬼豚族 / 鬼猪族

- 豚や猪に似た顔つきを持ち、ヒュームよりも一回り大きな体躯を持つ魔族
- 好戦的で粗暴だが、子鬼よりも知性があり、独自の部族社会を形成している
- 鍛冶の技術をある程度持ち、粗雑ながら武具を自作する
- 加工済の資材を求め、秩序文明を襲撃する
- 集団での戦闘を得意とし、各々に役割を定めて部隊を編成することがある

### トロール / 巨鬼族

- 巨体と驚異的な再生能力を持つ魔族
- 傷を負っても瞬く間に回復するため、冒険者にとって厄介な敵として知られる
- 知性は低く、単独で行動することが多い
- 強力な魔法種族から洗脳や調教を受け、下僕として扱われるケースがある

### オーガ / 悪鬼族

- 知恵を持つ魔族。巨大な体躯と膂力に加え、魔法を操る知性を備える
- 単体でも危険だが、子鬼族やオークなどの下位魔族を使役して群れを率いることが多い
- オーガが指揮する魔族の集団は組織的に行動し、村落や隊商を襲撃する
- ジェネラルと呼ばれる、高い知性と悪意を併せ持つ上位種も存在し、秩序文明にとっての大きな脅威となる
    - ジェネラルの討伐のために精鋭の冒険者のパーティが組まれるほどである
- オーガはその強大な力によって、魔族たちの軍勢を恐怖で従わせている
- 魔族の文明圏の支配層でもあり、独自の社会を形成している

> 悪鬼と悪鬼族:
> - 悪魔以外の魔族は、悪鬼として一括りにされることが多い
> - 魔族全般を悪鬼と呼称するのは、オーガが魔族たちを統べることが多いことに由来する
> - 秩序文明と敵対してきた魔族の軍勢はオーガに率いられており、故に他の魔族たちは悪鬼の子分として認定されているのである

### アラクネ / 鬼蜘蛛族 / 土蜘蛛族

- 蜘蛛の下半身に、悪鬼の上半身を持つ魔族
- 罠の構築に長ける種族であり、地下やダンジョンの暗がりに巣を張り、獲物を待ち伏せる
- ダンジョン内のトラップの一部はこの種族の仕業であるとも、その祖先が古代に仕掛けたものとも言われる
- 蜘蛛の脚は6本あり、毛に覆われている。大きな体躯を持ち、口に鋏角を備え、複数の目を持つなど、アラネアとは外見的にも大きく異なり、区別は容易
- 並以上の知性があり、他の魔族と協調して文明圏を脅かすこともある

> アラネアとの関係性:
> - 土蜘蛛であるアラクネは、地上で生きるアラネアを惰弱な種族として嫌悪している
> - 身体の大きさや個体としての強さはアラネアの比ではなく、他種族の力がなければ生きられないとアラネアを見下している
> - アラクネは蜘蛛の脚が6本、ヒトの腕が2本だが、上肢と合わせて8本足であるため、アラネアよりも蜘蛛の原型に近い種族であるとされる

### ラミア / 鬼蛇族

- 上半身がヒトの姿、下半身が蛇の姿を持つ魔族
- 妖艶な容姿と魅了の魔法を操り、旅人を惑わせる
- 砂漠や遺跡に潜むことが多く、その知性は高い
- 肉食性であり、他種族を構わず捕食する
- 妖艶な見た目に反し、蛇の胴体は筋肉質であり、強靭かつ素早い
- 全長は2m～5mほどであり、個体によってサイズが大きく異なる種族でもある

> ラミアの性別:
> - 雌雄ともに存在する
> - 卵生であり、産卵期のラミアは特に気が立っている

### ミノタウロス / 牛鬼族

- 牛の頭を持つ巨大な魔族
- 膂力は悪鬼に匹敵し、迷宮内での方向感覚に優れる
- 動物としての本能が強く、知性はあるが本能を優先する。雑食であり、生の肉を好む
- 迷宮に好んで住み着く。ダンジョンの守護者として遭遇することが多い
    - この傾向から、ミノタウロスには迷宮を感知する力があると、学者の間では推測されている

### ピスキシアン / 鬼魚族

- 海に住まう魔族。遠洋や深海に生息している
- いかつい魚の顔と身体に、四足を生やした半魚人。成体の体長は2m程度ほど
- エラ呼吸をしており、完全な水生生物だが、短時間であれば陸上でも活動ができる
- 水深200m以上の深海に縄張りを形成し、海面に浮上して狩りを行う

### 悪魔

- 魔族の一派、悪魔神の眷属たち
- 通常の生物から逸脱した生態を持つ個体が多く、その大半は人間目からして醜悪な見た目を持つ
- 魔法生物とは異なり、明確に意思を持っており、生物としてルゥレーンの地に生まれてきている
- 秩序文明からは悪魔と一括りに扱われているが、その種類や生態は様々である
    - 便宜上、悪魔神の眷属の種族を悪魔と呼称している状態である
    - 悪魔の文明内では、それぞれに種族が認められている
- 悪魔たちは悪魔神の化身として、魔王を崇拝している

> 魔族と悪魔:
> - 魔族とは「秩序文明に敵対する種族」のことを指す
> - 悪魔とは「悪魔神の眷属」のことを指す
> - 悪魔はルーツの段階で真実神と秩序文明に敵対しているが、その他の魔族は立場として秩序文明に敵対しているという差がある
>
> 悪鬼のルーツ:
> - 悪鬼たちは、悪魔と異なり、他種族と同様の亜人種である
> - 亜人種たちはそのすべてが秩序文明に属したわけではなく、敵対する種族も多く存在した
> - それらは秩序文明から悪鬼と呼称され、魔族として認定された
> - 悪鬼の多くは悪魔と協力関係にあるが、悪魔と違い悪魔神への忠誠心がないため、種の保存を本能的に優先する
>
> 魔王:
> - 悪魔神の化身として、悪魔たちから崇拝されている存在
> - 歴史上では、統一の時代において勇者に討ち果たされたと記録されている
> - 悪魔たちは、魔王の復活を目論み、各地で暗躍しているとされる

## アンデッド

- アンデッドとして扱われる種族について解説します

### ヴァンパイア / 吸血鬼

- 不死の呪いを受けた存在。当人たちは、それを祝福と呼称している
- 元はいずれかの種族であったが、血の渇きに囚われ、夜の世界に生きる種族
- 強大な魔力と身体能力を持つが、陽光や聖なる力に弱い
- 種族ではなく、呪われた状態とする学者もいるが、血統として受け継がれるケースが確認されている
- 高い知性を持ち、共通語も自在に操るが、統一文明で生きることができないため混沌の眷属となりやすい

### イモータル

- 死を超越した魔術師の成れの果て
- 膨大な魔力と知識を持つ不死の存在であり、ダンジョンの最奥に君臨することがある
- かつていずれかの種族であった者が、禁忌の術によってこの姿に至る

## その他の種族

- 竜族や龍などについて解説します

### ドラゴン / 龍 / 龍族

- 太古より存在する強大な種族。四足と二対の翼を持つドラゴン。龍族とも呼ばれる
- ワイバーンやドレイク、ドラゴニュート、リザードマンの始祖とされる種族であり、神聖視されている
    - 神に最も近い種族とされ、信仰の対象となっている
- 圧倒的な力と知恵を持つが、混沌戦争によってその数を大きく減らした
    - 秩序文明に対して友好な個体については、現代では保護対象とされている
- 繁殖行動は公式には記録されておらず、その生態には諸説がある
    - 神話の時代においては、番を形成して子を成したとも伝えられている
    - 死期が近づくと卵に還り、新たな生命として生まれ変わる
- 一部の龍は個体名で呼ばれており、他の龍とは一線を画する存在として認知されている

> ドラゴンから見た多種族との関わり:
> - ドラゴンは、自身が神の手により創造された種族と主張しており、それ以外の出生を語ることを種族レベルで固く禁じている
> - 智慧持つ龍の一部は、人間に擬態して文明社会で生きている者も存在する
> - ドラゴニュートは龍の末裔であると認めているが、リザードマンは卑小な竜族の末裔だと認識している
>
> 始祖の龍:
> - 始祖の龍は真実神の手によって生み出された原初の生命であり、すべての龍の始祖である
> - 他のドラゴンたちの主張は正しいが、一部間違ってもいるという状況
> - 神なき時代より、始祖の龍は他の龍との交流を断っている

### ワイバーン / 翼竜

- 上肢が翼と一体化した、空を飛ぶ竜種
- 大きな翼に加えて長い尾を持つ
- ドラゴンに比べると小型で知性も劣るが、それでも強力な生物
- 知性の高い個体は、人語を解することがあるが、文明には参画していない
    - 有翼人の一種として分類しようとする者もいるが、秩序に種族に含めるかは議論が分かれている

### ドレイク / 地竜

- 地竜と呼ばれる、大地を四足で駆ける竜種
- ドラゴンの末裔とされるが、翼が退化してなくなっており、知性も低い個体がほとんど
- 言葉を解さない個体が多く、動物の一種と扱われることも多い

### 魔法生物

- 魔法や古代技術によって作り出された人造物
- スライムやゴーレム、ミミックなどが該当する
- 意思を持たない単なる兵器とされることが多い

> 使役されぬ者:
> - 秘宝の力で自我を持つゴーレムがいるといった伝承があり、そうした者は周囲に気づかれることなく文明社会に混ざり込んでいるとも言われている
> - 自我とともに高い知性を持つ個体も存在し、文明社会で生きている場合もある
> - そうした個体は特殊なオートマタとして動作しており、一種の不死性を持ち、社会の支配者側に立つこともある

### グリフォン

- 鷲の上半身と獅子の下半身を持つ幻獣
- 言葉を持たない種族であるが、高い知性を持ち、騎乗獣として飼育されることもある

### ジャイアント / 巨人

- 5mを越える巨体を持つ種族。大きさ以外の外見的特徴はヒュームと似ている
- 巨人族は、鍛冶神を祖先に持つと主張している
    - 神話の時代、真実神の意を受けて大地の骨たる鉱脈を鍛え、山脈を打ち立てたのが巨人族たちであったと伝えられている
- 統一の時代において、巨人族は統一文明へ参加しており、秩序の種族として認められている
- その在り方から、他種族と交わって暮らすことが難しいため、文明圏から離れた地で暮らしている
