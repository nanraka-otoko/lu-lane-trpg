# 上級冒険者の作成 / 冒険の軌跡

- このドキュメントでは、上級冒険者の作成手順と、冒険の軌跡のルールについて解説します

---

## 冒険の軌跡

- PCがキャラクターメイキング以降に積み重ねてきた冒険の経験のこと
- GMはPCに対して冒険の軌跡を付与することができる
- 冒険の軌跡は1点につき1度の冒険を指し、その分だけPCは冒険を経験する
- PCは冒険表のロールを行い、結果に応じてWLT・EXPを獲得し、成長する

> 冒険の軌跡の目的:
> - 冒険の軌跡は、セッションを経ずに、冒険と同程度の経験やお金を入手する仕組みです
> - すでに何度もセッションを通じて冒険を重ねてきたPCは、冒険の軌跡を使用する必要はありません
> - 新規作成する冒険者で高難易度のセッションに挑みたい場合や、周囲の冒険者とのお金や成長状況を合わせるために使用すると良いでしょう

---

## 冒険の軌跡の獲得手順

PLは下記のルールで判定を行い、冒険を経験する:
1. 冒険表を1つ選ぶ
2. d6を振り、冒険内容を決定する
3. 判定前に、WLTの使用量を決める (使用しなくても良い)
4. 次の式でダイス数を決定する
    - $ダイス数 = 4 + 冒険回数 + WLT消費ボーナス$
    - 最初のダイス数は4d。ダイス数は1回の冒険を終えるごとに恒久的に+1dされ、以降の冒険に引き継がれる
5. 達成値と難易度を比較し、大成功・成功・失敗のいずれかを確定する
6. 冒険の結果に応じた報酬を獲得する

### 冒険の結果

- 大成功 / 成功 / 失敗のいずれかとなる
- 冒険の難易度と、ロールのMtVを比較して決定する
    - $MtV > 難易度$: 大成功
    - $MtV = 難易度$: 成功
    - $MtV < 難易度$: 失敗

### 冒険への投資

- 冒険の判定前にWLTを消費することで、その冒険に限り、判定にボーナスが得られる
- WLTの前借りはできず、WLTが0点未満となるように消費することもできない

| 消費WLT | ボーナス |
| ------: | -------: |
|       1 |      +1d |
|       3 |      +2d |
|       6 |      +3d |
|      10 |      +4d |

> 投資の意味:
> - ポーションや触媒など消耗性のあるアイテムを使ったりして、その冒険を有利に進めたということを意味する

### LUCの消費と判定の振り直し

- 1点のLUCを恒久的に失うことで、冒険の判定を振り直すことができる

> LUCの恒久的消費:
> - PCは、実際の冒険中でもLUCを消費することで判定を振り直すことができる
> - 冒険の軌跡においても、LUCを消費して判定をやり直せる
>
> ヒュームの場合:
> - ヒュームは特性「天運」の効果で、LUCの消費によって冒険の判定のMtVを+1することができる

### 冒険の軌跡と報酬

- PCは冒険の成否に関わらず、1度の冒険につき1点のEXPを獲得する
- PCは冒険の大成功 / 成功 / 失敗の結果に応じ、報酬を獲得する

> 結果の意味:
> - 成功: 依頼や目的を達成した。標準的な報酬を得る
> - 大成功: 追加の目的を達成したり、良い戦利品を得た。多くの報酬に加え、冒険によっては経験点を獲る
> - 失敗: 依頼や目的を果たせなかった。少しの戦利品を得て撤退したケースもあれば、消耗や違約金が必要になるケースもある

## 既存PCへの付与

- 既存のPCに対し、冒険の軌跡を付与しても良い
- GMはそのPCのこれまでのプレイ回数を目安として、判定ダイス個数を決定する

> PC間のステータス調整での活用:
> - 同じメンバーでセッションをプレイできなかったとき、未参加PCに冒険の軌跡を与えてステータス調整を図っても良い
> - 冒険の軌跡に与えるボーナスダイスは、GMの裁量で決定して良い
> - どの冒険表を選ぶかはPLが決定するが、状況に応じてGMが指定しても良い
>
> オリジナルの冒険表:
> - ここで紹介している冒険表は一例であり、GMとの合意のもとでオリジナルの冒険表を作成し、利用しても良い
