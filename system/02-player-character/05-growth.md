# 成長

- このドキュメントでは、プレイヤーキャラクターの成長についてのルールを解説します

---

## 成長のルール

- 経験点 (EXP) が旅路 (JNY) と同値以上になると、そのPCは成長する

$$
EXP >= JNY
$$

- 成長後の EXP は、成長前の EXP から JNY を引いた数値となる
- 成長後、JNY を+1する

$$
成長後のEXP = EXP - JNY
$$

$$
成長後のJNY = JNY + 1
$$

- PCは成長したら成長ロールを振り、ステータスを増加する

> 解説:
> - 若い冒険者ほど旅路の初期値が低いため成長しやすい
> - 成長するほど旅路が増えていくため、より多くの経験点が必要となる

## 成長ロール

- 成長時、D6を3回振り、出目に対応したステータスを参照し、その内の1つを選んで+1する
    - 3回の出目が同じ場合、そのステータスを+1する

| 出目 | ステータス |
| ---: | :--------- |
|    1 | CON        |
|    2 | MND        |
|    3 | STR        |
|    4 | DEX        |
|    5 | INT        |
|    6 | FTH        |

例: 
- 1, 1, 1 の出目が出た場合、 CON を+1する。
- 3, 4, 6 の出目が出た場合、 STR、DEX、FTH から1つを選んで+1する。

LUC / APP の成長:
- LUC と APP は、冒険による成長では上昇しない

## 成長の例

- JNY 3、EXP 2のPCが、冒険により2点のEXPを獲得した
- EXPがJNYを上回ったので、EXPを3点消費し、成長を行う
- D6を3回振る → 1, 4, 5
- CON、DEX、INTのいずれかを+1する → CONを選択
- JNYを+1する
