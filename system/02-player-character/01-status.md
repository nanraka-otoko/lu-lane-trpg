# ステータス

- このドキュメントでは、キャラクターのステータスとその意味について解説します
- プレイヤーキャラクターのセクションとして扱っていますが、ここで説明するステータスのルールは、エネミーを含めたすべてのキャラクターにも適用されます

---

## ステータスの説明

- すべてのキャラクターは、下記のステータスを持つ

| 略称 | 英語         | 日本語 |
| :--- | :----------- | :----- |
| CON  | Constitution | 生命力 |
| MND  | Mind         | 精神力 |
| STR  | Strength     | 筋力   |
| DEX  | Dexterity    | 技量   |
| INT  | Intelligence | 知力   |
| FTH  | Faith        | 信仰   |
| LUC  | Luck         | 運     |
| APP  | Appeal       | 魅力   |

---

## ステータス項目の説明

1. 生命力 / Constitution (CON)
    - 最大 HP を決める数値
2. 精神力 / Mind (MND)
    - 最大 MP を決める数値
3. 筋力 / Strength (STR)
    - 力強さを表す数値
    - 武器や防具の装備条件、道具の運搬に影響し、重い仕掛けを動かす時の判定にも使用する
    - 筋力が大きいほど、巨大な武器や重鎧などを装備できる
4. 技量 / Dexterity (DEX)
    - 器用さを表す数値
    - 探索、行動、攻撃、回避などの判定に影響する
5. 知力 / Intelligence (INT)
    - 賢さ、知性、知識量を表す数値
    - 文字の解読、言語の習得、魔法の行使条件に影響する
6. 信仰 / Faith (FTH)
    - 信心深さを表す数値
    - 神聖語の解読、奇跡の行使条件に影響する
7. 運 / Luck (LUC)
    - 人の本質的な力、幸運を引き寄せる能力
    - 危険の察知、アイテムや素材の収集など雑多な判定に影響する
    - 戦闘におけるイニシアチブの決定は、運判定により行う
    - すべてのキャラクターは、1点のLUCを恒久的に失うことで、その判定をやり直すことができる
8. 魅力 / Appeal (APP)
    - 人の本質的な力、他者を引き寄せる能力
    - 交渉などの対人的な判定に影響する
    - 容姿だけでなく、社会的な信用や栄誉によっても魅力は変動する

---

## 数値の目安

- ヒュームの成人の一般人は、すべてのステータスが3程度である
- ヒュームの成人を基準とした目安を、下記の表に示す

| 能力値 | 説明                                                                                 |
| -----: | :----------------------------------------------------------------------------------- |
|  0以下 | 最低限度の能力すら持っていない状態。日常生活、あるいは生命活動に致命的な影響を及ぼす |
|      1 | 最低限の能力。生活は送れるが、生産的なことはできない。偏差値30程度                   |
|      2 | 常人より劣るが、見渡してみればよくいる程度の能力。偏差値40程度                       |
|      3 | 常人の基準。偏差値50程度                                                             |
|      4 | 多少の訓練をしており、常人より優れる。偏差値55程度                                   |
|      5 | 専門家の基準を満たす程度の能力。偏差値60程度                                         |
|      6 | 優れた専門家。偏差値65程度                                                           |
|      7 | その道において失敗がないほどに熟練。偏差値70以上                                     |
|      8 | 通常の訓練では到達できない高み。才能が必要な領域                                     |
|      9 | その分野においては、世界屈指の能力者                                                 |
|     10 | 歴史に名前を残せるほどの能力者                                                       |
| 11以上 | 只人の身ではまず不可。人間をやめるほどの才能と努力が必要                             |

> ステータスの下限・上限:
> - キャラクターメイキング時、各ステータスは 1 ～ 10 の間で決定します
> - 成長や装備により、10 を超える場合があります
> - ステータスが0以下となった場合、致命的な状況がPCに生じます

## パラメータ

1. HP / Hit Point
    - 最大値は $CON×2$
    - HPが0以下になると瀕死となり、その後に気絶する
    - HPが-CON点以下になると、そのキャラクターは気絶する
    - HPが-(CON+LUC)点以下となると、そのキャラクターは死亡する
    - 1晩の休息で1点回復する。休息しない場合、回復しない
2. MP / Mind Point
    - 最大値は $MND$ と同値
    - 魔法、奇跡の発動コスト
    - 1晩の休息で全回復する。休息しない場合、回復しない
3. お金 / Wealth (WLT)
    - 資産の数値
    - この数値の分だけ、セッション内にアイテムを持ち込むことができる
    - 何かしらの原因でお金が負の値となる場合、その額がプラスに転じるまでは買い物ができない
    - PL自らが借金をすることはできない。GMが許可した場合は可

> HPの目安:
> - ダガーによる攻撃の最低ダメージは2点程度 (威力1+MtV1)
> - 一般人 (CON3、HP6) は、ダガーの攻撃を2回は耐えられるが、3回食らうと瀕死に陥る
> - ショートソードによる攻撃であれば2回で瀕死、クレイモアを食らうと1回で、という程度の耐久力が一般人の基準
>
> 瀕死/気絶/死亡:
> - PCはHPが尽きても、即座に死亡とはならず、意識を手放してから死に至る
> - 運に恵まれた者は、死を遠ざけやすい
> - 詳細は [../07-combat/01-hp-and-damage.md](../07-combat/01-hp-and-damage.md) を参照すること
>
> MPの目安:
> - 汎用的なスペルはMPを1消費する
> - MPが3の場合、一般人でも使える魔法なら、それを3回は使える

## キャラクターの設定

1. 才能
    - キャラクター作成時に決定する、生まれ持った才能
    - 才能により、特殊な能力が得られる
2. 背景
    - キャラクター作成時に決定する、これまで歩んできた生業
    - 選んだ背景により、ステータスが変動する
    - 判定時、自身の背景が役に立つ場合、GMに相談することでボーナスダイスが得られる
3. 老いの代償
    - キャラクター作成時、老いたPCが背負う代償を決定する
    - ステータスが低下する
4. 旅路 / JNY
    - これまでの冒険や人生の旅路
    - 初期値は年齢の背景数
    - PCが成長すると、旅路が1点増加する
5. 経験点 / EXP
    - 依頼の達成や、大きな功績により得られる、冒険者としての経験値
    - 経験点が旅路と同値になると、そのPCは成長する
    - 成長したら、旅路を+1点し、経験点を0にする (超過した経験点は持ち越し可)

## ステータスが0以下となった場合

1. 生命力
    - 生命力が 0 の場合、そのキャラクターは生命活動を停止し、死亡している
        - この状態のキャラクターは蘇生ができず、キャラクターロストとなる
    - 一時的に生命力が 0 になった場合も、そのキャラクターは恒久的に死亡する
        - 人の形を保てず、元に戻れない状態
2. 精神力
    - 精神力が 0 の場合、そのキャラクターは精神的な障害を抱えている
        - プレイ不可となり、実質的キャラクターロストとなる
    - 一時的に精神力が 0 になる場合、プレイヤーはキャラクターの操作権限を一時的に喪失する
3. 筋力
    - 筋力が 0 の場合、他者の介助なしでは移動などの物理的な行動ができなくなる
    - 戦闘中では、あらゆる行動・予備行動ができなくなる
4. 技量
    - 技量が 0 の場合、あらゆる道具を使用することができなくなる
    - 戦闘中では、あらゆる行動・予備行動ができなくなる
5. 知力
    - 知力が 0 の場合、他者とのコミュニケーションを行うことができない
        - プレイ不可となり、実質的キャラクターロストとなる
    - 知力が 1 の場合、統一言語の読み書きができない
6. 信仰
    - 信仰が 0 の場合、神の存在を信じていない or 理解できない
    - 信仰が 1 の場合、神の存在は知っているが、肯定していない
    - 信仰が 2 の場合、神の存在は信じることができるが、神聖語は理解できない
7. 運
    - 運が 0 の場合、運を用いるすべての判定に自動失敗する
    - 多くの場合、キャラクターロストの危険を伴う
    - セッションでの敗北時、LUCが0のPCは死の結末から逃れることはできない
8. 魅力
    - 魅力が 0 の場合、秩序の文明圏に立ち入ることができない
        - 孤独死によるキャラクターロストとなる
