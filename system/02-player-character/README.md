# プレイヤーキャラクター

- このドキュメントでは、プレイヤーキャラクター (PC) の情報と、作成方法について解説します

---

## キャラクターのステータス項目

- すべてのキャラクターは、生命力、精神力、筋力、技量、知力、信仰、運、魅力のステータスを持つ
- ステータスの項目については [01-status.md](./01-status.md) を参照すること

---

## キャラクターシート

- PLとGMは、プレイヤーキャラクターを「キャラクターシート」の形式で作成し、管理する
- キャラクターシートのテンプレートは [../10-appendix/character-sheet.md](../10-appendix/character-sheet.md) を参照すること

> フォーマットは自由:
> - 提示したキャラクターシートは、あくまでも一例です
> - 使いやすいように改良・改変して使用して構いません

---

## キャラクターメイキング

- 以下の順序でキャラクターを作成する
- 各ステップは前のステップの結果に依存するため、必ずこの順番で行うこと

> 才能・背景・老いの代償の選択について: 
> - 才能・背景・老いの代償を決定する際、通常はランダム表を使用するが、ダイスを用いずにPLが自分で選択しても良い
> - 手動での選択を希望する場合は、GMと相談の上で決定すること

### 手順の概要

| 順番 | ステップ         | 内容                                                             |
| ---: | :--------------- | :--------------------------------------------------------------- |
|    1 | 種族の決定       | ステータスのロール式と種族特性を決定する                         |
|    2 | ステータスロール | 種族ごとのロール式に従ってロールし、各ステータスを決定する       |
|    3 | 年齢の決定       | 背景の取得数と老いの代償を決定する                               |
|    4 | 才能の決定       | 6M6でロールし、達成値の数だけ才能を獲得する                      |
|    5 | 背景の決定       | 年齢で決まった数だけ背景を選択し、ステータス補正を獲得する       |
|    6 | 老いの代償の決定 | 高齢のキャラクターは老いの代償を選択し、ステータス補正を獲得する |
|    7 | 信仰対象の決定   | 真実神に加え、信仰する神を1柱選択する                            |

> 最終ステータス = ステータスロール結果 + 才能補正 + 背景補正 + 老いの代償 + その他補正

### 1. 種族の決定

- PLは、自身のPCの種族を選択する
- 種族によって、ステータスや特性、行使できる魔法・奇跡が異なる
- 種族の詳細は [../03-species/](../03-species/) を参照すること

### 2. ステータスロール

- 選択した種族のロール式を用いてステータスを決定する
- ステータスのロール式は、下記のように表される

$$
1 + 4M6
$$

- この場合、4M6でロールし、そのMtVに1を加算した数値が、そのステータスの数値となる
- 例: CON = 1 + 4M6 でMtVが2の場合、CONは3となる
- すべてのステータスについてロールし、数値を決定する

### 3. 年齢の決定

- PCの年齢を選択する
- 年齢ごとに背景の取得数が異なる
- 若いキャラクターは背景が少ないが才能に恵まれやすく、年老いたキャラクターは豊富な背景を持つが老いの代償を受ける

| 年齢 | 背景数 | 老いの代償 | 特殊効果                |
| :--- | -----: | ---------: | :---------------------- |
| 若年 |      2 |          0 | 才能を1つ自由に取得     |
| 壮年 |      3 |          0 | ステータスを1つ選んで+1 |
| 老年 |      4 |          1 | -                       |

1. 若年 (背景: 2)
    - 駆け出しの冒険者として扱われる年齢。実世界では中学生〜大学生・新卒程度
    - 才能を1つ自由に選んで取得する (6M6とは別に任意に1つ獲得できる)
2. 壮年 (背景: 3)
    - 仕事人としての全盛期から、肉体的な衰えが生じ始める年齢。実世界では 30〜40 歳前後
    - ステータス (CON/MND/STR/DEX/INT/FTH/LUC/APP) を1つ選んで+1する
3. 老年 (背景: 4 / 老いの代償: 1)
    - 現役を退きつつある年齢。実世界では 50 歳以上
    - 老いの代償を1つ抱える

### 4. 才能の決定

- 才能の数を $6M6$ で決定し、才能表を参照して才能を獲得する
- 才能決定のルールと才能表については [02-talent.md](./02-talent.md) を参照すること

### 5. 背景の決定

- 年齢で決まった数に応じて、そのキャラクターの生い立ちや生業を決定する
- 背景決定のルールと背景表については [03-background.md](./03-background.md) を参照すること

### 6. 老いの代償の決定

- 老齢のキャラクターは、老いの代償を1つ得る
- 老いの代償のルールと代償表については [04-price-of-aging](./04-price-of-aging.md) を参照すること

### 7. 信仰対象の決定

- すべてのPCは真実神を信仰している
- 真実神に加え、別の神を1柱、信仰対象に選択する
- PCは信仰対象の神の奇跡を行使できる
- PCが信仰可能な神については [../04-spells/](../04-spells/) を参照すること

> 「原理主義」の才能:
> - 「原理主義」の才能を所持しているPCは、信仰対象を得られず、真実神のみ信仰している状態となる
>
> 信仰対象の選択について:
> - 種族ごとに主要な信仰対象は異なるが、PCは好きな神を信仰して良い
> - 豊穣神は「主にヒュームが信仰する神」とあるが、ヒューム以外の種族で信仰しても良い
> 
> 外なる神の信仰:
> - PLが希望する場合、外星神や未踏神も信仰して良い
> - 外なる神を信仰した場合でも、真実神への信仰を失うということはない

### メイキング中にステータスが0以下となった場合について

- キャラクターメイキング時に特定のステータスが0以下となった場合、1としてキャラクターを作成する
    - この処理はメイキングでのみ適用する
    - 以後、特定のステータスが0以下となった場合については、ステータスの説明に従い処理を行う
        - 例: CON が0になったらキャラクターロスト
- お金が0未満となった場合、そのキャラクターは借金をした状態でゲームを開始する
    - 借金を返済するまでは物を買うことはできない
    - 特に利子などは発生しない

### メイキング中にステータスが11以上となった場合について

- キャラクターメイキング時に特定のステータスが11以上となった場合、10としてキャラクターを作成する
    - この処理はメイキングでのみ適用する
- 以後、特定のステータスが11以上となった場合については、その成長やステータス補正を認める
    - 例: STR10のキャラクターが成長でSTRを成長させたら、11として良い

## プレイヤーキャラクターの成長

- セッションを経て、PCはEXPを稼ぎ、成長してゆく
- EXPがJNYを上回った時、PCは成長する
- ルールの詳細は [05-growth.md](./05-growth.md) を参照すること

---

## 上級冒険者の作成

- 「冒険の軌跡」を用いることで、すでに冒険者として活動しているPCを、セッションを経ずに作成することができる
- PCは「冒険の軌跡」を獲得し、EXPとWLTを入手する
- ルールの詳細は [06-adventuring-path.md](./06-adventuring-path.md) を参照すること
