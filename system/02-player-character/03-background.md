# 背景

- このドキュメントでは、キャラクターメイキングにおける背景の決定方法と、背景の一覧について扱います

---

## 背景の決定

- PLは D666 を振り、出目の組み合わせで背景表を参照し、背景を決定する
- 背景は1回の決定につき3回 D666 を振って候補を決定し、その中から好きな背景を選択して獲得する
    - 2回とも同じ出目が出た場合、候補が3種類から2種類に縮小する
    - 3回とも同じ出目が出た場合、候補が1種類となり、その背景を自動的に獲得する
- 候補の中にすでに獲得済の背景がある場合、獲得していない背景を選択して獲得する
- 候補がすべて獲得済の背景である場合、獲得していない背景が出るまで3d6を振り、その背景を獲得する
- 背景の獲得後、次の背景の決定を行う
    - 背景数分のロールを先に全部行って決定するのはNG

すべての背景が決まったら:
- 背景に書かれた内容でPCのステータスを増減し、WLTを獲得する

種族専用背景を用いる場合:
- 背景決定をする際に、ロールせずに種族専用の背景表から選ぶ
- 種族専用のものを何個目の背景に選ぶかは、PLが自由に決めて良い
    - 壮年のキャラクターの例: 1個目は通常の背景表で決定し、2個目は種族専用背景から選び、3個目は通常の背景表で決定する

混血について:
- 混血の才能を持っている場合、種族専用の背景は、2種類の種族の背景表からそれぞれ獲得できる
- 例: エルフの生まれでドワーフの混血となったとき、エルフの背景とドワーフの背景をそれぞれ1つずつ選ぶことができる

---

## 背景決定の例

背景数4個のPCで背景を決定する例:
- 1回目の背景ロール: 1-1-2, 2-3-3, 2-3-3 → 候補は2種。 1-1-2 の兵士を選択して獲得
- 2回目の背景ロール: 1-1-2, 1-1-2, 4-6-5 → すでに 1-1-2 は獲得済なので 4-6-5 下水掃除人 を自動で獲得
- 3回目の背景ロール: 6-6-6, 6-6-6, 6-6-6 → 同じ出目が3回連続で出たので 6-6-6 ニート を自動で獲得
- 4回目の背景ロール: 1-1-2, 4-6-5, 6-6-6 → すべて獲得済なので 3d6 をロール → 2-3-5 魔法道具商が出たので自動で獲得
- 兵士、下水掃除人、ニート、魔法道具商の背景を持つPCが完成した
- CON+3, STR+1, DEX+1, INT+1, LUC+5, APP-1, WLT+3 のステータス補正を獲得した

背景数2個のPCで種族専用背景を用いて背景を決定する例:
- エルフの生まれなので、精霊の従者を獲得
    - この選択は、ダイスを振らずに好きなものを任意に選べる
- 2個目の背景ロール: 1-1-2, 4-6-5, 6-6-6 → 1-1-2 兵士を選択して獲得
- 精霊の従者、兵士の背景を持つPCが完成した
- CON+1, MND+2, STR+1, INT+2, LUC+1, WLT+1 のステータス補正を獲得した

## 背景とロールプレイ

- PCは冒険者になる前に、その背景を生業としている
- 背景と関わりのある物事について判定するとき、PLの提案でGMが認めれば、その判定に+1d～+2d程度のボーナスを与えても良い

ロールプレイボーナスの例:
- 「考古学者」の背景を持つキャラクターが遺跡の碑文を鑑定する場面で、PLが「考古学の知識があるので有利では？」と提案 → GMが認めれば知識判定に+2d

戦闘にはロールプレイボーナスを与えない:
- 「衛兵」の背景を持っていても、「衛兵だから剣での攻撃が得意」という提案があった場合、GMは却下すること
- 「衛兵だったので、街の警備体制についての知識判定にボーナスがほしい」というような戦闘以外での提案があった場合は、GMの判断で許可しても良い
