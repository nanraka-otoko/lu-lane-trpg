# 休息のルール

- このドキュメントでは、冒険中に休息を取る際のルールについて解説します

> 注:
> - このルールは、冒険のリアリティを出すために使用するものです
> - 休息のロールプレイによってテンポが削がれると感じる場合や、シナリオ上で不要な場合は、使用しなくても構いません
>     - その場合は、夜を明かしたらHPを1点回復し、MPを全回復する、通常の処理に従うと良いです
> - 冒険の旅情を出したい場合は、採用すると良いでしょう
> - PCに野営をさせることで、PCたちはHPやMPの回復など、有利な効果を得られます
> - PCの損耗状況に応じて休息をさせるなど、GMの判断でこのルールを活用してください

## 強行ペナルティ

PCは、強行軍を敢行するとペナルティを受ける。

- 1日に4時間以上の睡眠を取らない場合、すべての判定に-1d
- 1日間、食事を摂らない場合、すべての判定に-1d
- このペナルティは重複する

## 野営

- PCは任意のタイミングで野営を実行できる
- 野営を実行する際は、1日分の食料と、4～8時間を要する

### 食料調達判定 (DEX / INT / LUC から選択)

- PCは野営にあたり、1日分の食料を消費する
- 食料がない状態で野営を実行する場合、食料を調達する必要がある
- PCは1時間をかけて食料調達判定を実行できる
- 成功すると、1PTの1日分の食料を入手できる
- 食料調達の目標値は、GMが状況に応じて指定する

| 目標値 | 状況の目安                                           |
| -----: | :--------------------------------------------------- |
|      2 | 自然豊かな森の中、動物のいる草原、魚のいる河原       |
|      3 | 低木の生えた平原、山の中、海上、海中                 |
|      4 | 生物のいる迷宮の中、生物の往来がある洞窟、荒野、深海 |
|      5 | 迷宮の中、洞窟、砂漠                                 |

### 野営の時間

PCたちは野営の時間を決める。

- 4時間: PCたちは一斉に眠る。襲撃のおそれがある
- 6時間: PCたちは交代制で眠る。襲撃に備えることができる
- 8時間: 十分な睡眠が得られる

> これ以外の時間配分を希望する場合は、GMとの相談の上で取り入れても良い

### 野営ロール

- 各PCは、野営の快適度に応じて野営ロールを行う
- 野営ロールは6M6で行い、状況に応じて下記のボーナス・ペナルティを受ける
    - 睡眠時間が4時間未満: -4d
    - 睡眠時間が6時間未満: -2d
    - 食料がない: -4d
    - 野ざらしで雨を浴びる: -4d
    - 迷宮内: -4d
    - 迷宮内の安全な場所: -2d
    - 快適な寝床: +2d
    - 美味な食事: +2d
    - 十分な睡眠: +2d
- 野営ロールはPCごとに行い、達成値に応じて下記表の効果を得る

| 達成値 | 効果                                                                                                                                                    |
| -----: | :------------------------------------------------------------------------------------------------------------------------------------------------------ |
|      0 | MPを1点回復する。不眠と空腹のペナルティを受ける                                                                                                         |
|      1 | MPをMNDの半分 (小数点以下切り捨て) 回復する                                                                                                             |
|      2 | HPを1点、MPをMND点回復する                                                                                                                              |
|      3 | HPを2点、MPをMND点回復する。HPとMPの回復量が最大値を超過した場合、1日の間、最大値を1点超えた状態となる (最大値以下に減ったあとは、それ以上に回復しない) |
|      4 | 達成値3の効果に加え、1日の間すべての判定に+1dのボーナスを得る                                                                                           |

> 例: 
> - HP 7/8、MP 1/3 で達成値3 → HP 9/8、 MP 4/3 となる
> - この状態でHPが7点に減少した後にHPを2点回復した場合は、最大HPである8点で回復は止まる
> - 休息で得たHPは最大HPを超えて回復するが、最大HPは上昇しない
> - MP 4/3 の状態で一日を終えると、翌日のMPは3点に戻る

## 宿泊

- 冒険者が冒険の途中で都市や宿場町で泊まる場合、GMは冒険者たちに宿泊をさせても良い
- 宿泊は休息の代用として機能し、WLTを消費することで休息と同様の効果を得られる

| 宿泊費の相場 | 効果                                                                                                                                                                    |
| -----------: | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|            0 | 一般的な宿屋。PCはHPを1点、MPをMND点回復する                                                                                                                            |
|            1 | 少し豪華な宿屋。HPを2点、MPをMND点回復する。HPとMPの回復量が最大値を超過した場合、1日の間、最大値を1点超えた状態となる (最大値以下に減ったあとは、それ以上に回復しない) |
|            2 | かなり豪華な宿屋。宿泊費1の効果に加え、1日の間すべての判定に+1dのボーナスを得る                                                                                         |

> 調整の目安:
> - GMは、街の雰囲気やロケーション、冒険のシチュエーションに応じて、費用を吊り上げたり、逆に割り引いても良い
> - GMは、費用を抑えて小さい効果を得るか、お金を払って高い効果を得るかを、PLに選択させると良い
