# 跳躍と着地のルール

---

## 跳躍可能な高さ

- 垂直方向への跳躍可能な高度は、DEXによって決定される
- 登攀可能な高度は、身長と腕を加算し、跳躍可能な高度に+2mした距離となる
    - 種族や個体差によって実際の体格は異なるが、ルール上は一律で+2mとして良い

|  DEX | 跳躍可能な高度 | 登攀可能な高度 |
| ---: | -------------: | -------------: |
| 1～4 |             1m |             3m |
| 5～8 |             2m |             4m |
|  9～ |             3m |             5m |

> 高度について:
> - DEXのダイス数でのNM6ロールの期待値を、小数点以下切り下げした数値である
>     - すなわち、汎用行動判定の期待値を参照している
>     - 汎用行動判定にボーナスを得ているキャラクターは、この表の参照時にDEXの数値として加算しても良い
>     - 厳密な距離を求めたい場合は、汎用行動判定を行わせ、その達成値x1mを跳躍できたものとしても良い
> - 高度は、立ち高跳び・走り高跳び等の手段を問わず、生身で地面から足がどれだけ離れたかを表す
> - 実際には手を伸ばすことで、さらに高いところにある壁のフチや崖面を掴むことができる

## 跳躍可能な幅

- 水平方向への跳躍可能な幅は、DEXによって決定される

|  DEX | 跳躍可能な幅 |
| ---: | -----------: |
| 1～3 |           2m |
| 4～6 |           4m |
| 7～9 |           6m |
| 10～ |           8m |

> 水平跳躍の距離について:
> - DEXのダイス数でのNM6ロールの期待値を小数点以下切り下げした数値に、2をかけた数値を目安としている
>     - 垂直方向の跳躍と同様に、汎用行動判定の期待値を参照している
>     - 汎用行動判定にボーナスを得ているキャラクターは、この表の参照時にDEXの数値として加算しても良い
>     - 厳密な距離を求めたい場合は、汎用行動判定を行わせ、その達成値x2mを跳躍できたものとしても良い
> - 走り幅跳びのように、十分な助走がつけられた状態での飛距離を表す
>     - 助走のない状態での跳躍距離は、半分を目安として扱うこと

## 落下と着地

- キャラクターが無事に落下できる距離は、$DEX x 1m$ の式で決定する

例:
- DEX 1のキャラクター: 1m までは安全に落下・着地できる
- DEX 3のキャラクター: 3m までは安全に落下・着地できる
- DEX 10のキャラクター: 10m までは安全に落下・着地できる

### 墜落とダメージ

- DEX x 1m を超える距離を落下した場合、墜落ダメージを受ける
- ダメージは下記の式で決定する

$$
墜落ダメージ = (落下距離 - DEX) x 2
$$

ダメージの適用:
- 墜落によるダメージは、防護と装甲では軽減できず、直接HPにダメージを受ける
- 障壁は、墜落のダメージを軽減できる
- 後述する、着地によってもダメージを軽減して良い

> 具体的な質感について - 全ステータスが 3 の一般人の場合:
> - 3階建ての建物に相当する 6m から落下する
> - 安全に着地できる 3m を越えているため、超過した 3m x 2点のダメージを受ける
> - 6点のダメージを食らい、HPが0点となり、気絶する
> - 4階建ての屋上に相当する 8m から落下した場合は、超過した 5m x2点のダメージを受け、HPが-4となり、死亡する
>
> DEX 1だと2mの高さも安全に降下できないのか:
> - DEX 1では2mの高さでも安全ではない
> - 「生活は送れるが、生産的なことはできない」レベルの能力であり、背の高い物置からの落下でも痛痒を受ける
>     - CON 1, DEX 1 の子供が物置から落下した場合は、命の危険に関わる
> - 着地のルールにより、身構えていれば2mまではDEX 1でも安全に降りることができる

### 着地

- 墜落ダメージを受けるとき、汎用行動判定によってダメージを軽減できる
- 下記の式で、実際に適用するダメージを決定する
    - 適用ダメージは0点未満にならない (マイナスになっても、回復するといったことはない)

$$
適用ダメージ = 墜落ダメージ - 汎用行動判定MtV
$$

着地の条件:
- 不利な姿勢や、身構えていない状態での墜落に対しては、着地を行うことが困難になる
- 柔らかな地面に墜落する際は、有利に着地できる
- GMは状況に応じて、汎用行動判定にボーナス/ペナルティを与えて良い

| ボーナス / ペナルティ | 説明                                 |
| --------------------: | :----------------------------------- |
|                    -4 | 頭からの墜落                         |
|                    -2 | 背中からの墜落                       |
|                    +2 | 土や草のある柔らかな地面             |
|                    +4 | ネットやトランポリンのある安全な地面 |

> ≒ 回避判定:
> - 着地によるダメージ軽減は、攻撃判定に対する回避判定に近しい

### 登攀の扱い

- 垂直登攀を行う場合、GMの判断で汎用行動判定を行わせる
- 失敗した場合は、登攀する高度の半分を目安に墜落する

| 目標値 | 状況の目安                                                            |
| -----: | :-------------------------------------------------------------------- |
|      1 | 跳躍可能な高度に+2mした程度の高さ。判定なしで登攀できる               |
|      2 | ～10m程度の高さ。普段は登攀しないが、しようと思えばできる壁面や崖     |
|      3 | 10～30m程度の高さ。通常は登攀しようと思うことがない崖                 |
|      4 | 30～50m程度の高さ。他に道がない状況でもなければ、まず登ることはない崖 |
|      5 | 50m～の高さ。登るべきではない崖や壁面                                 |

> 壁面の状態:
> - 壁面がゴツゴツしていたり、つかまりやすい突起があったりする場合は、判定にボーナスを与えても良い
> - 手元が見づらい状況や、ツルツルしている壁面などでは、判定にペナルティを与えても良い
>
> 登攀自体の扱い:
> - セッションの進行上、GMが不適切と判断した登攀については、拒否をしても良い
> - シナリオ上で想定していないが、PLの提案による登攀が有効だと判断できる場合は、GMの裁量で登攀をさせても良い
> - 判定の目標値や状況は目安であり、シナリオの進行上で登攀を求める場合は、低い目標値を採用したり、判定不要で昇降させても良い

## 飛行

- 飛行の能力を持つ者は、跳躍・落下・着地において、その条件を無視して安全に移動ができる
- 困難な環境で飛行をする場合は、汎用行動判定を行う
    - 成功した場合は、その環境でも飛行ができる
    - 失敗した場合は、飛行ができない。墜落するかは、シチュエーションに応じてGMが決定する

| 目標値 | 状況の目安                                                                               |
| -----: | :--------------------------------------------------------------------------------------- |
|      1 | 飛行に問題のない状況。平時であれば判定なしで飛行して良い                                 |
|      2 | 多少の無理をすれば飛行できる状況。雨の中での飛行、移動する飛空船に追いつくなど           |
|      3 | 飛行種族が飛行をためらう状況。天井の低い場所での飛行・滞空、エレベーターの縦穴の昇降など |
|      4 | 飛行が困難な状況。強風の中での飛行、高速移動する物体への追従など                         |
|      5 | まず飛行ができない状況。竜巻の中での飛行など                                             |

ボーナス/ペナルティ:
- 目標値の設定に影響する状況ではないが、飛行の有利・不利に影響する要因がある場合、GMは汎用行動判定にボーナス/ペナルティを与えて良い

| ボーナス / ペナルティ | 説明                                                   |
| --------------------: | :----------------------------------------------------- |
|                    -4 | 著しい低所からの上昇、翼の可動に影響するような狭い空間 |
|                    -2 | 雲や霧の中、向かい風、微細な塵                         |
|                    +2 | 追い風、上昇気流                                       |
|                    +4 | 高所からの滑空、道具の補佐                             |

### 飛行可能な高度

- 飛行可能な高度は、種族に依らず、飛行種族であっても人族の行動可能な高度と一致する
- 地上から8km程度が生存限界の高度であり、そこを越えると身体に影響が出始める

| 高度 | 影響                                                                                                                                       |
| ---: | :----------------------------------------------------------------------------------------------------------------------------------------- |
|  9km | すべての判定に-1dのペナルティを受ける。1日経過するたびに目標値2で生命力抵抗を行い、失敗すると休息による回復効果が得られない                |
| 10km | すべての判定に-2dのペナルティを受ける。1日経過するたびに目標値3で生命力抵抗を行い、失敗すると休息による回復効果が得られず、HPを1点失う     |
| 11km | すべての判定に-3dのペナルティを受ける。6時間経過するたびに目標値3で生命力抵抗を行い、失敗するとHPを1点失う。休息による回復効果が得られない |
| 12km | すべての判定に-4dのペナルティを受ける。1時間経過するたびに目標値4で生命力抵抗を行い、失敗するとHPを1点失う。休息による回復効果が得られない |

> 高度とペナルティ:
> - 高度による影響は、飛行できない種族にも適用される
> - 高高度での活動時には、これらの影響が常に生じる
>
> 飛行船などの機構:
> - 高度8kmより上空を航行する飛行船などには、気圧変化から乗員を保護する魔導技術が施されている
>
> 虫:
> - 高度12kmよりもさらに上空に立ち入ると、虫の群れが現れる
> - 真実神の古い伝承の通り、太陽に近づくことは得策ではない
