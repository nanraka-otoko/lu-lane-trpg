# お金 / 所持品 / 装備品

- このドキュメントでは、アイテムの価値・重量・所持品の携行などのルールやデータを扱います

---

## お金 - Wealth / WLT と価値のルール

- すべてのキャラクターはお金 (WLT) のパラメータを持つ
- WLTの数値だけ、アイテムや装備をセッションに持ち込むことができる

> 獲得 WLT の目安:
> - 駆け出しの一般的な依頼での稼ぎはWLT 2程度
>     - 冒険で得た金品を換金することで、1セッションでWLT 4程度を稼ぐことができる
> - 一般的な冒険者向けの依頼はWLT 3～5程度の報酬が提示される
>     - 戦利品と合わせて、最大でWLT 10程度を稼ぐことができる

### 価値の目安

|  価値 | 目安                                                   | 現実世界での感覚           |
| ----: | :----------------------------------------------------- | :------------------------- |
|     0 | よくある物品。無料ではないが、ルール上は価値を持たない | 筆、メモ帳、日用品         |
|     1 | まとまったお金。奮発して買う道具や資産                 | 全鋼包丁、万年筆           |
|     2 | 一般生活からしたら高級品                               | エントリーモデルのPC       |
|     3 | 一般生活では必須ではない物品                           | ハイエンドPC               |
|    10 | 高級品。大きな買い物                                   | 車を買うくらいの決断       |
|   100 | 資産。人生の大きな決断                                 | 家を建てるくらいの決断     |
|   500 | 一般市民が個人で蓄財できる限界                         | 起業できるくらいのお金     |
|  1000 | 人生を左右する金額。貴族でも使うのをためらう           | 大規模な事業を起こせる額   |
|  5000 | 国宝。取引されることはまずない                         | 学際的な事業、他社との協業 |
| 10000 | ひとつの都市と釣り合う価値                             | 国家予算                   |

> お金・価値の扱い:
> - ゲームとして処理しやすくした、抽象的な指標である
> - お金と価値は、ルゥレーンの世界における実際の金額と正比例しない
>     - 価値:1のアイテムを10000個集めても、都市と釣り合うだけの価値は生まれない
>
> レアリティと入手期間:
> - 価値:1あたり、入手までに1日を要する程度の希少性がある
>     - 例: 価値:5000 の勇者の剣は、お金の価値が大きいだけでなく、入手までに5000日を要する程度の希少性がある
> - ゲーム上では、必要な価値を支払えば、直ちにそのアイテムを入手できたこととして良い

### 換金

- すべての道具類や戦利品は、売値と買値を1:1で換金できる
- 消耗性のある道具を除き、PCの所持品の価値とWLTの合計値は、常に一致する

## 重量 - Weight / WGT のルール

- すべてのアイテムは「重量」のパラメータを持つ
- キャラクターは、装備によって2種類の制限を受ける

### 装備重量

- 武器・盾・防具の重量合計は、STR以下でなければ装備できない
- STRを超える装備は、そもそも身につけることができない
- アクセサリは、この制限に含まれない

> STR 4 のキャラクターの例:
> - ショートソード(重量1) + レザーシールド(重量1) + チェインメイル(重量2) = 装備重量4 → STR 4以内なので装備可能
> - ショートソード(重量1) → ロングソード(重量2)にする = 装備重量5 → STR 4を超えるので装備不可
>
> STR 3 のキャラクターの例:
> - ツヴァイヘンダー(重量4)を装備したい → STR 3を超えているため、装備不可
> - クレイモア(重量3)なら装備可能
>
> STR 3 のキャラクターでアクセサリを装備する例:
> - クレイモア(重量3)を装備している
> - レザーマント(重量1)はアクセサリであるため、装備することができる
> - レザーマントは携行重量に加算されるため、後述するルールによってDEXが-1される

### 携行重量

- 装備品を含むすべての所持品の重量合計が携行重量となる
- 携行重量が STR を超過した場合、超過分だけ DEX が低下する
- DEX が 0 以下になるような荷物は携行することができない
- DEX が 0 以下の荷物を持っている場合、その PC は行動不可となる
- 戦闘など DEX の低下が問題となる場面では、携行品を地面に置くことで一時的に携行重量を軽減しても良い

> STR 4 のキャラクターの例:
> - ショートソード(重量1) + レザーシールド(重量1) + チェインメイル(重量2) = 装備重量4 → STR 4以内なので装備可能
> - さらにアイテム枠に医療キット等(重量1)を入れると携行重量5 → STR超過1 → DEX -1
> - 戦闘開始時に不要なアイテムを地面に置く → 携行重量が4となり、DEXのペナルティが解消された
>
> STR 3 のキャラクターの例:
> - ツヴァイヘンダー(重量4)は装備しないけど、携行したい → STR超過1 → DEX -1
> - ツヴァイヘンダーは装備できないが、持ち歩くことができる
>
> STR 3、DEX 3のキャラクターの例:
> - ショートソード(重量1) + レザーシールド(重量1) + レザーアーマー(重量1) = 装備重量4 → STR 3以内なので装備可能
> - さらにアイテム枠に医療キット(重量1)を携行 → 携行重量4 → STR超過1 → DEX -1
> - 戦利品として美術品(重量2)を獲得した → 携行重量6 → STR超過3 → DEX -3
> - DEX が 0 となったので、このままでは行動不可。所持品を捨てるか、他のPCや同行者に荷物を預ける必要がある

### 重量の目安

|  重量 | 目安                                                                   | アイテムの例                                           |
| ----: | :--------------------------------------------------------------------- | :----------------------------------------------------- |
|     0 | それ単体では重たさを感じないもの                                       | ナイフ、ポーション、スクロール                         |
|     1 | 片手で容易に扱える重たさ。人間が普通に投擲できる限界                   | ショートソード、ジャベリン、魔導書などの分厚い本、食料 |
|     2 | 一般人でも振り回せる重さ。複数を同時に使うのは難しい重さ               | ロングソード、メタルシールド、チェインメイル           |
|     3 | 並の人間は普段持たない重量物                                           | バスタードソード、チェストプレートメイル、小麦粉の袋   |
|     4 | 鍛えている人間でしか持てないような重さ                                 | ツヴァイヘンダー、フルプレートメイル                   |
|     5 | とても重たく、普通の人間では抱えて動くことも難しい物体                 | グレートソード、ラージメタルシールド、酒樽             |
| 6以上 | 非力な人間では持ち上げることすらできない重さ。通常は複数人で持ち上げる | 大柄な人間の死体、荷のない馬車、熟成用バレル           |

> 重量の扱い:
> - 重量は実際の重たさと正比例しない
> - ゲームとして処理しやすくした、抽象的な指標である
> - 例えばバスタードソードは3kg程度で、小麦粉の大袋は20kg以上あるが、ゲーム上では同じ重量となる
> - ツヴァイヘンダーは4kg程度、フルプレートは20kg以上あるが、これもゲーム上では同じ重量として扱う
>     - その重量物を持てることと、扱えることは別の問題である
>     - STRが4未満の者は、ツヴァイヘンダーを振るうことができず、フルプレートを身に着けて行動することもできない

## 道具の所持・収納・携行のルール

- すべてのキャラクターはアイテム・装備品を10個まで所持・携行できる
- 重量の制限を超えない範囲で、PCは自由にアイテムを携行できる
- 装備枠以外に装備を携行する場合、この所持品枠で持ち歩くことができる

---

## アイテムの名称のルール

- 「酒」や「青銅の剣」などの一部のアイテムは、同一名称で道具類や武器としてデータが記載されている
- これらの同一名称のアイテムは、同じアイテムとして扱う
- 例: 「酒」は道具類として使用可能であり、触媒・供物としても使用可能である

---

## 装備のルール

- 武器は1Hを2つ、または2Hを1つまで装備できる
    - 腕の数だけ装備できる
    - 何かしらの要因で腕が1H分欠けている場合、そのキャラクターは1Hの装備を1つまで装備できる
    - 何かしらの要因で腕が1H分増えている場合、そのキャラクターは3H分の装備まで同時に装備できる
- 防具は1つのみ装備できる
- アクセサリは無制限に装備できる
- 装備中の装備品も所持品数としてカウントする

> 装備品と携行品:
> - 装備品は、携行している所持品としてもカウントする
> - 武器や防具、アクセサリを複数装備することでさまざまな効果を得られるが、その分だけ携行できるアイテムが減る

### 装備品の数値

- 持ち手 (武器のみ):
    - 1H: 片手持ちの武器。両手にそれぞれ1つずつ装備できる
    - 2H: 両手持ちの武器。同時に1つしか装備できない
    - 1H/2H: 片手持ちと両手持ちの両方が可能な武器。PCが任意に持ち手を変更できる
- 重量:
    - その装備品の重量
- 射程 (武器のみ):
    - その武器で攻撃可能な距離
    - 0=接触、1=近距離、2=遠距離、3=長距離
- 威力 (武器のみ):
    - その武器が与えるダメージの基礎値
    - $威力 + 攻撃判定MtV$ が、攻撃のダメージとなる
    - その武器で攻撃する場合にのみ参照する。他の武器との合算はしない
- 命中 (武器のみ):
    - 攻撃判定のダイス数のボーナス/ペナルティ
    - $DEX + 命中$ が、攻撃判定でロールできるダイス数となる
    - その武器で攻撃する場合にのみ参照する。他の武器との合算はしない
- 防護:
    - その武器で攻撃を防いだときの防護点
    - $ダメージ - 防護$ が、装備者が攻撃を受けたときのダメージとなる
    - 装備中の武器・盾・防具・アクセサリと合算する
- 回避:
    - 回避判定のダイス数のボーナス/ペナルティ
    - $DEX + 回避$ が、回避判定でロールできるダイス数となる
    - 装備中の武器・盾・防具・アクセサリと合算する
- 装甲 (防具のみ):
    - 装備者がダメージを受けたとき、先に装甲の点数が減少する
    - 装甲を超過するダメージを受けたとき、超過分だけHPが減少する
    - 装甲は1時間をかけて修理判定で修理できる。詳細は [../06-adventuring/01-exploration.md](../06-adventuring/01-exploration.md) を参照すること

### 同一名の装備について

- 同一名の装備は、同時に装備しても効果は重複しない
    - 例: 1Hの杖「マジックセプター」を2つ装備しても、「魔法行使判定に+1d」の効果は1つしか発揮されない
- 防護・回避は、同一名の装備でも重複する
    - これらは「効果」ではない
    - 例: 1Hの盾「メタルバックラー」を2つ装備した場合、防護:2、回避:+2となる

## 汎用的な道具類について

- 下記の道具類は、所持品としての携行を明言せずとも、常に携行しているものとして扱っても良い

| 道具名                     | 説明                                                                                                   |
| :------------------------- | :----------------------------------------------------------------------------------------------------- |
| ペンとインク               | 紙やそれに準じるものに、文字や模様を書き込むことができる                                               |
| 紙 / メモ帳                | 羊皮紙やメモ帳など、書き込める紙類                                                                     |
| ポーチ / ザック / かばん類 | 所持品は背嚢やポーチなどのかばん類に入れていることにして良い                                           |
| 普段着 / 私服              | 装甲や防護などの数値を持たない衣類。PLの好きにコーディネートして良い                                   |
| 水筒 / 水袋                | 飲料水を入れる容器。1～2L程度を携行できる。節約しながら飲めば、3～4日程度は持つだろう                  |
| その他GMが認める道具       | GMとの合意の上で、上記に類似する道具類であれば、所持品として明記しなくても所持していたこととしても良い |

制約:
- これらは所持品のルール上で無制限に携行できるが、無限に所持しているわけではない
    - 「紙を無限にばらまきます」といったロールプレイは認められない
    - 「着火剤として紙を少量使います」といった宣言は、認めても良い
        - 「紙を無限に燃料にします」というのはNG。また、光源に使うこともできない
    - 冒険中に地図やメモを残す場合に、無制限に使用できる、という裁定となる
        - 荷物の処理を簡略化するためのルールであり、悪用するためのルールではない、ということ
- 荷物を没収されるなどして所持品を失った場合、これらの道具類も一緒に失う
    - 所持品として明記されていないだけであり、荷物の一部としてはまとめて携行しているためである
- 水筒については「わざわざ飲水について明記しなくても、一般的な冒険のための必要量がある」ということを示している
    - セッションによっては、渇水が生じる可能性が十分にある
    - 水場などが描写されたセッションでは、PLが給水を宣言せずとも、暗黙的に水筒に水を詰めていたことにしても良い
