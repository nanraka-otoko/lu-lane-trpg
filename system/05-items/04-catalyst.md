# 触媒 / 供物

- 魔法・奇跡の行使を手助けする道具
- 使用すると消費される
- 複数同時に使用可能だが、2個以上を同時に使用するとスペルの行使の難易度が上昇する
- 使用ルールの詳細は [../04-spells/](../04-spells/) の「触媒 / 供物」を参照すること
- 装備品や道具と同一名の触媒・供物は、それらのアイテムを触媒・供物として活用することができる
    - 触媒・供物として使用した場合、そのアイテムは消費され、失われる
