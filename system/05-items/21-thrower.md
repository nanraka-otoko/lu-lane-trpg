# 遠隔武器: 投擲

- 石や種を弾として使用する場合、携行品になくてもその場にあるものを使用したことにして良い (弾の持ち込みは不要)

| 武器名           | 持ち手 | 価値 | 重量 | 射程 | 威力 | 命中 | 防護 | 回避 | 備考                                                                                                          |
| :--------------- | :----- | ---: | ---: | ---: | ---: | ---: | ---: | ---: | :------------------------------------------------------------------------------------------------------------ |
| 投擲             | 1H     |    0 |    0 |    2 |    0 |   -1 |    0 |    0 | その辺の石や硬い種、木の実などを投げる。無手の時に使用可能                                                    |
| ポーションの投擲 | 1H     |    0 |    0 |    2 |    0 |   -1 |    0 |    0 | 所持しているポーションを投擲する。味方に投擲する場合もダメージが生じる                                        |
| 矢の投擲         | 1H     |    0 |    0 |    2 |    0 |   -2 |    0 |    0 | 所持している矢筒の矢を投擲する                                                                                |
| 投げ紐           | 1H     |    1 |    0 |    2 |    1 |    0 |    0 |    0 | その辺の石や硬い種を投げる                                                                                    |
| スリングショット | 1H     |    2 |    1 |    2 |    2 |    0 |    0 |    0 | その辺の石や硬い種を放つ。1Hの枠で装備・使用可能                                                              |
| 二連投げ紐       | 1H     |    8 |    0 |    2 |    1 |   -3 |    0 |    0 | この武器で遠隔攻撃する場合、1回の行動で2回まで攻撃できる。その辺の石や硬い種を投げる。扱いが非常に難しい      |
| 精霊の投げ紐     | 1H     |   12 |    0 |    2 |    2 |   +2 |    0 |    0 | その辺の石や硬い種を投げる。森の精霊の力が宿った糸で作られた投げ紐                                            |
| 気合の投げ紐     | 1H     |   18 |    0 |    2 |    2 |    0 |    0 |    0 | MPを1点消費することで、射程を+1、威力を+2。その辺の石や硬い種を投げる。地味な見た目だが、秘宝のひとつとされる |
