# 魔導書 / 聖典

- 魔法や奇跡について記された書物
- 魔法と奇跡の使用条件を緩和する
- 魔導書や聖典の内容は暗記できず、スペルの行使の際に手元に持つ必要がある
    - これはINTやFTHが十分であれば暗記できる内容を、その場で探しながら行使するためである
- 戦闘中に用いてスペルを行使する場合、1Hに魔導書または聖典を所持しておく必要がある
- 戦闘外で使用する場合は、手に持たずに床や卓上に置いて読むことができる
    - 落ち着いて読める状況では、両手に別の道具を持っても良い
