# ポーション類

- すべて消耗性であり、使用すると失われる
    - 使用済みのポーションは「空のポーション瓶」となる
    - 空のポーション瓶はその場で捨てても良い
- 予備行動として、自身の行動手番に手番を消費せずに使用可能
    - 詳細は [../07-combat/](../07-combat/) の「予備行動」を参照すること
- 飲むか、振りかけることで使用可能
    - 振りかける際の射程は0 (接触) となる
- 投擲の弾として使用可能
    - 「ポーションの投擲」の手段で攻撃判定を行い、命中した相手にポーションの効果を及ぼす
    - ポーションの投擲は予備行動では実行できず、自身の行動を消費する
    - 命中時には攻撃によるダメージが生じる (味方に対する投擲でもダメージが生じる)
