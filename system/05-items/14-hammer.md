# 近接武器: 槌

- 両手持ちでの攻撃時、追加で2体までを対象に同時攻撃可 (最大3体まで)
- 追加した対象の数だけ攻撃判定に -3d のペナルティを受ける
- 追加した対象の数だけ、防護を無視する効果の点数が1点減少する
    - 例: ウォーハンマーを用いて3体を対象に攻撃した場合、防護の無視は1点のみとなる
