# スクロール

- すべて消耗性であり、使用すると失われる
- 予備行動で使用可能
    - 詳細は [../07-combat/](../07-combat/) の「予備行動」を参照すること
- 魔法や奇跡を行使する巻物。現代では失われた魔法や奇跡を行使するスクロールも存在する

| 名前             | 価値 | 重量 | 効果                                                                                                                     |
| :--------------- | ---: | ---: | :----------------------------------------------------------------------------------------------------------------------- |
| 幸運のスクロール |    3 |    0 | 次の自分の運判定に+4d                                                                                                    |
| 看破のスクロール |    3 |    0 | 次の自分の探索・感知・知識判定に+4d                                                                                      |
| 雄弁のスクロール |    3 |    0 | 次の自分の対人判定に+4d                                                                                                  |
| 加速のスクロール |    5 |    0 | 次の自分の攻撃判定に+4d                                                                                                  |
| 鉄壁のスクロール |    7 |    0 | 自分の回避判定に+4d。効果は3ラウンド持続                                                                                 |
| 結界のスクロール |   10 |    0 | 射程:2以内の味方全員に+3の障壁を付与。障壁は3ラウンド持続                                                                |
| 鍛冶のスクロール |   10 |    0 | 自身の装甲を5点回復                                                                                                      |
| 詠唱のスクロール |   30 |    0 | 次の自分の魔法・奇跡の行使判定に+6d                                                                                      |
| 脱出のスクロール |   25 |    0 | 魔法「テレポーテーション」を発動する。冒険者ギルドに帰還する                                                             |
| 転移のスクロール |   50 |    0 | 魔法「テレポーテーション」を発動する。転移先はスクロールごとに固定                                                       |
| 圧縮のスクロール |   70 |    0 | このスクロールは任意のタイミングで使用できる。発動した瞬間を自身のターンとして扱い、1回の行動権を得る                    |
| 助命のスクロール |    3 |    0 | 死亡後1ラウンド経過していない味方1人のHPを-CON点まで回復し、気絶状態に留め置く                                           |
| 再起のスクロール |   10 |    0 | 死亡後1ラウンド経過していない味方1人を、HP1で復帰させる                                                                  |
| 反魂のスクロール |   25 |    0 | 死亡後3ラウンド経過していない味方1人を、HP1で復帰させる                                                                  |
| 蘇生のスクロール |  100 |    0 | 真実神の奇跡「蘇生」を発動する                                                                                           |
| チートコード     |    - |    0 | このスクロールは任意のタイミングで使用できる。未踏神の奇跡「チート」を発動する。異界の言葉が記された気味の悪いスクロール |
