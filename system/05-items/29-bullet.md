# 遠隔武器: 矢弾・矢筒

## 矢筒

- 矢筒は携行品として所持する必要があり、戦闘中に使用する場合は携行している必要がある
- 携行する際は所持品枠を1つ使用する
- 矢筒に入る矢弾は有限だが、ゲームシステム上では無制限に扱って良い (矢筒内には矢を無制限に収納可能)
- 矢筒に特殊効果がついている場合、その矢筒に収納した矢を用いる際に効果を発揮する

| 矢筒名             | 価値 | 重量 | 備考                                                                                           |
| :----------------- | ---: | ---: | :--------------------------------------------------------------------------------------------- |
| 矢筒               |    0 |    0 | 一般的な矢筒                                                                                   |
| 聖堂の矢筒         |   12 |    1 | アンデッドに対して威力を+2。矢筒内に充填された聖なる力が、矢に聖性を付与する                   |
| 軽業の矢筒         |   25 |    0 | 収納した矢弾の重量を-1。金魔細工により軽量化の魔法がかけられた矢筒                             |
| クイッククィーバー |   40 |    0 | カテゴリ:弓の武器は予備行動で遠隔攻撃を行える。予備行動での遠隔攻撃には-4dのペナルティを受ける |

## 矢弾

- 特殊な矢弾を使いたい場合、その矢を購入して使用する
- 購入した矢弾の本数は有限だが、ゲームシステム上では無制限に扱って良い
    - 携行している矢は無制限に使用可能
    - 矢弾を購入すると、そのセッション中はその種類の矢を使い放題となる
- 矢弾は消耗品であり、1セッションが終了したら消耗する
    - 1回しか矢を使用していなくても消耗扱いとなる
    - 同じ矢を異なるセッションで使用したい場合、都度購入する必要がある
- 矢弾は携行品としてカウントしない
    - 矢筒のみを携行品としてカウントする

> 弓と矢の両方に属性効果がある場合について:
> - 矢弾の属性が優先されます

> 長杉産業で:
> - 矢は毎セッション買う
> - 買ったセッションでは使い放題
> - 1本でも使ったら消耗扱いでお金が減ります
