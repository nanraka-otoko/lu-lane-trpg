# 金属

## 鉄 / iron

- ルゥレーンで良く用いられる汎用金属
- 大量に産出するため、様々な用途で活用されている
- 古くはドワーフの手にのみあった金属であり、文明の統合とともに精錬技術が拡散した
- 古代のエルフの一部がこの技術を得るために地下に潜り、ダークエルフの祖となったとも言われる
- 鍛冶神信仰の基礎ともなる金属であり、ドワーフ以外の種族にも広く信徒がいる理由となっている

## ヘビーメタル / 重鉄

- 重たく硬い金属
- 鉄と似て多く産出するが、鉱石もインゴットもとにかく重たいため、流通量が少ない
- 高度な建材として用いられることが多く、武具となることは稀
- 古代遺跡にはヘビーメタルを建材としたものもあり、発破などが通用しない構造体となっているケースもある

> 解説:
> - 鉄よりも重たいけど、安価に高威力・高防護な装備が得られる、という立ち位置の金属素材です
> - 建材として利用されているという前提を示すことで、ダンジョン自体を崩壊させるようなロールプレイに制約をかけるのも良いでしょう

## 銅 / Copper

- 加工のしやすい金属として、古くから活用されてきた金属
- 自然銅として産出し、錫と合わせた合金として用いることが多い
- 鉄が盛んとなった現代では、銅単体として装飾や貨幣(銅貨)として用いられることが多く、武具には用いられない

### 錫 / Tin

- 柔らかく加工しやすい金属
- 食器や装飾品の溶接に用いられる
- ヒュームとドワーフの間では古くから一般的な金属である

### 青銅 / Bronze

- 銅と錫の合金
- 古代のヒュームが用いており、農耕文明の祖となった
- 豊穣神信仰に欠かせない金属であり、現代では祭具としてその名残が生きている

## 銀 / Silver

- 白く輝く金属
- ルゥレーンでは自然銀として産出することも多く、古くから用いられてきた
- 邪気を払うとされ、神殿の装飾や祭具に用いられる

### 真なる銀 / 聖銀 / アルジェント

- 聖別された銀
- 銀は聖なる力と相性の良い金属であり、神官によって聖別された銀は様々な性質を得る
- 聖銀は銀と同じように美しくありながら、武具としても使える強度を持つ。
- 主に真実神に連なる神の祭具と、神殿騎士たちの武具に用いられる。

## 金 / Gold / Aurum

- 自然金属として産出する金属
- ルゥレーンにおいては古代から発見・利用されている
- 宝飾品として用いられることが多い
- 汎用金属よりも魔力を通しやすく、高度な魔道具にも用いられる

## 軽銀 / 青銀鋼 / アズリア / アズリア鋼

- 青白く輝く金属
- 軽くて強度もあり、優秀な武具の素材として珍重される
- 銀のような美しさがあるため、軽銀や青銀とも呼ばれる
- 銀と違い、自然金属としては露出せず、精錬が必要
- 武具用に精錬されたものはアズリア鋼とも呼ばれる

## 真なる金 / オリヴェリタス

- 真実神が生み出したとされる原初の金
- 強度と靭性に長けた万能の金属
- 希少な金属であり、真実神の神職である神殿鍛冶師によってのみ加工が可能
- 通常の鍛冶ではなく、儀式によって加工がなされるため、鍛冶神の手にない金属である
- 真なる金は、未加工の状態であっても、それだけで秘宝として扱われる代物である
- 例えば、勇者の剣はオリヴェリタスの刀身を持つ

## 魔鉄 / 魔鉄鉱 / マギアタイト

- 魔法を通しやすい金属
- 鉄に似た性質があり、魔力を宿した鉄と考えられている

## 磁魔鉄 / マグネタイト

- 魔力に対して引き寄せられる性質のある金属
- 魔石鉱山の探知機や、魔力計にも用いられる
- 流した魔力を記憶する性質があり、磁魔鉄製の投擲物を手元に引き寄せるといったことも可能

> 磁石との違い:
> - ルゥレーンにも地鉄や磁石がありますが、磁魔鉄は異なる鉱物・金属です
> - 磁魔鉄は、魔力に対して引き寄せ合うような反応を示します

## 隕鉄 / メテオライト

- 宇宙から飛来した鉱物
- 未知の物質であり、重たく、加工が難しい

> 解説:
> - 必ずしも鉄とは限りませんが、ルゥレーンの人々は隕鉄と呼称しています

# 動植物

## 幻影樹

- 大地に生える、割れた枝や枯れた木など
- 神聖な力や魔力が満ちる夜に輝きを放ち、本来の姿を表す

## 空魚

- 空海に棲まう魚類
- ヒレと肺腑が発達しており、空を飛びながら別の海へと移動する

### 空鯨

- 空海に棲まう鯨
- 肺呼吸が可能であり、数ヶ月ほど空を漂って海を探す

# 精霊

- ルゥレーンの各地に偏在する精霊

## 火の精霊 / 水の精霊 / 風の精霊 / 氷の精霊 / 土の精霊 / 雷の精霊

- 自然の中に存在するとされる精霊
- 小精霊とも呼ばれ、世界の各地に点在する
- 一般魔法の多くはこの精霊の力を借りて行使する

風→土→雷→水→炎→氷→風:
- 水は炎を消す
- 炎は氷を溶かす
- 氷は風をせき止める
- 風は土を風化させる
- 土は雷を止める
- 雷は水に流れる
